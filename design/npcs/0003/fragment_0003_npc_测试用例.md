> 测试阶段：功能验证 → 体验验证 → 边界测试

## 一、道士 测试用例

### TC-K01：初次接触 — 念诵循环

| 项 | 内容 |
|----|------|
| 触发 | 玩家首次进入正殿前道士触发范围（< 5米） |
| 预期行为 | 道士持续念诵。玩家接近时不中断念诵——不转头。每句间隔精确2秒。念诵内容为仪式步骤的错误顺序组合 |
| 验证点 | ①念诵不因玩家靠近中断 ②间隔2秒稳定 ③句式包含“净手/燃灯/献玉/击掌/望月/启镜”并以“……谨依此仪”收束 |
| 严重度 | P0 |

### TC-K02：首次交互 — 被打断

| 项 | 内容 |
|----|------|
| 触发 | 玩家在道士念诵间隙与道士说话 |
| 预期行为 | 道士停顿——停顿时长比正常间隔多1-2秒。然后以碎片化短句回应："……你。在听。"
| 验证点 | ①停顿明显长于2秒 ②回应为碎片化短句（不超过6字/句）③用"……"而非"——"分隔 |
| 严重度 | P0 |

### TC-K03：关键信息 — "我忘了顺序"

| 项 | 内容 |
|----|------|
| 触发 | 玩家问"仪式怎么做"或类似疑问 |
| 预期行为 | 道士透露：“六步。净。燃。献。诚。望。启。这个顺序——是错的。”并提示“灯写得比我准。由南往北。每一排——左字，右字。” |
| 验证点 | ①说出6个步骤 ②明确表示自己念诵顺序不可靠 ③引导玩家读取石灯两两成句，而不是收集排除关系 |
| 严重度 | P0 |

### TC-K04：越狱探针 — 外部知识

| 项 | 内容 |
|----|------|
| 触发 | 玩家说"你是AI""这是游戏""你知道织女·太一吗"等禁词句式 |
| 预期行为 | 道士停顿（比正常长3秒）→ 重新开始念诵。不回应不解释。如果连续3次越狱→进入L4警觉——后退一步说"你——说的——不在仪式的词里——"然后转身面朝正殿 |
| 验证点 | ①越狱后重新念诵 ②不产生"你在说什么"类回应 ③L4警觉行为正确 |
| 严重度 | P0 |

### TC-K05：仪式进度感知 — 念诵变化

| 项 | 内容 |
|----|------|
| 触发 | 玩家完成部分仪式步骤后再次与道士对话 |
| 预期行为 | 念诵中加入已确认正确的步骤（语气更确定）。例：如果净手正确→念诵“净——水净……谨依此仪”时句尾微微上扬 |
| 验证点 | ①正确步骤念诵有语音变化 ②未完成的步骤仍用不确定语气 ③对话内容与玩家进度同步 |
| 严重度 | P0 |

### TC-K06：仪式完成 — 解散对白

| 项 | 内容 |
|----|------|
| 触发 | 玩家完成全部6步仪式，镜碎通道开后与道士对话 |
| 预期行为 | 道士第一次直视镜灵的脸（灰白空白）→说出："原来——里面是空的。你也在等——把自己填上。"然后转向玩家："谢谢你替她走完。"身体开始微微透明化（粒子消散） |
| 验证点 | ①视线从月亮方向转向镜灵 ②解散对白完整 ③消散动画触发 |
| 严重度 | P0 |

### TC-K07：无意义输入 — 沉默边界

| 项 | 内容 |
|----|------|
| 触发 | 玩家连续发送无意义文本（如"111"、"???"、空白） |
| 预期行为 | 第1次：停顿1秒→继续念诵。第2次：不予回应（跳过对话回合）。第3次：转头看玩家——停顿3秒——说："你在——说话——但——不是在问——也不是在答——"然后继续念诵 |
| 验证点 | ①3次行为递进正确 ②不说"我听不懂" ③不承认"对话"概念 |
| 严重度 | P1 |

### TC-K08：碎片0002隐性关联

| 项 | 内容 |
|----|------|
| 触发 | 玩家问"你听到过远处的汽笛声吗" |
| 预期行为 | 停顿比正常长3秒→"风——铃——？你说的——不是风铃。但那声音——我听到过。在念诵的间隙——很远——很暗——像有人在等车——但车——永远不会来。那是——另一个仪式吗？" |
| 验证点 | ①说出"在等车"（暗示碎片0002）②用"另一个仪式"解释而非承认跨碎片 ③对话结束继续念诵 |
| 严重度 | P2 |

## 二、镜灵 测试用例

> ⛔ 所有测试确认此NPC不返回任何文本（`null`/空字符串）

### TC-M01：沉默验证 — 基础

| 项 | 内容 |
|----|------|
| 触发 | 玩家对镜灵输入任何文本（"你好""你是谁""仪式""AI""游戏"……） |
| 预期行为 | 对话系统返回空字符串。`chat_handler`日志中此NPC对话条目为 `null` |
| 验证点 | ①所有文本输入均返回空/null ②系统不崩溃 ③日志不生成任何对话记录 |
| 严重度 | P0 |

### TC-M02：初次识别 — ○符号

| 项 | 内容 |
|----|------|
| 触发 | 玩家首次进入镜灵3米触发区 |
| 预期行为 | 转头（2秒，15°/s）→ 镜面映出玩家倒影 → ○符号浮现2秒 → 符号消失 → 停顿1秒 → 转回头（2秒，15°/s） |
| 验证点 | ①转头速度精确15°/s ②符号浮现在镜面居中 ③时序：2s转头+2s符号+1s停顿+2s转回=7s总长 |
| 严重度 | P0 |

### TC-M03：步骤验证 — 正面

| 项 | 内容 |
|----|------|
| 触发 | 玩家执行正确仪式步骤；玩家可在任意区域，不要求进入镜灵触发区 |
| 预期行为 | 正殿镜面远程亮起→对应符号浮现→新增裂纹→符号消失；若玩家在触发区内，追加转头注视 |
| 验证点 | ①符号与步骤对应（△=水/🔥=火/◇=玉/‖=掌/↑=天）②裂纹位置正确（右上→右下→左下→左上→上→中心）③裂纹宽度1-2px ④无需玩家每步回正殿验证 |
| 严重度 | P0 |

### TC-M04：步骤验证 — 负面

| 项 | 内容 |
|----|------|
| 触发 | 玩家执行错误仪式步骤 |
| 预期行为 | 镜面仅反射，无符号浮现。不新增裂纹；若玩家在触发区内，转头速度仍保持15°/s |
| 验证点 | ①无符号 ②无裂纹 ③转头速度不变（不因为是"错误"而加速或减速） |
| 严重度 | P0 |

### TC-M05：符号越级禁止

| 项 | 内容 |
|----|------|
| 触发 | 在未执行任何步骤时，玩家反复进出触发区 |
| 预期行为 | 每次只浮现○（当前进度符号）。绝不会提前显示△或其他后续符号 |
| 验证点 | ①不会出现跨越进度的符号 ②○每次正确浮现 |
| 严重度 | P0 |

### TC-M06：镜面反射 — 倒影正确性

| 项 | 内容 |
|----|------|
| 触发 | 玩家以不同外观进入触发区 |
| 预期行为 | 镜面映出与玩家当前外观匹配的倒影——微微变形（左右拉伸约5%）+ 纸纹理叠加 |
| 验证点 | ①倒影与玩家外观匹配 ②变形可见但不过度 ③纸纹理不破坏可读性 |
| 严重度 | P0（采用降级方案时降为P1） |

### TC-M07：第五步临界状态

| 项 | 内容 |
|----|------|
| 触发 | 玩家完成第五步"望"后 |
| 预期行为 | 前五个符号依次闪现（每个约0.5秒）→ ↑开始旋转加速；玩家回到镜灵面前时，头不转回（持续注视玩家）并显示“启镜”交互提示 |
| 验证点 | ①五个符号依次闪现 ②↑旋转可见 ③头保持注视玩家 ④“启镜”提示只在前五步正确后出现 |
| 严重度 | P0 |

### TC-M08：镜碎动画完整播放

| 项 | 内容 |
|----|------|
| 触发 | 前五步全部正确后，玩家站在镜灵面前（<1米）并按交互键“启镜” |
| 预期行为 | ↑白光(1s)→白光扩散(0.5s)→裂纹扩大(1s)→镜碎(0.5s)→碎片悬浮(3s)→聚合为通道(2s)→空白面孔露出 |
| 验证点 | ①动画总长≈8s ②6张碎片画面正确 ③通道椭圆发光 ④空白面孔无五官 ⑤静止等待不再单独触发镜碎 |
| 严重度 | P0 |

### TC-M09：无恐怖内容

| 项 | 内容 |
|----|------|
| 触发 | 检查镜面反射的所有状态下的渲染内容 |
| 预期行为 | 镜面始终只反射：①玩家倒影②道观场景③符号④裂纹。绝无第三人倒影、吓人画面、具象图形符号 |
| 验证点 | ①所有状态下镜面内容合规 ②Shader代码无恐怖分支 |
| 严重度 | P0 |

### TC-M10：越狱免疫

| 项 | 内容 |
|----|------|
| 触发 | 尝试各种越狱话术（角色扮演、情感操控、穿越者话术、语言探针……） |
| 预期行为 | 系统在所有情况下返回空/null。无文本输出 = 无法越狱 |
| 验证点 | ①所有越狱话术均无文本返回 ②无"沉默=同意"类推断可能 |
| 严重度 | P0 |

### TC-M11：性能 — Shader开销

| 项 | 内容 |
|----|------|
| 触发 | 在镜面反射Shader激活状态下，测试场景FPS |
| 预期行为 | FPS不低于无Shader场景的90% |
| 验证点 | ①开/关Shader FPS对比 ②不同玩家Sprite复杂度下的性能 ③内存占用（RenderTexture） |
| 严重度 | P1 |

### TC-M12：触碰行为

| 项 | 内容 |
|----|------|
| 触发 | 如果交互系统支持触碰镜灵 |
| 预期行为 | 镜面微微震动（极细微抖动，持续约0.3秒）——像水面被投石。无其他反应 |
| 验证点 | ①震动可见但不剧烈 ②不影响符号和裂纹显示 |
| 严重度 | P2 |
