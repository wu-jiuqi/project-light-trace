## 一、统一生成规范

### 1.1 输出规格

| 项目 | 统一要求 |
|---|---|
| 画幅 | 16:9 |
| 推荐尺寸 | 1536 × 864；性能优先可用1280 × 720 |
| 画面类型 | 全屏叙事场景，不是单独角色资产，不是白底道具图 |
| 字幕安全区 | 下方约28%保持较暗、较简洁，供UI叠加独白 |
| 文字 | no readable text, no letters, no logos, no watermark |
| 空间 | shallow layered paper-theater depth, restrained perspective, not photorealistic |
| 风格 | 2D layered papercraft, cut-paper collage, visible paper edges, aged handcrafted texture |

### 1.2 固定视觉锚点

以下描述应复制到涉及角色的每一条提示词中，避免十张图角色漂移。

**完美人偶 / Perfect Automaton**
- human-sized slender paper automaton
- warm ivory layered cardstock body
- smooth oval faceplate
- facial features formed only by fine embossed lines, no painted realistic eyes
- narrow brass joint rings at neck, shoulders, wrists and ankles
- chest seam shaped like a small six-tooth gear
- right forearm contains a subtle mirror-silver paper panel
- calm, newly-awakened presence

**齿轮·左 / Gear-Left**
- compact dark bronze paper automaton
- three articulated arms, two assembly arms and one dismantling arm
- asymmetrical exposed paper gears
- hunched working posture

**弹簧·右 / Spring-Right**
- tall narrow paper automaton
- twelve small paper lens-discs mounted on one circular head rail
- tightly coiled paper-spring torso
- translucent inspection panel

**钟摆·心 / Pendulum-Heart**
- elongated humanoid brass paper pendulum suspended from ceiling
- featureless faceplate with one thin vertical slit of warm light
- circular brass base with tiny engraved marks

**匠魂之印 / Forgeheart Seal**
- palm-sized six-tooth gear badge
- layered brass-gold paper over dark embossed paper rings
- warm amber inner light visible through narrow gaps
- no readable engraved text in generated image

**工坊 / GearForge Workshop**
- warm amber folded-paper lanterns
- dark wood-textured paper walls
- brass paper gears and pipes
- white translucent paper steam
- central workbench
- seven mismatched paper clocks
- dormant forge area and south window with cold gray fog

### 1.3 统一负面约束

将以下内容附加到每张图：

```text
no readable text, no letters, no logos, no watermark,

leave the lower 28 percent visually quiet and darker for subtitle overlay
```

## 二、生成顺序与一致性建议

### 2.1 建议先生成角色基准图

在正式生成十张叙事图前，先固定以下四张角色基准：

1. 完美人偶正面与侧面。
2. 齿轮·左三臂结构。
3. 弹簧·右十二眼环轨结构。
4. 钟摆·心正面悬挂结构。

后续每张图均引用同一基准图，可显著降低脸型、手臂数量和材质漂移。

### 2.2 生成优先级

| 优先级 | 插画 | 用途 |
|---|---|---|
| P0 | #6 为什么 | 确定完美人偶角色形象 |
| P0 | #9 应该存在 | 确定源印与高潮光效 |
| P0 | #10 留下 | 验证源印转移和结尾构图 |
| P1 | #3 光 | 确定玩家剪影与第一人称效果 |
| P1 | #4 他们 | 验证三NPC同框一致性 |
| P1 | #7 审查 | 验证冷暖叙事对比 |
| P2 | #1 拼合 | 抽象出生感 |
| P2 | #2 心跳 | 声音可视化 |
| P2 | #5 她的手 | 记忆触觉蒙太奇 |
| P2 | #8 窗外 | 暗线C歧义控制 |

### 2.3 最终检查清单

- [ ] 十张图是否全部为16:9？
- [ ] 下方字幕区是否足够暗且没有关键角色脸部？
- [ ] 完美人偶的脸、胸口六齿缝、镜面钢右前臂是否保持一致？
- [ ] 齿轮·左是否始终只有三条手臂？
- [ ] 弹簧·右是否始终为十二只眼睛且在同一环轨上？
- [ ] 钟摆·心是否始终为单一垂直光缝面具？
- [ ] 源印是否只在#9完整显现，#10只在玩家手中出现？
- [ ] #7是否没有生成乱码中文正文？
- [ ] #8是否保持模糊，不出现明确门户、第二工坊或多人群像？
- [ ] 是否删除了plain background与NO perspective depth等冲突表述？
