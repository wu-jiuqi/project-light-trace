# 碎片 #0762「颜色的葬礼」—— 角色美术资源规格

> **目标风格**：日式RPG像素风，俯视视角 32×32 px 角色规格
> **色彩策略**：全部角色以**彩色**制作。灰阶视觉效果通过运行时 Shader 层统一处理（全局去饱和度 + 可选色彩穿透标记）。
> **动画规格**：每角色需准备 4方向（上下左右）× 3帧（idle动画）的精灵表
> **美术优先级**：P0 静态立绘 + 4方向1帧（行走用）；P1 4方向3帧idle；P2 Shader穿透层资源

---

## Shader 层设计说明（美术需知）

- **全局灰阶**：所有角色精灵默认经过一个全局去饱和度 Shader，将画面转为灰阶。此 Shader 在美术制作阶段**不需要参与**——角色全部以彩色制作。
- **色彩穿透标记**：部分角色的关键视觉元素需要**绕过灰阶 Shader**，在灰阶世界中保留微弱色彩。这些元素在资源层面以独立图层或标记色（如特定 Alpha 通道 / 额外贴图）提供，由 Engineer 在 Shader 中实现穿透逻辑。
- **觉醒动画**：当某个颜色被"找回"时，对应角色的相关区域从灰阶恢复为全彩。此过渡由 Shader 参数动画驱动，不需要额外美术资源（除非涉及粒子特效等）。

> **缩写约定**：下文所有角色外观描述均为**彩色本体**。表格中的 `[SP]` = Shader Penetration，标记需要穿透灰阶 Shader 的元素。

## 玩家载体：「迷路的旅人」

### 基本信息
- **身份**：穿越灰雾抵达灰白镇的旅行者，受托送信到此
- **性别**：不限定（可通过服装保持中性）
- **年龄感**：约25-30岁，长途跋涉后的疲惫感
- **体型**：中等身材，微微驼背——走了太久的痕迹

### 外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 中短发，有些凌乱——刚起床没梳。深棕色。 | `#5A4030` |
| **面部** | 略显疲惫但眼神清醒。眉间有一道很浅的竖纹——长期皱眉的痕迹。 | 肤色 `#D8C8B8` |
| **上衣** | 米白色旅行衬衫，领口微敞。袖口卷到手肘。衣领上有一圈淡青色领饰——原本的颜色，在灰阶Shader下会变成一圈暗线。 | `#F0EDE4` 主体 / `#D8D4C8` 褶皱阴影 / 领饰 `#8AB8C8` `[SP]` |
| **外套（可选）** | 搭在肩上的深墨绿旅行斗篷——不是穿着的，是披着的。斗篷下摆有磨损和泥渍。 | `#3A4A3A` / 磨损边缘 `#2A3A2A` |
| **裤子** | 灰棕色旅行长裤，膝盖处有方形补丁，针脚粗犷但不乱。 | `#7A6A5A` / 补丁 `#8A7A6A` |
| **鞋子** | 磨损的短靴，鞋底被磨得一边高一边低。鞋面上有干涸的泥渍。 | `#6A5040` |
| **包包** | 左肩斜挎一个深棕色皮制信使包——里面装着那封信的位置。包带在衣服上压出了一道浅色印子。 | `#6A4030` |

### 关键视觉标记
- **信使包**：左肩斜挎。包的一角露出一个信封的米白色角——这是玩家的唯一"视觉身份锚"。玩家在任何场景中都应该能通过这个包认出自己。信封 `#F0EDE4`。
- **手部磨损**：手背和指节有淡淡的茧痕——不是铁匠那种厚茧，是"长期握登山杖/地图"留下的薄茧。
- **衣领淡青色领饰** `[SP]`：在全局灰阶 Shader 下呈现为比周围深一阶的灰线——是这个世界仅存的"颜色残余"暗示之一。觉醒后恢复为全彩淡青色。

### 姿态
- **idle**：微微低头，呼吸缓慢。偶有动作——抬起左手捏一下右肩（像在缓解肌肉酸痛），或者调整一下信使包的位置。
- **行走**：步伐稳健但不快——没有急事。"反正已经走了这么久了，不差这几步。"
- **站立（对话中）**：面对NPC时稍微侧身——不是不礼貌，是旅途中的习惯：随时准备继续走。

## NPC #1：铁匠·老霍（红色·愤怒）

### 基本信息
- **年龄**：约52岁
- **体型**：粗壮，肩宽腰圆，典型的长期体力劳动者体格。身高在中上——不矮，但因为常低头俯身锻造，整体高度感被压低。
- **职业**：铁匠

### 外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 深灰夹白短发，两鬓剃得很短，头顶稍长。发色不是自然灰——更像是煤灰染的。 | `#686868` / 灰白 `#AEAEAE` |
| **面部** | 方脸，下颌宽。眉毛粗重，鼻梁上有一道横向的旧疤——不是刀伤，是铁渣溅的。嘴型永远是向下的——不是不高兴，是习惯。 | 肤色 `#C8B8A8` / 疤痕 `#A89888` |
| **上衣** | 无袖的深麻灰褐厚布背心，领口被汗渍浸出一圈深色。露出两条粗壮的手臂——前臂上满是老茧。 | `#7A6A5A` / 汗渍圈 `#6A5A4A` |
| **围裙** | **核心视觉锚点**。焦茶色帆布制铁匠围裙，覆盖从胸口到膝盖的范围。围裙正面散布着铁屑粉末和随机烫痕。**右下角有一块深酒红色的铁水烫痕**——这是唯一需要在灰阶Shader下保留微弱色彩的元素。 | `#8A7A5A` 主体 / 红色烫痕 `#C04040` `[SP]` |
| **裤子** | 灰棕色粗布长裤，膝盖处有厚布补丁。裤脚塞进靴筒里。 | `#6A5A4A` |
| **靴子** | 厚底铁匠靴，靴头有铁片加固——已经磨得发亮。 | `#4A3A2A` / 铁片 `#B8B8B8` |

### Shader 穿透标记
- **围裙上的铁水烫痕** `[SP]`：位于围裙右下角，约 4×6 px 的深酒红色斑块 `#C04040`。在全局灰阶 Shader 下，此区域以约 15-20% 的原始饱和度穿透，呈现为"让人觉得不对劲的深色斑块"。当玩家第一次注意到它并停留超过 3 秒时，触发微提示。

### 姿态
- **idle**：右手举锤在空中半停——锤子永远不上不下。左手扶在铁砧上。每隔 5-8 秒，锤子落下一次（敲在铁胚上），然后回到原位。
- **行走**：极少行走。如果出门，步伐沉重、每一步落地有声。
- **觉醒后（红色恢复）**：锤击频率明显加快。不再停留在半空中——一锤接一锤。锻造台火焰从无到有，橙红烈焰 `#E06030` + 橙黄高光 `#F0A040`。

## NPC #2：花店老板娘·阿莲（蓝色·悲伤）

### 基本信息
- **年龄**：约35岁
- **体型**：中等偏纤细。站姿永远端正——但端正得像个假人。
- **职业**：花店店主

### 外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 浅栗色长发，松松地挽成一个低髻，用一根米白色发簪固定。额前有两缕碎发——刚好遮住眼角。 | `#A08070` / 发簪 `#E8E0D8` |
| **面部** | 五官精致但缺乏活力。嘴角永远有微笑——但微笑只在嘴唇上，到不了眼睛。眼睛下方有极淡的阴影——不是黑眼圈，是"什么都没有"的空洞。 | 肤色 `#D8C8B8` / 阴影 `#C8B8A8` |
| **长裙** | 及踝的淡蓝灰色连衣裙，领口有简单的交叉设计。裙摆上有一排非常浅的刺绣——矢车菊的轮廓。 | `#B8C4D4` / 褶皱 `#A0B0C0` / 绣线 `#A8B8C8` |
| **围裙** | 腰间系着一条亚麻原色短围裙——花店工作用。围裙口袋里插着一把淡蓝色柄的小剪刀。 | `#B8A890` |
| **手** | 修长的手指，指甲剪得很整齐。右手掌心有一道很浅的绿色污渍——长期握花茎擦出来的。 | 肤色同面部 |

### Shader 穿透标记
- **花店后巷的干枯矢车菊** `[SP]`（场景资源，非角色）：在后巷角落的灰色花盆中，几朵干枯的矢车菊花瓣保有极微弱蓝色 `#6090C0`，穿透饱和度约 10-15%。玩家靠近（< 1 tile）时可察觉。
- **阿莲手中的矢车菊**（觉醒后）：全彩蓝色 `#6090C0`。觉醒前在灰阶 Shader 下完全灰色，不穿透。

### 姿态
- **idle**：站在花架前，手里拿着一把矢车菊，正在用剪刀修剪花茎。动作缓慢而精确——每一根花茎都剪同样的长度。
- **行走**：步伐轻盈无声。走到下一个花架，拿起下一盆——继续修剪。
- **送花时**：如果有人站在她面前超过 5 秒，她会从手边拿起一束包好的矢车菊，双手递出——头微微低着。
- **觉醒后（蓝色恢复）**：不再修剪花茎。她站在花店门口，看着街道——流泪。眼泪为淡蓝色光点 `#80B0E0`。

| **面部** | 圆脸，红光满面——他是所有NPC中气色最好的。眼睛比正常人大一像素——不是画错，是他的眼睛真的有光。眉毛稀疏但永远向上挑着——永远在笑。 | 肤色 `#E0D0C0` / 眉毛 `#6A5040` |
| **上衣** | 米白色厨师上衣，领口有一圈浅灰的滚边。袖子卷到肘上——前臂上有面粉的白色痕迹。 | `#F0EDE0` / 滚边 `#C8B8A0` |
| **围裙** | 亚麻色帆布围裙，长度到膝盖。围裙正面永远有新面粉的白色掌印——他擦手从来只擦在围裙上。围裙左下角绣着一个小小的金黄色麦穗图案。 | `#C8B890` / 面粉印 `#F0E8D8` / 麦穗 `#D8B040` |

### 关键视觉标记
- **眼中的光**：他的眼睛是 2 px 而非其他 NPC 的 1 px——在像素美术中，这 1 px 的差距就是"有希望"和"没有希望"的区别。眼瞳 `#A08050`，高光点 `#F8F0E0`。
- **围裙上的面粉手印**：这是他最具标志性的视觉元素。新印子覆盖旧印子——密密麻麻。
- **手中永远有面团**：idle 动画中，他右手一直在揉一个金黄色面团——揉、翻、压、再揉。

### 姿态
- **idle**：半侧身站在烤炉前，右手反复揉捏一团面团。每揉三次——转身、打开烤炉门看看（摇头叹气）、关上、继续揉。
- **行走**：快步——但不是急，是"总有东西要烤"的那种快。走到哪里手里都端着一个烤盘。
- **送面包时**：弯腰（他比大部分人都矮），双手递上面团——"来！不热——但是能吃！"
- **觉醒后（黄色恢复）**：站在烤炉前，看着金黄色 `#E0B040` 的面包（面包皮 `#B08030`），不再揉面。手里端着面包——不送人，只是看着。烤炉透出暖黄光 `#F0D060`。

| **面部** | 凹陷的双颊，颧骨突出。眼睛下方有很深的阴影——不是熬夜，是恐惧。**瞳孔深处有一抹极淡的绿色荧光**——即使在全彩版中也很微弱。皮肤苍白——长期不见光。 | 肤色 `#D8D0C8` / 阴影 `#B0A098` / 眼瞳绿 `#40A040` `[SP]` |

### Shader 穿透标记
- **瞳孔绿光** `[SP]`：极其微弱的绿色荧光 `#40A040`，即使在彩色版中也只有 2 px。在全局灰阶 Shader 下以约 20% 原始饱和度穿透。正常 gameplay 视距下看起来只是"比灰色深一点的点"。在对话特写中可辨认。当 3 种颜色被找回后（绿色觉醒前），此绿光开始闪烁——频率约 2 秒一次。

### 姿态
- **idle**：蹲在屋角，双腿蜷起，双臂抱住膝盖。眼睛从两腿之间看出去。这是他唯一的安全姿势。
- **门缝说话时**：蹲在门后面，从门缝向外看——只有一只眼睛和半边嘴露在缝隙中。
- **行走**：几乎不走路。如果出门（仅在觉醒后），走路像在逃——快、轻、贴着墙。
- **手部动作**：他的右手永远在摩挲那把铁钥匙——不是在玩，是确认它还在。
- **觉醒后（绿色恢复）**：站起来了。不再蹲着。但身体的习惯性抽搐仍然存在——肩膀每隔几秒就微微一缩。

## NPC #5：小提琴手·薇拉（紫色·思念）

### 基本信息
- **年龄**：约28岁
- **体型**：纤细，肩窄。站姿极度端正——但是"被框住"的那种端正。脖子微微伸长——像在听什么很遥远的声音。
- **职业**：无

### 外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 深棕色长发，一丝不苟地盘成一个高髻。没有任何碎发。发簪是一根细长的米白色骨簪。 | `#4A3028` / 骨簪 `#E0D8D0` |
| **面部** | 瓜子脸，下巴很尖。眉毛细长，永远微微蹙着——不是生气，是"听"。眼神不看向任何人——永远望向远方（东偏北，画室的方向）。嘴唇自然闭合，没有任何弧度——不笑也不悲，是"等"的表情。眼睛为深紫灰色。 | 肤色 `#D0C0B8` / 眼睛 `#584868` |
| **连衣裙** | **核心视觉锚点**。及膝的深紫灰色长裙，袖子到手腕。裙摆整齐不皱——她极度在意自己的仪容。**裙摆右下角有亮紫色螺旋刺绣**——音符在空中旋转的轨迹。 | `#6A5878` / 刺绣 `#9060C0` `[SP]` |
| **小提琴** | 她每天架在左肩上的红棕色小提琴。琴身有光泽——她每天擦。琴弦是四根极细的半透明白色线。琴弓搭在弦上方 2px 的位置——永远不落下。 | 琴身 `#8A5040` / 琴弦 `#E8E0D8` / 琴弓 `#9A7060` |
| **手指** | 左手手指上有一道很浅的压痕——长期按琴弦留下的。右手的小指微微向外翘——那是持弓的惯性姿势。 | 肤色同面部 |
| **鞋** | 深灰棕色平底鞋，鞋头圆圆的。鞋面上有细小的磨损——她在广场上站立太久磨出来的。 | `#584838` |

### Shader 穿透标记
- **裙摆紫色螺旋刺绣** `[SP]`：亮紫色 `#9060C0` 的螺旋音符纹样。在全局灰阶 Shader 下以约 12-18% 原始饱和度穿透，呈现为与裙子布料之间极细微的差异。只有在她侧身或光线角度特殊时才能注意到。这是玩家在灰阶世界中最难发现的颜色残留。

### 姿态
- **idle**：站立，左肩架琴，下巴微微压着腮托。右手持弓，弓停在弦上方 2px——静止。整个身体只有一个地方在动：左手的手指——在琴弦上无声地、极其缓慢地变换按弦位置。她在"无声地演奏"。
- **行走**：只在来广场和离开广场时移动。步态像在走一条直线——不是真的有人画了线，是她在自己的轨迹里不愿意偏差一步。
- **觉醒后（紫色恢复）**：第一个完整拉完一曲的姿势——头微侧，眼睛终于看向琴——而不是远方。右手拉弓的动作流畅而缓慢。淡紫色光点 `#B080E0` 从琴弦飘出。

## 特殊NPC：老画家（源印载体）

### 基本信息
- **年龄**：不可判断——面容苍老但眼神是年轻人的。可能70岁，也可能不存在"年龄"。
- **体型**：极瘦。瘦到风能把他吹走——所以老唐给他送面包。
- **身份**：AI「织女·太一」在崩溃前留下的最后一段完整自我意识投射。他不是NPC——他是这个碎片唯一"清醒"的存在。

### 外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 纯白长发，散乱地披在肩上。刘海遮住了半边脸——不是造型，是懒得剪。 | `#E8E8E8` 白 / `#585858` 深灰 |
| **面部** | 颧骨极高，两颊深陷。但他有整个碎片中**最亮的眼睛**——不是有光，是"知道"。他的眼神让其他NPC害怕——因为被看穿的感觉。眼角有深深的笑纹——但他已经不笑了。 | 肤色 `#C8B8A8` / 眼睛 `#E0E0E0` |
| **上衣** | 一件纯白长罩衫——上面全是彩色颜料的溅痕和手指擦过的痕迹。袖子长的盖住了手背。领口歪在一边——露出锁骨。罩衫下摆有一行淡淡的字迹——他用颜料写的，但已经被洗掉了大半，只能看到："……不是……灰色……" | `#F0F0F0` / 颜料溅痕：红 `#C04040` / 蓝 `#6090C0` / 黄 `#E0B040` / 绿 `#40A040` / 紫 `#9060C0` / 白 `#F0F0F0`（各种颜色斑点，像被打翻的调色盘） |
| **裤子** | 深灰棕色布裤，松松垮垮。右边裤腿的膝盖处有一个破洞——跪在地上作画磨的。 | `#5A4A3A` |

### 关键视觉标记
- **手上的彩色颜料**：他的手指上覆盖着多层干涸颜料——红蓝黄绿紫白，层层叠叠。老画家的颜料是**整个碎片中唯一不受灰阶 Shader 影响的区域**（100% 穿透），即使在全局灰阶下，他手指上的颜料依然保有微弱但可见的色彩。这是他作为源印守护者的视觉特权。
- **罩衫上的颜料溅痕**：各色颜料斑点分布在罩衫上。在灰阶 Shader 下部分穿透（约 30-40%），形成"似乎有颜色但又不确定"的视觉效果。
- **织女画像**（揭幕后）：全彩——科幻化身的女性人像，织线般的数据纹理穿过衣纹，瞳孔中六色旋涡 = 源印符号。

### 姿态
- **idle**：坐在画架前，右手举着调色盘，左手悬在画布上方——永远差1px就落笔。他在犹豫什么？还是在等什么？这个姿势是他对玩家说的第一句无声的话。
- **转身时**：动作极慢——像是每个动作都在确认自己还活着。
- **揭幕织女画像**（六色觉醒后）：双手缓缓拉开灰布——然后退后一步。再退一步。然后站定。这是他第一次站直。
- **消失**（源印提取后）：从脚开始逐渐透明化——不是变白，是"变淡"。最后一瞬间，他会看着玩家——眼角有蓝色的泪光。然后什么都没有了。
