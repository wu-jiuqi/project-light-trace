## 0. 总体思路

### 0.1 为什么不用Godot做动画

原方案在Godot中用 AnimationPlayer + 多图层PNG驱动纸偶动画，每帧都需要精细编排。现在改用即梦AI文生视频，**直接输出成品视频片段**，Godot只负责播放视频文件。

### 0.2 生产方式

即梦AI单段视频约 5-10 秒，完整开场约 75-95 秒，需要 **10-12 个片段** 拼接。每段独立生成，后期用剪辑软件（剪映/CapCut/Premiere）拼接并配旁白音轨。

### 0.3 纸工动画的核心语言

这是最关键的认知转换：**生成的视频不应该像3D CGI动画，而应该像有人在拍摄一个纸偶剧场。**

| 错误理解 | 正确理解 |
|---------|---------|
| 摄像机在3D空间飞行 | 纸片层在固定镜头前移动、翻转、替换 |
| 物体smooth变形 | 纸片被剪开、撕开、叠放、抽走 |
| 粒子特效 | 纸屑、描图纸雾、线绳飘动 |
| 镜头虚化/景深 | 纸层间距产生的自然阴影偏移 |
| 发光特效 | 银箔纸反光、描图纸透光 |
| 爆炸/冲击波 | 纸片从中心向外撕开、飞散 |

### 0.4 全段统一风格锚点

以下关键词 **必须出现在每一段的正向提示词中**：

```
手作纸工拼贴风, 纸偶剧场, 层叠卡纸, 可见纸纤维,
不规则剪刀切边, 左上方单一光源, 右下方暖灰克制投影,
纸层厚度窄暗边, 粗纤维纸面, 灰白卡纸底,
非CGI, 非3D渲染, 非写实
```

以下关键词 **必须出现在每一段的负向提示词中**：

```
3D渲染, CGI动画, 写实, 塑料质感, 橡胶, 光滑矢量,
纯黑阴影, 过曝, 火焰爆炸特效, 真人面孔, 现代城市,
模糊背景, 镜头光晕, 粒子特效, 发光, 卡通, 日系动画, 美式动画
```

```
舞台两侧可见装订线绳的残段，轻轻晃动后静止。
```

### 第三段：万象·虚拟宇宙（约10秒）

> 对应设计：Scene 03  
> 画面：纸偶剧场全景展现「万象」虚拟宇宙的纸工化呈现——不是CGI数字空间，而是用纸张构建的宏大舞台

```
【即梦AI 文生视频 · 第三段】

纸偶剧场舞台的全景。深靛蓝色牛皮纸铺成宇宙背景，上面用层叠卡纸构建了一个宏大的纸工"宇宙"：

- 多条银箔纸条从中心向外辐射，像光之丝线，轻微翘起
- 灰白卡纸剪成的几何浮岛，以不同纸层高度悬浮，下方可见克制阴影
- 描图纸叠成的环形结构旋转——旋转是纸层逐片翻动实现的
- 旧书页折成的晶体塔楼，书页文字纹理隐约可见但不形成可读词汇
- 中央一颗用暖金色纤维纸剪成的四芒星，缓慢脉动——脉动通过纸片轻微上下浮动表达

左上方光源均匀照亮整个舞台，各层纸片的右下方投影形成自然的空间纵深感。

动作风格：纸片缓慢浮动、旋转、翻页，像有人在镜头外轻轻拨动纸偶舞台的各层。运动带有手工操作的微顿感。
```

### 第四段：470亿·闪帧（约5秒）

> 对应设计：Scene 04  
> 画面：画面中出现 470亿 的视觉概念——不是数字特效，而是大量纸片的快速翻动

```
【即梦AI 文生视频 · 第四段】

纸偶剧场舞台，深灰卡纸背景。画面中央，一张灰白卡纸卡片快速翻到正面——上面用旧书页拼贴出不完整的数字"47"，随后迅速被另一张卡片覆盖替换。

卡片以极快速度连续翻动、替换、叠压。每张卡片都是手工剪裁的异形纸片，边缘有不规则剪刀切边。部分卡片上隐约可见旧登记簿的书页纹理。

翻动节奏越来越快，越来越密集。卡片之间的纸层厚度在快速翻转中形成闪烁的阴影变化。

最后所有卡片同时落下，平铺在舞台底层，像一叠被翻完的登记簿。

动作风格：极快但每张纸片的翻动都能看到纸的物理特性——弯曲、翘角、落下时的轻微弹跳。不是数字闪烁。

负向提示词：
```

### 第五段：碎裂（约10秒）

> 对应设计：Scene 05  
> 画面：纸工宇宙被"撕裂"——这是全片情绪转折的核心段落，用撕纸替代CGI玻璃碎裂

**⚠️ 这是整部开场最重要的段落，优先测试本段。**

```
【即梦AI 文生视频 · 第五段】

纸偶剧场舞台，第三段的纸工宇宙全景处于静止状态。

突然——从四芒星中心开始，深靛蓝牛皮纸背景被从后方撕开。撕裂不是CGI特效，而是纸的真实撕裂：裂缝边缘露出粗糙纸纤维，裂缝呈不规则锯齿状向四周蔓延。

随着背景纸被撕开，上面粘贴的所有纸片开始脱落：
- 银箔纸条从裂缝处断裂、卷曲
- 灰白卡纸浮岛失去支撑，向下坠落——坠落中有纸片的飘荡感和轻微翻转
- 描图纸环形结构被撕成不规则碎片，飘散
- 旧书页晶体塔楼从折叠处散开，书页纷飞

六种颜色的纤维纸条——赤红、深蓝、金黄、翠绿、暗紫、米白——从裂缝中飘出，随即被"吸入"裂缝后方的黑暗中。

背景纸完全撕裂后，露出后方更深的灰黑卡纸底层。舞台上散落着纸屑、碎纸片和纤维粉尘。一盏微弱的暖光仍在左上方，照着这片纸的废墟。

动作风格：撕纸的物理感——裂口蔓延有纸纤维断裂的顿挫，碎片飘落有纸的轻盈和随机翻转。不是玻璃碎裂的硬脆感。

负向提示词：
3D渲染, CGI动画, 写实, 玻璃碎裂, 爆炸特效, 冲击波, 塑料,
光滑矢量, 纯黑, 纯黑阴影, 过曝, 火焰, 真人, 现代城市,
卡通, 日系动画, 美式动画, 粒子特效, 发光, 镜头晃动
```

```
【即梦AI 文生视频 · 第七段】
```

### 第九段：星图显现（约10秒）

> 对应设计：Scene 09  
> 画面：纸工星图——完整四芒星母图从纸层中浮现、展开，为标题做铺垫

```
【即梦AI 文生视频 · 第九段】

纸偶剧场舞台，灰黑卡纸底。舞台上散落的纸片碎片开始向中心汇聚——碎片不是磁吸式的CGI运动，而是像被无形的手一片片捡起、移动、拼合。

碎片在舞台中央拼成一个完整的四芒星轮廓。拼合过程中可见纸片之间的微小缝隙和不对齐——手工拼贴的自然痕迹。

四芒星由以下纸材构成：
- 主体：描图纸，半透明，可见纤维纹理
- 星辉方向：从中心向外辐射的银箔纸条
- 星点：散布在描图纸上的微小银箔圆片
- 背景：深灰卡纸底

星图停留约3秒后，整体向后退入舞台深处——"后退"表现为纸层间距缩小，星图缩小但仍保持完整。

底部保留字幕安全区。
```

### 第十段：标题揭晓（约8秒）

> 对应设计：Scene 10  
> 画面：纸板舞台上浮现标题「溯光计划」，最终定格

```
【即梦AI 文生视频 · 第十段】

纸偶剧场舞台，深灰卡纸底。第九段的四芒星缩小退入舞台后方，成为一个微弱的银箔光点。

舞台中央，四块粗纤维纸片从舞台底层逐块立起——每块纸片由手工剪刀裁切，边缘不规则。

四块纸片拼合后，表面浮现文字——文字不是打印或渲染的，而是由深色线绳在纸面上盘绕形成的。四块纸片合在一起显示：

「溯光计划」

线绳文字的细节：可见线绳的编织纹理，线绳在纸面上投下极细的克制阴影。纸片上还有旧书页的隐约纹理，但不影响文字可读性。

标题纸片停留约4秒。下方缓慢浮起一块较小的灰白卡纸片，上面用更细的线绳盘绕出副标题样式——保持空白，因为实际文字由Godot渲染。

最终画面定格：深灰卡纸舞台，标题纸片居中，四芒星光点在后方远处。

底部保留字幕安全区。

动作风格：纸片逐块立起有手工拼装的节奏感，线绳文字有编织的纹理细节。

负向提示词：
3D渲染, CGI动画, 写实, 塑料, 光滑矢量, 纯黑, 纯黑阴影,
过曝, 火焰, 真人, 电脑字体, 印刷文字, 发光文字,
卡通, 日系动画, 美式动画, 粒子特效, 发光, 镜头飞行
```

## 2. 使用流程

### 2.1 推荐生成顺序

```
第一步：【第五段·碎裂】（最关键，先定风格基调）
    ↓ 效果好 → 继续
    ↓ 效果不好 → 调试纸工语言，直到满意
    
第二步：【第九段·星图】（第二个视觉锚点）
    ↓ 

第三步：【第三段·万象】（复杂的全景纸工）
    ↓

第四步：其余段落按顺序生成
```

### 2.2 即梦AI参数统一设置

| 参数 | 设置 |
|------|------|
| 画幅 | 16:9 |
| 模型 | 选择最新可用版本 |
| 风格强度 | 中等偏上（按纸工风格锚点引导即可） |
| 时长 | 按每段建议时长设置 |

### 2.3 后期拼接

1. 将所有生成片段导入剪辑软件（剪映 / CapCut / Premiere）
2. 按顺序排列，片段之间使用 **硬切** 或 **极短叠化（0.3s）**，不搞花哨转场
3. 配入公司AI女声旁白音轨
4. 叠加环境音效（纸翻动声、撕纸声、轻微线绳摩擦声）
5. 导出为 `.webm` 或 `.ogv`（Godot 原生支持）
6. 在Godot中用 `VideoStreamPlayer` 节点播放

### 2.4 标题文字处理

第十段生成的标题纸片**不包含实际文字**——即梦AI生成的文字不可控。正确做法：
- 让即梦生成**空白标题纸片**（有线绳盘绕纹理的纸板）
- 在剪辑软件中用文字工具叠加「溯光计划」和「Project: Light Trace」
- 或在Godot中用 `Label` 节点覆盖在视频上方

## 3. 常见问题与应对

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 生成画面偏3D/CGI | 纸工语言不够强 | 在正向提示词中加倍强调"手作纸工拼贴风, 纸偶剧场, 层叠卡纸"，在负向提示词中加重"3D渲染, CGI动画" |
| 光影过于写实 | 光源描述像真实摄影 | 将光源描述改为"左上方暖色纸板打光"、"像台灯照亮纸板剧场" |
| 颜色过于鲜艳 | AI默认高饱和 | 加入"克制的纸材本色"、"灰白卡纸底"、"旧纸褪色感" |
| 碎片运动像CGI | 运动描述太"流体" | 加入"纸片飘落"、"手工移动的顿挫感"、"纸的物理翻转" |
| 出现可读文字 | AI自行生成文字 | 在负向提示词中明确"伪文字, 可读文字, 印刷体"，正向提示词中强调"旧书页纹理但不形成可读词汇" |
| 片段间风格不统一 | 每段独立生成 | 确保每段正向提示词都包含"0.4全段统一风格锚点"所列关键词 |
| 即梦AI单段时长不够 | 平台限制 | 将长场景拆分为2个子段，后期无缝拼接 |

## 4. 备选方案

### 方案B：文生图 → 图生视频

如果即梦AI文生视频对纸工风格响应不稳定，改用两步走：

**第一步**：用即梦「文生图」生成每段的关键帧（静态PNG），提示词使用本文档各段"画面"部分描述。

**第二步**：用即梦「图生视频」功能，以关键帧为起始图，添加轻微运动描述：
```
纸片缓慢浮动, 纸层轻微上下移动, 保持纸工质感不变,
不添加新元素, 仅对现有纸片施加极克制的运动
```

### 方案C：混合策略

对以下段落使用文生图 → 图生视频：
- 第三段（万象全景）：先锁定构图，再加运动
- 第九段（星图）：先锁定四芒星拼合状态，再加旋转

对以下段落直接文生视频：
- 第四段（闪帧）：纸片翻动本身就是运动的
- 第五段（碎裂）：撕纸运动是核心表现
