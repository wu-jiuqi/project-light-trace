# 碎片 #0001「启程之镇」— BGM 提示词

> **撰写日期**：2026-06-10
> **关联文档**：`design/story/0001/fragment_0001.md`
> **用途**：晨曦镇（DawnVale）探索阶段背景音乐
> **输出格式**：提交 AI 音乐生成工具（Suno / Udio / MusicGen）或交作曲家参考

---

Genre: Ambient Soundtrack / Papercraft Chamber Minimal
Tempo: 60 BPM（一秒一世界心跳——钟楼齿轮咬合频率正好是 60/11 ≈ 5.45 拍，暗示"时间在走但指针不动"）
Key: D dorian（D–E–F–G–A–B–C–D）→ D phrygian 渗透（Eb 侵入——"不对劲"的听觉信号）
Duration: 4:30–5:00（loopable，预计玩家停留 10-15 分钟）

---

## 一、情感叙事弧线

### 弧线结构（匹配玩家在镇内的心理旅程）

```
阶段                    音乐状态                    玩家心理
─────────────────────────────────────────────────────────
0:00–1:00 踏入        温暖、稳定、微微"太完美"       "好漂亮的小镇"
1:00–2:00 行走        重复感出现、细微偏离           "等等……不太对"
2:00–3:00 怀疑        不协和音渗透、断裂开始          "这里是假的"
3:00–4:30 发现        暗室气息、低频真相关键          "底下有东西"
4:30–loop 回归        回到 warm-but-aware 状态         "我知道了。继续走。"
```

## 二、乐器与音色配置

### Layer 1 — 纸艺基底（全程贯穿）

- **材质**：真实牛皮纸揉搓声（Foley 录音，经 Noise Gate 和混响处理后成为纹理层）— 每 8 秒一次循环，**和喷泉水声的 8 秒断点同步**
- **纸板轻敲**：用指尖轻敲灰色卡纸板——以 60 BPM 的 1/4 音符间隔出现——对应钟楼齿轮咬合的节奏暗示
- **手工纸剪切声**：极偶尔——每 30-45 秒一次细微的"剪刀裁纸"声——暗示"这个世界是被手工做出来的"

### Layer 2 — 永恒晨曦（0:00–全曲，主基底）

- **Felt piano（毛毡钢琴）**：单个音符悬浮在空气中——低音区 D、A 两个音，每 4 小节重复一轮——attack 极软（hammer strike at 40% velocity），sustain 漫长（约 3 秒 for each note）
- **处理**：加 20% wet 的板式混响（Plate Reverb，Decay 2.5s）——模拟"空旷小镇广场"的空间感
- **速度**：每个音之间间隔不等——前两个音之间 4 秒，第三个迟了 5.2 秒才来——**这是故意的**，暗示"脚本在执行但偶尔卡帧"

### Layer 3 — 电子设备的呼吸（0:00–全曲，极低混合）

- **电子底噪**：50Hz 低频嗡声（和光墙电流声同频）——混合在 -27dB，贯穿全曲
- **数据板蜂鸣**：极轻的 1200Hz 正弦波——每 11 秒一次短促的"哔"——对应林指导数据板刷新频率
- **处理**：不加混响——保持"干燥"——这是电子设备的声音，不属于"纸艺物理空间"

### Layer 4 — 永动但失真的钟（0:00–全曲）

- **钟楼齿轮**：金属咬合声——但经过时间拉伸处理（stretch to 200%），变成一段漫长的、形变的低频共振——每 11 秒完成一次完整循环
- **钟鸣**：单一铜钟音——**但永远不完整**。每次只敲击前 0.8 秒的声音就突然切断——像钟被卡住了。每 60 秒出现一次（对应 06:12 中的 12 分钟相位）
- **处理**：钟鸣切断后留下 3 秒的静默残响——这是空间中的"负空间"，对应钟楼台阶第 7 级没有声音的概念

### Layer 5 — 四层脚本化人声纹理（0:00–全曲，嵌在环境声中）

以下不是歌词——是经过 granulation 处理和降噪的人声片段，像"被遗忘的培训音频片段"悬浮在声场中：

- **林指导**：一句"欢迎来到溯光计划"——每 2 分钟浮现一次（和广场路人循环同频），granulation 处理后只剩下"溯……光……"两个字的气声，几乎听不见
- **陈技术**：极低的"没问题吧？"——随机出现在左声道深处（-30dB），暗示"培训师在远处另一个房间里对别人说话"
- **赵安保**：一段短促的电流声——然后一声极模糊的"停"——出现在右声道边界方向（pan +60～+80）

### Layer 6 — 暗室地鸣（仅在 3:00–4:30 出现）

- **18Hz 正弦波**：和设计文档中暗室的地鸣频率完全一致——从 -35dB 渐入到 -22dB
- **body feel**：这个频率在人耳听觉下限——玩家不会"听到"但会感觉到胸腔的轻微压迫

### Layer 7 — "颜色"的预兆（仅在 3:30–4:30 出现）

- **一丝极细的高频谐波**：在 18Hz 地鸣之上叠加一个 14000Hz 的泛音（几乎听不见）——极轻、极远。这是暗室地面裂缝透出的"彩色光"的听觉等效——在这个灰白世界中，颜色的第一个信号是**听不见的声音**

## 四、时间轴详细编排

```
00:00 — 开场（踏入晨曦镇）
  └ Felt piano: D (低音) 独奏 — 攻击极浅，持续 4 秒
  └ 50Hz 电子嗡声渐入（-30dB → -27dB）
  └ 纸板轻敲开始（60BPM）
  └ 远处城市白噪音已存在（非 BGM 层）— 模糊车流声

00:04 — 第一个音符回应
  └ Felt piano: A (低音) 加入 — 和 D 形成五度，温暖但空洞
  └ 牛皮纸揉搓声第一次出现（8 秒循环开始）

00:08 — 第二组音符
  └ Felt piano: D, A 重复 — 间隔变为 5.2 秒（轻微延迟）
  └ 钟楼齿轮拉伸声第一次出现（11 秒循环开始）

00:20 — 第一道细微裂痕
  └ Felt piano 第三个 D 音 — 轻微走调（下调约 15 cents）
   → D dorian 中突然掺入一个带 Eb 倾向的 D——"不对劲"的第一步

00:30 — 脚本化世界开始呈现
  └ 数据板蜂鸣第一次出现（1200Hz，11s 循环）
  └ 剪刀裁纸声第一次出现
  └ Granulated 人声："溯……光……"（几乎听不见）

00:45 — 第一个完整循环结束
  └ 所有基底元素就位
  └ Felt piano 回到标准 D–A 五度——但"走调"已经被玩家无意识记住了

01:00 — 第一阶段结束 / 第二阶段开始
  └ 截断铜钟第一次出现（只响前 0.8s，突然切断）
  └ 暗示："时间"在这个世界里被什么东西打断了
  └ Felt piano 间隔进一步拉长——现在每 6 秒一个音

01:15 — 重复感
  └ 牛皮纸揉搓循环到第 9 轮——玩家开始注意到"这个声音一直在重复"
  └ 和喷泉水声的 8 秒断点同步——两种循环形成共振

01:30 — Eb 首次完整出现
  └ Felt piano 低音区——不是 D 了，是 Eb
  → D dorian 首次被 phrygian 渗透——"暖"开始变"冷"
  └ 50Hz 电子嗡声微微上升至 52Hz（+2Hz 偏移，对应"不精确"主题）

02:00 — 第二阶段结束 / 第三阶段开始（怀疑阶段）
  └ 截断铜钟第二次出现——切得更快（前 0.5s 就切断）
  └ 钟楼齿轮拉伸声开始出现相位失真——声音变"糊"
  └ Felt piano 偶尔跳过一拍——像脚本跳帧

02:15 — 陈技术的远声
  └ Granulated 人声："没问题吧？"——左声道深处
  └ 和玩家此时的心理状态对应——"有问题，当然有问题"

02:30 — 边界暗示
  └ 赵安保的"停"——右声道边界方向（pan +70）
  └ 50Hz 嗡声——此刻 FFT 分析应能看到 50Hz 主波 + 一个微弱的 18Hz 副波（暗室地鸣的前兆——但此时听不见）

03:00 — 第三阶段结束 / 第四阶段开始（发现阶段）
  └ 18Hz 地鸣渐入（-35dB → 逐渐增强）
  └ Felt piano 音符数量减少——从每 6 秒一个变为每 12 秒一个——让出空间给低频
  └ 纸板轻敲仍在继续——但音量从 -12dB 降至 -18dB——退为背景
  └ 截断铜钟第三次——这次它响了整整 1.2 秒才被切断（几乎完整）——暗示"接近真实了"

03:30 — 暗室门开了
  └ 18Hz 地鸣达到 -22dB——身体可以感觉到
  └ 14000Hz 泛音出现——极轻，像一根银线悬浮在空气中最高的位置
  → 这是暗室地面裂缝透出的"彩色光"——在听觉上是第一个不属于灰度世界的声音
  └ 所有循环声（纸揉搓、蜂鸣、齿轮）同时停止 2 秒——**全曲唯一的完全静默**
  → 这段时间对应玩家踏入暗室的瞬间——"真实的东西不需要循环"

03:32 — 静默结束
  └ Felt piano 回到纯净的 D 音——不再走调
  └ 但只弹一个音——然后沉默
  └ 18Hz 地鸣继续

04:00 — 真相在场
  └ 14000Hz 泛音渐强到 -28dB——最"亮"的时刻——然后开始 fade out
  └ 18Hz 地鸣从 -22dB 渐弱到 -30dB
  └ Granulated 人声（林指导的"祝您好运"）——最后一次浮现——比所有之前的浮现都清晰约 3dB
  └ 纸板轻敲恢复——但节奏微微变了——从精确的 60BPM 变为 58BPM——"不再是完美的脚本了"

04:30 — 回落 / 开启循环
  └ Felt piano 回到 D–A 五度——温暖但在"已知真相"的前提下，温暖变得复杂
  └ 牛皮纸揉搓恢复（8s 循环）
  └ 50Hz 嗡声恢复
  └ **不再有走调**——但也不再"天真"
  → 循环点：04:30 无缝回到 00:20（跳过开场建立阶段，从"已经走过了"的状态开始）
```

---

## 五、循环方案

**Loop Point**：04:30 → 00:20

- 04:30 的末尾（所有基底元素就位、不再走调、D dorian 纯净）与 00:20 的早期状态（所有元素刚就位、尚未出现走调）交接
- 交叠 3 秒 crossfade
- 循环后不再包含 00:00–00:20 的"纯开场建立"阶段——那段只在首次踏入时播放一次

**首次播放 vs 循环播放**：

| 段落 | 首次 | 循环 |
|------|------|------|
| 开场建立 (00:00–00:20) | ✓ 播放 | ✗ 跳过 |
| 行走阶段 (00:20–01:00) | ✓ | ✓ |
| 怀疑阶段 (01:00–03:00) | ✓ | ✓ |
| 发现阶段 (03:00–04:30) | ✓ | ✓（每轮循环都触发——因为暗室地鸣是"环境信息"而非"剧情事件"） |
| 完整截断铜钟（03:00 处） | 1.2 秒 | 1.2 秒（保持不变——暗示"每次靠近真实，都能得到更多"） |

## 六、与已有声音设计的融合

本 BGM 需要与碎片 #0001 的现有环境音效共存（参见设计文档第十章）：

| 设计文档中的声音元素 | BGM 中的应对 |
|---------------------|-------------|
| 远城市白噪音（循环） | BGM 不包含城市白噪——此层由游戏环境音效系统独立播放，和 BGM 分层 |
| 喷泉水声 8 秒断点 | BGM 的牛皮纸揉搓循环同样是 8 秒——两种循环共振，强化"一切都是循环"的感觉 |
| 数据板蜂鸣（林指导） | BGM 中 1200Hz 的电子蜂鸣使用的是与游戏 SFX 不同的频率（BGM 用 1200Hz，游戏 SFX 可能用 800Hz），避免听觉混淆 |
| 50Hz 光墙电流声 | BGM 中的 50Hz 嗡声与光墙 SFX 同频——玩家在靠近光墙时会感受到两种 50Hz 的叠加——物理上更响，心理上"边界更近了" |
| 暗室地鸣（18Hz） | BGM Layer 6 的 18Hz 与游戏暗室 SFX 的 18Hz 完全同频——二者叠加时产生谐波增强——物理感受翻倍 |
| 日晷指针旋转（石金属摩擦声） | 不在 BGM 中出现——留给 SFX 系统独占——确保交互有清晰的听觉反馈 |
| 指针正确（66°）—单声低频钟鸣 | 不在 BGM 中出现——留给 SFX 独占——这是世界级的叙事信号，不能被 BGM 覆盖 |
| 暗门开启（齿轮+石材滑动） | 不在 BGM 中出现——留给 SFX 独占 |
| 绝对安静（无风无鸟） | BGM 在"绝对安静"段落（03:30–03:32）不添加任何声音——尊重设计的沉默 |

## 七、技术规格

| 参数 | 值 |
|------|-----|
| 格式 | 44.1kHz / 24bit / 立体声 WAV master → OGG Vorbis q5 |
| 时长 | 4:30（首次播放）/ 4:10（循环播放，从 00:20 开始） |
| 峰值电平 | -3 dBFS（为 SFX 保留余量） |
| 综合响度 | 约 -20 LUFS（比主界面主题曲 -18 LUFS 更轻——因为需要和大量环境 SFX 共存） |
| True Peak | -1.0 dBTP |
| 动态范围 | LRA 约 12 LU（比主界面主题曲更大——因为包含静默段落） |
| Godot bus | "Music" bus（独立于 "Master" bus 上的 SFX 和旁白） |
| Godot 音量 | -8 dB（在 Music bus 上进一步压低——BGM 在此场景是氛围层次，不是主角） |

## 八、风格参考

| 作品 | 参考元素 |
|------|---------|
| **Disasterpeace — "FEZ" OST** | 数字温暖 + 模拟裂痕 + 极简重复。特别参考 "Home" 和 "Compass" 中"稳定中有不稳定的微偏移" |
| **Alva Noto + Ryuichi Sakamoto — "Vrioon"** | 钢琴单音悬浮 + 电子纹理 + 极低混合。参考"极小事件被放大到听懂"的感觉 |
| **The Caretaker — "An Empty Bliss Beyond This World"** | 78rpm 唱片 Degradation 美学——不是复制其风格，而是借鉴"听起来被时间磨损了"的质感 |
| **实际牛皮纸录音（Foley）** | 曲目中的纸纹理不采样已有音乐——必须亲自录制。揉搓牛皮纸、轻敲灰卡纸、裁剪刀划过纸面——三者在不同麦克风距离下录制（10cm / 50cm / 1m），然后分层混合 |
| **钟楼齿轮录制（可选）** | 如果有条件：在真实的机械钟内录制齿轮咬合声——然后时间拉伸至 200%、添加卷积混响——这是"晨曦镇钟楼"的声音原型 |

## 九、严禁项

- ❌ 禁止传统弦乐群/交响编制——这是纸艺小镇，不是史诗叙事
- ❌ 禁止完整的旋律线——BGM 不应该"唱一首歌"——应该是"世界在呼吸时的声音"
- ❌ 禁止鼓组/节奏乐器（除纸板轻敲外）——纸板本身就是唯一的"节奏乐器"
- ❌ 禁止大调明亮音阶——D dorian/D phrygian 的模糊性是核心——不可偏向 C major 或 G major 的"阳光感"
- ❌ 禁止低于 -30dB 的人声被清晰听到——granulated NPC 人声必须嵌在"几乎听不见"的层次
- ❌ 禁止合成器 pad——唯一允许的合成音是 18Hz 正弦波和 14000Hz 泛音（都是科学级纯净信号，不是"pad"）
- ❌ 禁止出现任何完整小节——乐句长度应始终略微不对称（4.2 秒、5.2 秒、3.8 秒）——对应"脚本不完美"主题
- ❌ 禁止在 03:30–03:32 的完全静默段落插入任何声音（包括混响尾音）

## 十、叙事功能

这段 BGM 不是在"配乐"——它本身就是叙事：

- 在 0:00–1:00 阶段告诉玩家："这是一个漂亮的地方——但你不确定为什么它让你不安"
- 在 1:00–3:00 阶段告诉玩家："那些细微的偏差不是你的幻觉——它们是有意为之的"
- 在 3:00–3:32 阶段告诉玩家："这里有真实的东西——就在那个暗室里——它不是公司建的"
- 在 3:32–4:30 阶段告诉玩家："你已经知道真相了。现在——带着这个真相，继续走在这个假的世界里"

它和设计文档 2.4 节「违反物理规则的可疑细节」是同一层叙事——不是剧情对白，是环境在**自己说话**。
