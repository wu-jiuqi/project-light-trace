# 《溯光计划》纸工拼贴 Style Bible

> 版本：V1.0  
> 状态：生产基线  
> 适用范围：全项目 UI、星图、碎片世界、角色、道具与异常演出

## 1. 视觉定位

项目统一采用：

**俯视 RPG + 整片纸偶 + 高分辨率纸片资产 + 局部超现实拼贴。**

常态世界应像可游玩的纸工绘本或桌面纸偶剧场。异常节点允许突变为密集、超现实的混合媒介拼贴，但不能长期压过路线、文字和交互提示。

现有 Godot 原型保留 `16px` 逻辑网格、房间结构和碰撞逻辑。纸工资产使用更高分辨率主文件，逐步替换像素占位资源。

## 2. 不可变规则

### 2.1 材质族

正式资产只从以下材质族中选择：

| ID | 材质 | 主要用途 | 限制 |
|---|---|---|---|
| `kraft` | 牛皮纸 | 木材、泥土、普通建筑、箱子 | 避免大面积高饱和色 |
| `gray_card` | 灰白卡纸 | 天空、石材、灰布、雕像 | 允许压痕和磨损 |
| `book_page` | 旧书页 | 信件、线索、招牌、异常文字 | 正式 UI 中禁止不可读伪文字 |
| `fiber_paper` | 粗纤维纸 | 人物皮肤、衣物、布帘 | 纤维纹理不能盖住五官 |
| `tracing_paper` | 描图纸 | 雾、裂缝、源印、星图碎片 | 控制透明度，不能影响交互可读性 |
| `foil` | 银箔纸 | 星图、源印、少量高价值标记 | 只用于强调 |
| `fastener` | 胶带、线绳、铆钉 | 固定纸层、局部装饰 | 每个主体最多选择两类 |

禁止使用：

- 光滑矢量边缘。
- 塑料、橡胶或写实 3D 玩具质感。
- 将普通插画简单叠加纸张纹理后当作纸工资产。
- 每张资产使用完全不同的纸纹和光源方向。

### 2.2 裁切边缘

- 外轮廓必须保留轻微不规则裁切。
- 常态资产使用剪刀切边；异常演出可以使用明显撕边。
- 角色脸部、手部和关键道具不能因撕边损失识别度。
- 纸层厚度使用窄暗边表达，不使用厚重写实倒角。

### 2.3 光源与阴影

- 全项目默认光源来自左上方。
- 纸片投影向右下方偏移。
- 普通纸片投影保持克制；悬浮层、UI 浮层和异常演出可增加偏移。
- 阴影颜色偏暖灰或冷灰，不使用纯黑。

| 层级 | 用途 | 建议阴影偏移 |
|---|---|---|
| `L0` | 地面纸层 | `0-1px` |
| `L1` | 地面装饰、矮道具 | `1-2px` |
| `L2` | 建筑、角色、主要道具 | `2-4px` |
| `L3` | 前景遮挡物、UI 卡片 | `4-8px` |
| `L4` | 异常拼贴悬浮层 | 按演出单独定义 |

## 3. 尺度与导出

### 3.1 环境

- Godot 逻辑网格继续使用 `16px`。
- 环境主文件按 `4x` 制作：每逻辑格对应 `64px`。
- 建筑、地面、道路、花架等资产分模块导出，不生成整张可玩地图。
- 气氛板和过场图可以是完整画面，但不得混入 TileMap 资产目录。

### 3.2 角色

- 普通角色主画布建议 `128x192px`。
- 运行时显示尺寸约为 `32x48` 世界单位。
- 脚底锚点统一放在画布底边中央。
- 碰撞盒只覆盖脚下区域，不随纸偶外轮廓扩大。
- 非对称元素必须保留：例如玩家左肩信使包、围裙污渍、裙摆刺绣。

### 3.3 文件格式

- 正式纸偶、道具和模块：透明背景 PNG。
- 描图纸和半透明效果：PNG。
- 工作底稿可以保留高分辨率 PSD、KRA 或源图，但不提交到运行时目录。
- 所有正式资源使用小写英文、数字和下划线命名。

## 4. 整片纸偶动画

角色不拆肢体，不制作复杂骨骼。基础表现来自完整纸片的轻量变换：

| 状态 | 表现 |
|---|---|
| `idle` | 极轻微呼吸缩放与慢速摇摆 |
| `walk` | 左右倾斜、垂直弹跳、阴影错位 |
| `talk` | 降低摇摆幅度，按角色性格保留轻微停顿 |
| `happy` | 替换独立开心姿态纸片，可增加短促弹跳 |
| `sad` | 替换低头或收缩姿态纸片 |
| `panic` | 替换恐慌姿态纸片，提高小幅抖动频率 |
| `awakened` | 替换觉醒姿态纸片，配合颜色恢复 Shader 和局部 FX |

玩家需要 `down`、`up`、`left`、`right` 四个方向。NPC 只制作实际会在场景中出现的方向和剧情姿态。左右方向不能直接镜像。

命名格式：

```text
<character>_<pose>_<direction>.png
<character>_<pose>_<direction>_sp_mask.png
```

示例：

```text
player_idle_left.png
player_walk_right.png
blacksmith_idle_down.png
blacksmith_awakened_down.png
blacksmith_idle_down_sp_mask.png
```

## 5. 灰阶、残留色与觉醒

- 所有基础资产只维护彩色版本。
- 初始灰白世界由运行时 Shader 去饱和处理。
- 需要保留微弱色彩的区域使用同尺寸独立 Mask，文件后缀为 `_sp_mask`。
- 颜色觉醒通过 Shader 参数插值恢复，不维护重复的灰阶图。
- 觉醒后的额外火焰、花瓣、裂缝、音符和源印使用独立 FX 纸片。

碎片 #0762 的六色映射：

| 颜色 | 区域 | 常态残留 | 觉醒表现 |
|---|---|---|---|
| 红 | 铁匠铺 | 老霍围裙烫痕 | 火焰、炉光、围裙污渍恢复 |
| 蓝 | 花店与后巷 | 干枯矢车菊 | 花瓣、泪光、衣裙恢复 |
| 黄 | 面包房 | 无明显残留 | 烤炉暖光、金黄面包 |
| 绿 | 墓园 | 老崔瞳孔微光 | 苔藓、嫩芽、裂缝 |
| 紫 | 广场 | 薇拉裙摆刺绣 | 音符纸片与琴弦光点 |
| 白 | 全域 | 白色雕像 | 雕像碎裂、天空恢复、全域转彩 |

## 6. 异常拼贴规则

高强度拼贴只用于关键节点：

- 颜色觉醒。
- 墓园绿色裂缝。
- 灰布揭幕与源印解码。
- 后续 Boss 或规则崩塌演出。

允许素材：

- 旧报纸、旧照片、书页、印章、铅笔线。
- 不合理透视、重复眼睛、钟表、餐具、文字碎片。
- 线绳、胶带、撕裂层、半透明描图纸。

限制：

- 常态探索路线必须始终可读。
- 异常文字只能作为纹理，不承担必须阅读的信息。
- 关键交互按钮和字幕区域必须留出稳定负空间。

## 7. UI 与星图

所有外层 UI 后续统一纸工化：

| 系统 | 设计语言 |
|---|---|
| 标题页 | 多层纸板舞台、撕纸标题、拼贴按钮 |
| 星图 | 深色纸板底、描图纸碎片、银箔纸修复纹、裁切四芒星 |
| 对话框 | 卡纸框、胶带固定的姓名条、留白充足的正文区 |
| 背包 | 旅行者剪贴簿、卡纸收纳槽、印章式图标 |
| 暂停菜单 | 夹在剪贴簿中的纸质标签页 |
| 按钮 | 裁切卡纸，使用位置、阴影和色差区分状态 |

可读性优先级：

1. 正文与按钮文字。
2. 当前选中状态。
3. 功能分组。
4. 纸纹、胶带和装饰。

禁止让纸纹覆盖小字号正文。所有按钮必须有 `normal`、`hover`、`pressed`、`disabled` 四种明确状态。
