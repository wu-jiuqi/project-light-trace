## 视觉一致性约束

为确保13个场景的视觉连贯性，所有提示词遵循以下统一约束：

| 约束项 | 规范 |
|--------|------|
| **风格** | 纸雕暗黑童话（Papercraft Dark Fairy Tale）—— 平面分层、手工质感、故事书插图感 |
| **禁止项** | 透视深度、真实阴影、体积光、3D感、雕塑感、玩具感、照片感、微缩模型感 |
| **色彩基调** | 暖琥珀金（暮光） + 深暮紫蓝（暗影） + 米白泛黄（纸张） + 褪色深红（书） + 褪色靛蓝（衣装） |
| **角色视觉** | 纤细木偶比例、小手小脚、风格化剪影、面孔由纸雕组件拼合 |
| **环境视觉** | 建筑由层叠纸板/瓦楞纸构成、可见缝线、纸折装饰 |
| **纸张质感** | 所有表面必须呈现纸张肌理——纤维纹理、毛边、折痕、缝线痕迹 |
| **灯光处理** | 暮光以琥珀色纸层叠加表现、暗影以深紫色纸层叠加表现——禁止体积光与投影 |
| **色彩限制** | 每场景严格控制在4-5色内——禁止全色域 |

## 使用建议

1. **生成顺序**：建议从 S01（核心环境）和 S05/S07（核心道具/手部特写）开始，先确定视觉基准。
2. **角色一致性**：老教师的视觉形象在所有含角色的场景（S01/S03/S08/S09/S12/S13）中必须保持一致——建议首先生成 S01 作为角色基准图，后续场景以 S01 的角色形象为参考。
3. **色彩校准**：S01 和 S10 分别建立了"黄昏站台环境"和"时钟道具"两个色彩基准，生成其他场景前应先确认这两个场景的色彩方向。
4. **画面尺寸**：建议全部使用 `1280x896`（横版）以适配游戏内 16:9 显示比例；如需竖向特写可改为 `1024x1536`。
5. **情感曲线**：S01-S03 铺垫 → S04-S06 深化（记忆层打开）→ S07-S09 转折（从擦除到抱书）→ S10-S11 外化（时钟·书在长椅）→ S12-S13 收束（背影·闭眼）。视觉节奏应与情感曲线同步。
