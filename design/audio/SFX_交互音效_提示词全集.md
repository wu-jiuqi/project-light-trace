### SFX-UI-01：按钮悬停

```
文件名：ui_button_hover.wav
时长：约 0.15 秒
类型：合成 + Foley 混合
优先级：P0
```

**声音描述**：指尖轻触卡纸板表面的声音——极短的纸张摩擦声 + 微弱的静电放电。不是"咔哒"键程声——是触碰，不是按下。

**提示词（Foley 实录说明）**：
```
指尖轻触灰色卡纸板（gray_card）表面。
接触面积约 1cm²，接触时间 < 0.1 秒。
麦克风距离：5cm，心形指向。
录制 20 次，选择触感最一致的那次。
保留自然的纸纤维吸附声——不加混响。
```

**音高/处理**：原始录音 → 轻微高频衰减（LPF @ 8kHz）→ 峰值 -12dBFS。

**叙事功能**：纸工世界的材质一致性——按钮不是像素，是纸片。

### SFX-UI-02：按钮确认（按下）

```
文件名：ui_button_confirm.wav
时长：约 0.3 秒
类型：Foley 实录
优先级：P0
```

**声音描述**：纸板被轻盈地按下的声音——两层牛皮纸（kraft paper）之间发生微小摩擦后，纸板边缘触碰到底层支撑面的"笃"声。类似用手指轻敲一本精装书的封面，但更脆、更薄。

**提示词（Foley 实录说明）**：
```
将两张 A6 尺寸的牛皮纸（kraft paper，约 0.3mm 厚）叠放在木桌上。
用食指以约 200g 力道垂直压下——快到接触面但不过度用力。
录制接触瞬间的"笃"声 + 纸张移位的微摩擦尾音。
麦克风距离：15cm，全向。
后期：去除 200Hz 以下的桌面共振。
```

**音高/处理**：No pitch shift。峰值 -6dBFS。保留纸张尾音（约 0.05 秒的纤维静默）。

**叙事功能**：确认 = "纸张世界接受了一个指令"。

### SFX-UI-03：按钮取消/返回

```
文件名：ui_button_cancel.wav
时长：约 0.25 秒
类型：合成 + Foley 混合
优先级：P0
```

**声音描述**：一个更柔和、更短暂的反向动作——纸板滑回原位 + 极短的电子音（800Hz 正弦波，0.08 秒）。

**制作方式**：
```
纸板部分：同 SFX-UI-02，但录制"松开"而非"按下"瞬间。
电子部分：800Hz 正弦波，0.08 秒，attack 3ms / decay 50ms，-18dBFS。
混合：纸板 -8dBFS，电子音 -18dBFS。
```

**叙事功能**："取消" = 机械层（系统）在撤回纸工层的状态。

### SFX-UI-04：通知/消息到达

```
文件名：ui_notification.wav
时长：约 0.5 秒
类型：合成
优先级：P0
```

**声音描述**：林指导数据板刷新时发出的电子音——1200Hz 正弦波，两次短"哔"，间隔 0.15 秒。第一次音量 -12dBFS，第二次 -15dBFS。类似老式寻呼机的提示音，但更干净。

**合成参数**：
```
波形：1200Hz 纯正弦波
第一声：attack 2ms，sustain 80ms，decay 30ms，-12dBFS
静默：150ms
第二声：attack 2ms，sustain 60ms，decay 30ms，-15dBFS
不加混响——保持"干燥"的电子感。
```

**叙事功能**：碎片 #0001 中陈技术发来半解密提示、林指导发来进度消息时触发。

### SFX-UI-05：源印显现

```
文件名：ui_source_mark_reveal.wav
时长：约 1.5 秒
类型：合成 + Foley 混合
优先级：P0
```

**声音描述**：一段持续约 1.5 秒的仪式性声音——以 30Hz 低频开始（0.3 秒），渐变为 60Hz（0.3 秒），再分叉为两个频率（120Hz + 0.8 倍频泛音 96Hz），最后以极轻的银箔触碰声结束。听起来像"世界被解锁了"。

**合成参数**：
```
0.0-0.3s：30Hz 正弦波，从 -30dBFS 渐入到 -12dBFS
0.3-0.6s：30Hz→60Hz 滑音（frequency glide）
0.6-1.2s：60Hz 分叉为 120Hz（-15dBFS）+ 96Hz（-20dBFS）双频，持续 0.6s
1.2-1.5s：双频渐弱 → 银箔纸轻触声（Foley，指尖触碰铝箔，-8dBFS，0.15s）
```

**叙事功能**：碎片通关时触发——钟楼暗门开启、晨曦之印显现——此后星图更新、新碎片解锁。

### SFX-UI-06：线索收集（物品拾取）

```
文件名：ui_clue_collected.wav
时长：约 0.4 秒
类型：Foley 实录
优先级：P0
```

**声音描述**：一张对折的牛皮纸被展开的声音——先是轻微的纸纤维分离声，然后是纸张展开时低频的空气振动声（the "whoomp" of a paper unfolding）。类似打开一封信。

**Foley 实录说明**：
```
将一张 A5 牛皮纸对折一次。录制"展开"的完整动作：
1. 手指捏住折角，缓慢拉开——捕捉纸纤维分离的撕裂感（但不要真的撕破）
2. 展开到一半时停顿 0.1 秒——然后快速完全展开
麦克风：15cm，心形。录制 10 次，选用纤维声最清晰的那次。
```

**叙事功能**："拾取"在纸工世界不是捡起来——是"发现了一张被折起来的纸，打开了它"。

### SFX-UI-07：背包打开/关闭

```
文件名：ui_backpack_open.wav / ui_backpack_close.wav
时长：各约 0.5 秒
类型：Foley 实录
优先级：P0
```

**声音描述**：
- 打开：一个纸质档案夹的封面被翻开——牛皮纸封面与底层纸板分离的粘滞声 + 翻页的空气振动。类似翻开一本厚重的老式账簿。
- 关闭：封面合上——更快的"啪"声，比打开短约 0.2 秒。牛皮纸与牛皮纸直接接触的闷声。

**Foley 实录说明**：
```
打开：使用一个 20 页左右的纸板档案夹。手握封面缓慢翻起——
      录制封面与内页分离时的粘滞声（持续约 0.3 秒）+
      封面完全翻到另一侧时与桌面的轻触声（0.1 秒）。
关闭：封面快速合上——直接录制两张牛皮纸的接触闷声（0.2 秒）。
麦克风：20cm，全向。
```

### SFX-FRAG01-01：日晷指针旋转

```
文件名：sfx_sundial_rotate.wav
时长：约 0.3 秒（循环用）
类型：Foley 实录
优先级：P0
```

**声音描述**：石制日晷指针在石制底座上旋转时发出的摩擦声——中低频的石头与金属（青銅）之间的干涩咬合声，每个刻度点有极短的"咔"（金属触碰到凹槽）。类似在老式石磨上转一把铜钥匙。

**Foley 实录说明**：
```
使用一块约 10cm×10cm 的花岗岩板 + 一支 10cm 长的铜管（模拟青铜指针）。
铜管一端压在花岗岩上，以 15° 角缓慢旋转。
录制连续的石头-金属摩擦声（基频约 200-400Hz）。
在每旋转约 5° 时，用铜管轻敲花岗岩表面——录制"咔"的瞬态声。
后期：将"咔"声均匀间隔排列（每 5° 一个刻度点），分为 12 等分。
```

**叙事功能**：对应日晷 72 个刻度（360°/5°），从 12° 旋转到 66° 的 11 个刻度点各触发一次"咔"。

---

### SFX-FRAG01-02：日晷错误反馈（接近但不正确）

```
文件名：sfx_sundial_wrong.wav
时长：约 0.6 秒
类型：合成 + Foley
优先级：P0
```

**声音描述**：指针短暂发橙光→回弹的声音。石金属摩擦声中断 + 一段渐弱的 400Hz 嗡声（持续 0.3 秒，从 -12dBFS 衰减到 -30dBFS）+ 指针金属弹回原位的清脆微响。

**合成参数**：
```
0.0-0.1s：齿轮卡住声——石金属摩擦声突然中断（just a truncation of the rotate SFX）
0.1-0.4s：400Hz 正弦波，从 -12dBFS 渐弱到 -30dBFS（"橙光"的听觉等效）
0.4-0.6s：铜管弹回——极短（0.05s）的金属共振，1.5kHz 峰值
```

**叙事功能**：转到 60° 时触发——"接近了，但不完全对"。

### SFX-FRAG01-03：日晷正确（66°）

```
文件名：sfx_sundial_correct.wav
时长：约 2.0 秒
类型：Foley + 合成
优先级：P0
```

**声音描述**：单声低频钟鸣——但不是楼顶那口大钟（那口钟永远敲不响），而是从钟楼地基深处传来的**另一口钟**的声音。低沉的共鸣（基频约 110Hz）持续约 1.5 秒然后逐渐消失，混有微弱的齿轮咬合声。类似把耳朵贴在地面上听到的地下钟声——闷、暖、远。

**制作方式**：
```
Foley：用软木槌轻敲一口小型铜碗（直径约 15cm）的边缘——录制共鸣声。
       将录音降调 2 个八度（pitch -24 semitones）→ 110Hz 基频。
       添加板式混响（Decay 2.5s，wet 40%）。
叠加：钟楼齿轮拉伸声（同 BGM Layer 4 的齿轮声，但仅持续 0.5 秒作为"引子"）。
      齿轮声 → 0.3 秒交叉淡入到钟鸣。
```

**叙事功能**：指针校准正确——暗门开启前的解锁信号。

---

### SFX-FRAG01-04：暗门开启（钟楼底层）

```
文件名：sfx_chamber_door_open.wav
时长：约 3.0 秒
类型：Foley 实录 + 合成
优先级：P0
```

**声音描述**：三段式连续声音——① 地下齿轮咬合声从远处传来（0.0-0.8s）；② 沉重的石材在地面上滑动的低频摩擦声（0.5-2.5s）；③ 石材停止时的闷响 + 极轻的灰尘落下声（2.5-3.0s）。

**Foley 实录说明**：
```
① 齿轮声：使用钟楼齿轮拉伸录音（同 BGM Layer 4），不做拉伸处理——
           原始速度播放 0.8 秒。加 40% wet 的卷积混响（模拟"从地板下传来"）。
② 石材滑动：在水泥地面上拖动一块约 5kg 的粗糙石板（约 30cm×30cm）。
           录制滑动全程——基频约 80-150Hz 的低频摩擦 + 高频的砂砾碎裂声。
           加速度曲线：0.5-1.5s 加速启动 → 1.5-2.2s 匀速滑动 → 2.2-2.5s 减速停止。
③ 停止：石材停下的闷响 + 一张薄纸被轻轻撒落的声音（灰尘落下）。
```

**叙事功能**：解密完成——暗室是本碎片中唯一"真实物理空间"的解锁。

### SFX-FRAG01-05：合规度变化（升/降）

```
文件名：sfx_compliance_change.wav
时长：约 0.3 秒
类型：合成
优先级：P0
```

**声音描述**：
- 合规度**下降**：电子蜂鸣——800Hz→600Hz 的下降滑音（frequency glide），持续 0.2 秒，-15dBFS。像一台设备在报告"警告"。
- 合规度**触底（L5/L6）**：同一蜂鸣 + 叠加一次 50Hz 的电流脉冲（0.15 秒，-10dBFS）——光墙的电流声瞬间侵入。暗示公司系统在"惩罚"你。

**合成参数**：
```
下降版：
  800Hz 正弦波 → 600Hz 滑音，0.2 秒
  -15dBFS，无混响
触底版（追加层）：
  50Hz 正弦波脉冲，0.15 秒
  attack 5ms，decay 100ms
  -10dBFS，与 600Hz 尾巴重叠 0.05 秒
```

**叙事功能**：MONITORED 模式的听觉反馈——合规度条变化时触发，通过音高滑落方向告知玩家"安全/危险"。

### SFX-UI-10：错误/禁止操作

```
类型：合成
```

### SFX-FRAG01-06：光墙触碰（电流纹波）

```
文件名：sfx_light_wall_touch.wav
时长：约 1.0 秒
类型：合成
优先级：P1
```

**声音描述**：50Hz 低频电流声——从稳定状态突然变为波动（手指触碰光墙激起波纹）→ 短暂增强（0.2 秒透明外景渗透时混入极微弱的鸟鸣+风声——来自"外面"的音频泄露）→ 恢复到稳定 50Hz。

**合成参数**：
```
0.0-0.2s：50Hz 正弦波，稳定 -18dBFS
0.2-0.3s：50Hz → 88Hz 快速上升滑音（波纹扩散的最强点），-12dBFS
0.3-0.5s：外景渗透——叠加极微弱的鸟鸣（单声）+ 风声（白噪 low-passed @ 2kHz），
          混合在 -30dBFS 以下，几乎听不见但"感觉到不一样"
0.5-1.0s：50Hz 恢复稳定 -18dBFS
```

**叙事功能**：与设计文档 7.3 节"边界光墙"的环境叙事完全对齐——0.2 秒透明外景透过光墙波纹可见，对应音频的"外景渗透"。

### SFX-FRAG01-07：光墙控制柱屏幕闪烁

```
文件名：sfx_light_wall_flicker.wav
时长：约 0.15 秒
类型：合成
优先级：P1
```

**声音描述**：极短的 CRT 屏幕刷新声——高压电流的瞬态脉冲（15kHz 以上，几乎听不见）+ 微弱的静电放电。只出现一次，低于 -20dBFS。

**合成参数**：

### SFX-VOX-01：系统语音 — "Welcome, Light Tracer"

```
文件名：sfx_vox_welcome.wav
时长：约 2.5 秒
类型：AI TTS（英语）
优先级：P1
```

**内容**：
```
"Light Tracer — identity confirmed. Welcome to Project: Light Trace."
```

**TTS 参数**：

```
引擎：任意高质量英语 TTS（推荐：ElevenLabs / Azure）
音色：中性女性声音（25-35 岁）
      - 不温暖、不甜美、不"助手音"（不是 Siri 2018+ 版）
      - 参考：Siri 早期冷感版本 / 机场自动广播的"无特征"播音员
      - 但不是机器人音——是一个真实的人在说"非人"的话
语速：1.0x 正常对话速度
语调：平坦——没有句尾上扬（不是问句）、没有情感强调
      唯一的重音落点："Welcome"中的"Wel"比其余词重约 10%
处理：导入后添加 -50dB 白噪底噪（暗示"来自电子设备"）

```

- BGM 在此处短暂降低 3dB（ducking），2.5 秒后恢复

**禁止**：
- 禁止温暖的语调（不能听起来像"欢迎回家"）
- 禁止句尾上扬（不能听起来像在问"你准备好了吗？"）
- 禁止任何情感标签

### SFX-VOX-02：边界光墙远声（granulated，嵌在环境声中）

```
文件名：sfx_vox_boundary_whisper.wav
时长：约 3.0 秒
类型：AI TTS 英语 → granulation 处理 → 极低混合
优先级：P1
```

**内容（经过 granulation 处理——只能听清单个音节/词语碎片）**：
```
"... seventeen batches ... three walked to the wall ... 
 ... the light was red ... the light was red ..."
```

**未经 granulation 的完整文本（供 TTS 引擎朗读用）**：
```
"Seventeen batches. Three walked to the wall. The last time you stood here, 
the light was red."
```

**处理流程**：
```
1. TTS 引擎朗诵完整文本（同 VOX-01 音色，语速 0.9x——比欢迎词略慢）
2. 将音频导入 Audacity → 使用 Granular Synthesis：
   - 颗粒大小：50-80ms 随机
   - 颗粒重叠：30%
   - 颗粒音高：原始 ± 4 半音随机偏移
   - 输出一段约 3 秒的"碎片化人声"
3. 混合至 -30dBFS——几乎听不见
4. 添加 50Hz 光墙电流声作为底层（同 SFX-FRAG01-06 的 50Hz 基频）——
   两种声音"粘在一起"——暗示"这个声音来自光墙本身"
```

**使用位置**：
- 玩家站在边界光墙前超过 30 秒时触发
- 在左声道或右声道随机出现（模拟"声音从墙里渗出来"）
- 和光墙电流声（SFX-FRAG01-06）叠加——玩家难以分辨是人声还是电流声的谐波

**叙事功能**：
- 对应设计文档 7.3 节光墙的"极低人声"叙事
- 对应林指导幕间台词——但这里说的是英语——暗示"每一个批次的溯光者都可以是不同的语言"（暗线C种子）
- 对应边界刻字"上一个你走到这里的时候，光墙是红色的"——这里说"the light was red"

### Godot 集成策略

| SFX 类型 | 播放方式 | 说明 |
|---------|---------|------|
| UI SFX | `AudioStreamPlayer`（全局，常驻于 UI Root） | 不受空间定位影响 |
| 碎片 SFX | `AudioStreamPlayer2D`（场景内，带位置衰减） | 如钟楼齿轮声仅在钟楼附近可闻 |
| 人声 VOX | `AudioStreamPlayer`（全局，Main bus） | 不受空间定位影响 |
| 环境循环 SFX | `AudioStreamPlayer` node with `loop = true` | 如喷泉水声 |

### 混音层级

```
Master Bus
├── BGM Bus (volume: -6 ~ -10 dB, pre-fader)
├── SFX Bus (volume: 0 dB, pre-fader)
│   ├── UI SFX:  -3 ~ -6 dB
│   ├── World SFX: -6 ~ -12 dB（含空间衰减）
│   └── VOX: -6 ~ -12 dB
└── Ambient Bus (volume: -6 dB)
    └── Environment loops: -12 ~ -18 dB
```

## 十、风格一致性检查清单

所有 Foley 实录 SFX 生成后逐条检查：

- [ ] **材质一致性**：所有纸/石/金属声的材质是否与 style_bible 的材质族对应？
  - 纸 → kraft paper / gray_card / book_page
  - 石 → 花岗岩（日晷底座）/ 石灰石（钟楼）
  - 金属 → 青铜（日晷指针）/ 铜（暗室）/ 银箔（星辉）
- [ ] **碰撞真实**：纸板接触声是否保留了自然的纤维尾音？（不要用 EQ 切除高频尾音——那是"纸的呼吸"）
- [ ] **干燥 vs 空间**：电子/合成音不加混响（干燥），Foley 实录音保留自然房间反射但不额外加人工混响
- [ ] **动态余量**：所有 SFX 峰值不超过 -3dBFS，主音量在 -6 ~ -18dBFS 之间——为 BGM 和环境音保留空间
- [ ] **禁止项**：无塑料/橡胶/玻璃质感（不属于纸工世界）、无纯合成科幻感（除了明确指定的电子音）、无电影预告片式 epic hit
- [ ] **人声禁止**：不温暖（不"欢迎"）、不甜美（不"助理"）、不机器人（不"合成"）——"真实的人在做非人的事"
- [ ] **频率分离**：50Hz 光墙声、18Hz 暗室地鸣、400-800Hz 系统蜂鸣、1200Hz 数据板——每个频率只归属于一个 SFX，避免多层叠加
