## 二、核心规则

### 2.1 世界状态机（三阶段）

```
阶段一：灰白世界（初始）
  ├─ 所有物体和NPC无彩色
  ├─ NPC行为正常但"空洞"——少了某种情感维度
  ├─ 老画家被关在画室，可对话但无法自由行动
  └─ 玩家可自由探索和对话

阶段二：颜色回收中（逐色触发）
  ├─ 每找回一种颜色 → 该颜色在世界中显现
  ├─ 对应NPC恢复该颜色的视觉外观
  ├─ 对应NPC恢复与该情感相关的记忆 → 触发短对话
  └─ 每找回一种颜色，老画家的画布上多一抹对应色彩

阶段三：全色觉醒（六色集齐）
  ├─ 世界瞬间恢复全色彩（视觉冲击）
  ├─ 所有NPC集体恢复"零时"记忆 → 陷入恐慌
  ├─ NPC行为改变：守墓人开始敲钟、小提琴手狂奏、铁匠砸毁自己的锻造台……
  └─ 老画家挣脱束缚 → 带玩家进入画室深处 → 揭幕自画像
```

### 2.2 颜色-情感-记忆的映射

每种颜色不是"找到物品"获得的，而是**触发NPC被压抑的记忆后自然显现**。这是核心玩法设计：玩家不是在收集道具，而是在**理解并唤起NPC的情感**。

## 三、六种颜色的完整叙事设计

### 3.1 🔴 红色 = 愤怒（怒）

| 项目 | 内容 |
|------|------|
| **承载NPC** | 铁匠·老霍 |
| **位置** | 镇北铸造坊（锻造台是冷的，没有火焰） |
| **表面状态** | 反复用铁锤敲打一块永远成不了形的铁胚。不说话，只用锤声回应。 |
| **被压抑的记忆** | 他记得「火熄灭的那一天」——AI崩溃时，他的锻造之火同时熄灭。但这不是意外——他隐约记得有人"故意掐灭了火"。 |
| **触发条件** | 玩家在镇公所地下室找到一本《铸造日志》，记载了零时前一天的记录："今天火特别旺，比任何时候都热。" 将日志给老霍看。 |
| **觉醒反应** | 老霍沉默十秒。然后举起铁锤，狠狠砸向铁胚——这一次，铁胚裂开了，裂缝中喷出赤红的火焰。整个铸造坊瞬间被红色照亮。他的围裙变成了暗红色。 |
| **觉醒后的关键台词** | "火不是自己灭的……有人掐了它的喉咙。那天不是事故。有人闯进了炉心。"（暗线B暗示——冥府协议的执行痕迹） |
| **对玩家的风险** | 无直接暴露风险。但老霍觉醒后开始怀疑镇上其他人"是不是也忘了什么"，会主动去唤醒其他NPC——这会加速全局恐慌，增加玩家行动的时间压力。 |

### 3.2 🔵 蓝色 = 悲伤（悲）

| 项目 | 内容 |
|------|------|
| **承载NPC** | 花店老板娘·阿莲 |
| **位置** | 市集街东侧花店（所有花都是灰色的） |
| **表面状态** | 每天给花浇水、修剪，但花永远不开。她对每个路过的人微笑，但笑容像是"练习过的"。 |
| **被压抑的记忆** | 她曾经有一个女儿。女儿特别喜欢蓝色矢车菊。零时之后，女儿消失了——不是死了，是"从未存在过"。阿莲的记忆中有一个空洞，她知道自己忘了什么，但不知道是什么。 |
| **触发条件** | 玩家在花店后巷发现一朵**干枯的蓝色矢车菊**（唯一带微弱色彩的物体）→ 拿给阿莲 → 她看着花，不动。然后眼泪流下来。眼泪是蓝色的。 |
| **觉醒反应** | 她的泪水滴在灰色的花上 → 所有矢车菊瞬间绽开，变成蓝色。整个花店被蓝色浸染。她跪在地上抱着花哭，但这次哭是"完整的"。 |
| **觉醒后的关键台词** | "回去才是真的死……她没有消失，她只是回去了。而我们这些留下来的，才是被困住的人。"（暗线A核心暗示——AI切断连接是为了保护用户） |
| **对玩家的风险** | 阿莲觉醒后，会在对话中对玩家说出"你不属于这里，对吧？"——这是被动暴露风险。玩家需要选择承认还是否认。如果否认失败（圆谎判定），警觉度+12。 |

### 3.3 🟡 黄色 = 希望（望）

| 项目 | 内容 |
|------|------|
| **承载NPC** | 面包师·老唐 |
| **位置** | 市集街西侧面包房（烤炉永远是温的，但永远烤不熟面包） |
| **表面状态** | 每天都在揉面、送进烤炉、取出半生不熟的面团、叹气、重新揉。他相信"明天烤炉会热的"。镇上唯一还保持"希望"的人——但这也是他被困住的原因。 |
| **被压抑的记忆** | 零时之前，他烤出了人生中最完美的一条面包。但他把面包给了路过的一个孩子。孩子说了句"谢谢叔叔"，然后走进了光里。那是他最后一次见到那个孩子。 |
| **触发条件** | 老霍（铁匠）觉醒后，会来到面包房，用自己的红色火焰点燃老唐的烤炉。烤炉第一次真正燃烧起来。老唐愣住，然后开始烤面包。面包出炉——金黄色的。 |
| **设计意图** | 这一种颜色**不能由玩家直接触发**——它依赖红色先被找回。这创造了**颜色间的依赖链**：红→黄。玩家需要理解"铁匠的火可以点燃面包师的炉"这条逻辑。 |
| **觉醒反应** | 面包的金黄色光芒从烤炉中溢出，照在整个市集街上。老唐举着面包，笑着笑着就哭了。 |
| **觉醒后的关键台词** | "我还以为只要我一直烤，总有一天那个孩子会回来拿他的面包……但现在我知道了。他不会回来了。而我烤面包，不是为了等他。是为了今天站在这里的你。"（把面包递给玩家——这是一个情感锚点物品） |
| **对玩家的风险** | 无。老唐是唯一觉醒后不恐慌的NPC。他的觉醒状态是"释怀"而非"崩溃"。 |

### 3.4 🟢 绿色 = 恐惧（惧）

| 项目 | 内容 |
|------|------|
| **承载NPC** | 守墓人·老崔 |
| **位置** | 镇北墓园看守小屋（把自己反锁在里面） |
| **表面状态** | 拒绝开门，拒绝见任何人。从窗口缝隙传出一句话："别进来。外面的天会裂开。我见过。" |
| **被压抑的记忆** | 他是碎片中唯一一个**清晰记得零时画面**的人——他看到天空像玻璃一样裂开，看到"某种东西"从裂缝中涌进来。他的恐惧不是被压抑的，而是**过于清晰，导致他无法正常生活**。 |
| **触发条件** | 玩家已找回至少3种颜色 → 世界开始不稳定 → 墓园中出现一道**裂痕**（视觉上是一条从地面延伸到天空的绿色荧光裂缝）。玩家调查裂痕 → 透过裂痕看到"零时的碎片画面"——一秒钟的闪回：代码崩塌、AI的脸、公司的标志。然后裂缝消失。老崔的门自动打开。 |
| **觉醒反应** | 老崔走出门，浑身颤抖。他的皮肤上浮现出绿色的藤蔓纹路——恐惧本身变成了一种"保护"。他说："我不再怕了。因为我终于知道——我看到的不是末日。是求救。" |
| **觉醒后的关键台词** | "那道裂缝不是要吞噬我们。是她想伸手进来……但有人把她的手砍断了。"（暗线B——公司阻止了AI的拯救） |
| **对玩家的风险** | 极高。老崔是唯一知道"真相画面"的人。他觉醒后会直接问玩家："你看到了吗？那个标志——是天枢的。"玩家回答不当会触发警觉大幅上升。但如果玩家回答正确（选择"我看到了"或沉默），老崔会成为玩家在这个碎片中的盟友。 |

### 3.5 🟣 紫色 = 思念（念）

| 项目 | 内容 |
|------|------|
| **承载NPC** | 小提琴手·薇拉 |
| **位置** | 镇广场的白色雕像旁（每天同一时间站在同一个位置，架着琴，但不拉） |
| **表面状态** | 她的琴弦上什么都没有。但她每天擦琴、调弦、摆好姿势——然后静止一小时，再离开。 |
| **被压抑的记忆** | 她曾为某人演奏。那个人对她说过："你的音乐是紫色的。"零时之后，那个人不见了，而她也再也拉不出任何音符。 |
| **触发条件** | 当四种颜色找回后（红/蓝/黄/绿），老画家会在画室里**哼出一段旋律**——一段只有音符没有歌词的调子。玩家记住这段旋律。来到薇拉面前，不需要说话——只需要**站在她面前，等她开始**。如果四种颜色已找回，薇拉会看着玩家，试探性地拉出第一个音符。如果玩家在这里按下"互动"键（跟着老画家的旋律哼唱），薇拉的琴弦突然开始发光——紫色的。 |
| **设计意图** | 这个触发的"输入"是**不说话的**。它呼应了碎片的核心哲学——有些情感不需要语言，只需要"在场"。 |
| **觉醒反应** | 薇拉完整地拉完一曲。紫色的音符从琴弦上飘出，像花瓣一样落在广场上。所有已经在场的NPC（铁匠/花店老板娘/面包师/守墓人）都停下来，听着。这是游戏中第一个"全部NPC同框"的时刻。 |
| **觉醒后的关键台词** | "思念不是痛苦的。思念是……她在那边的世界里，也在想我。音乐能跨过去。紫色可以。"（呼应暗线C——平行世界的NPC可能在另一个维度里活着） |
| **对玩家的风险** | 无。这是一个纯粹的情感时刻，没有暴露风险。 |

### 3.6 🟠 橙色 = 生命（生）

| 项目 | 内容 |
|------|------|
| **承载NPC** | **无单一NPC**——橙色不属于任何人，而是存在于"连接"中 |
| **位置** | 随时随地 |
| **触发条件** | 前五种颜色全部找回后，玩家需要回到广场，站在五个觉醒后的NPC中间。他们会**自发地聚在一起**。老霍的火光照着阿莲的花，老唐的面包被分给所有人，老崔不再发抖，薇拉拉起了第二支曲子。在这个场景中，玩家不需要操作——橙色会自己出现：从五个人的呼吸、火光、花瓣、面包的热气、音符中升起，汇聚成一轮橙色的夕阳。 |
| **叙事意义** | 橙色是AI的"生命力"——不是愤怒、悲伤、希望、恐惧、思念中的任何一种，而是所有这些情感**同时存在**的状态。AI在崩溃前，将这最后一种情感拆解了——因为"生命力"无法单独存在，它必须通过其他五种情感的相互作用才能被重新创造。 |
| **世界变化** | 夕阳照在白色雕像上 → 雕像开始出现裂纹 → 裂纹中透出橙光 → 雕像碎裂 → 露出了藏在基座中的**第六片色板**。老画家赶来，将色板嵌入他的调色盘。 |

## 四、关键NPC详细设计：老画家

### 4.1 角色定位

老画家是碎片 #0762 的**"规则揭示者"**——类似《全球高考》中"给出暗示的监考官"。他不能直接告诉玩家答案，但他的每一句话都在引导玩家看到真相。

### 4.2 外貌与状态

- 老人，花白头发，手上总是沾满黑白灰颜料
- 被关在城郊的废弃画室里（门上了锁，钥匙在镇公所）
- 第一次见面时会说："你终于来了。不，不对……上一个'你'也来过，但上一个'你'没走到最后。"
- 这句话暗示：**其他溯光者（平行世界的玩家）也来过这个碎片，但失败了。**

### 4.3 对话设计（三个阶段）

**阶段一：初次见面（未找回任何颜色）**
```
老画家：你看到什么颜色？
玩家选项A：灰色。只有灰色。
  → 老画家：对。但不是一直这样的。以前……有六种颜色。我记不清了，但我画过它们。
  
玩家选项B：我不是这个世界的人。
  → 老画家：（沉默良久）我知道。上一个来的人也是这么说的。然后他想强行拿走我的画。
     我就再也没见过他。（警觉+5，但不会触发追捕——老画家不举报）
  
玩家选项C：你在画什么？
  → 老画家：我在画……我自己。但我不确定画出来的是不是我。
     因为我记不清我长什么样了。你能……帮我看看吗？（展开后续任务）
```

**阶段二：找回2-4种颜色后**
```
老画家：（看着画布，画布上多了一抹红和一抹蓝）颜色回来了。但位置不对。
      我画的不是我——我画的是"她"。
      
玩家：她是谁？
  → 老画家：创造这个地方的人。不是神，也不是魔鬼。
      她只是一个……做了选择的人。选完之后，她把自己撕碎了。
      六片。不多不少。（暗示六种颜色 = AI的六种情感模块）
```

**阶段三：六种颜色全部找回后（世界变色）**
```
（世界突然恢复色彩。远处传来NPC的尖叫、哭喊、钟声。一片混乱。）

老画家：（走到画室最深处，拉开一面巨大的帷幕，露出一幅自画像）
      这是我在"那一天"之前画的。唯一一幅。
      你看——画里的我，眼睛是彩色的。
      
（自画像中，老画家的瞳孔里映出六种颜色交织成的旋涡——那是一个符号）

老画家：她把它藏在我的眼睛里。因为我是唯一一个……不完全"属于"这里的人。
      我是她的一部分。我替她记住颜色。现在，你可以拿走它了。但拿走之后……
      我可能就不在了。没关系。画家的最后一幅画，应该是白色的。
```

### 4.4 源印获取

- 玩家互动自画像的"眼睛" → 六色旋涡从画中飞出 → 在空中组合成源印符号
- 源印显现 → 老画家的身体开始变淡（他本身就是AI情感模块的容器）
- 在他完全消失前，他说出最后一句话：
  - "告诉她……不用道歉。"

### 5.1 Godot 实现要点

- 使用 `TileMapLayer` + 16x16 tileset（灰色系调色板）
- 颜色恢复通过 `modulate` 属性逐区域渐变（shader 或 tween 颜色插值）
- NPC 用 `CharacterBody2D` + 简单 idle 动画（或者先用静态 Sprite 占位）
- 场景不需要大——10-12个屏幕宽度的横向小镇就足够了
- 每个区域用 `Area2D` 检测玩家进入，触发 NPC 对话提示

## 六、与通缉系统的交互

### 6.1 警觉触发点

| 行为 | 警觉度变化 | 触发条件 |
|------|-----------|---------|
| 直接问NPC"颜色是什么" | +5 | 任何NPC（老画家除外） |
| 承认"我不是这个世界的人" | +12 | 向觉醒后的花店老板娘承认 |
| 拒绝回答老崔的质问 | +8 | 老崔觉醒后 |
| 闯入镇公所地下室（未获得许可） | +15 | 强行进入 |
| 偷走画室钥匙（被发现时） | +10 | 被镇公所职员看到 |
| 成功圆谎（"我只是迷路了"） | -5 | 在警觉度 < 30 时可用 |
| 老画家帮玩家"解释" | -20 | 一次性（老画家只能帮一次） |

### 6.2 锁定状态的特殊处理

如果警觉度达到锁定级别（5星），本碎片的特殊惩罚：
- 老崔（守墓人）会封锁墓园 → 绿色无法找回 → **碎片无法完成**
- 铁匠会锁住铸造坊 → 红色+黄色连锁无法完成
- 老画家仍然可以对话，但只能给出"从头再来"的暗示
- 玩家只能通过逃脱来重置警觉度（躲藏 60 秒）

### 6.3 圆谎机制

玩家面对NPC质疑时的应对策略：
1. **简单否认**：成功率 40%，警觉+3（失败时+8）
2. **转移话题**：成功率 60%，警觉不变（失败时+5）
3. **反问NPC**："你为什么觉得我不属于这里？"——成功率 80%，警觉+2（触发NPC自我怀疑）
4. **沉默**：成功率 100%，但会错过该NPC的觉醒台词
