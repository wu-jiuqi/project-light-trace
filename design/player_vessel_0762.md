# 碎片 #0762「颜色的葬礼」—— 玩家载体身份文档

> **用途**：定义玩家在本碎片中的伪装身份。此身份是玩家的"载体"——NPC通过这个身份认知你。你的目标是维持这个身份的合理性和连贯性，不被NPC发现你"不属于这个世界"。


---

## 一、载体身份（Vessel Identity）：「迷路的旅人」

### 1.1 身份概述

你的伪装身份是一个**长途旅行者**——声称自己在穿过灰白色荒野时迷了路，偶然发现了这个小镇。

你不是来"调查"的——你是来"休息、补给、然后继续上路"的。

### 1.2 身份的边界

这个身份的优劣：

| 优势 | 劣势 |
|------|------|
| 天然的外来者身份——解释了你为什么对镇上一切不熟悉 | 天然的外来者身份——意味着你必须尽快"离开"，停留过久会触发怀疑 |
| 不绑定任何本地关系——不需要假装认识任何人 | 没有任何人替你"背书"——你只能靠自己建立信任 |
| 对"不同的地方"的好奇心是自然的——旅人天然会对新地方有兴趣 | 对规则的"不知"和对世界的"不熟悉"必须一致——如果你对一个"不知道颜色"的世界表现得太适应，会穿帮 |
| "过路人"的视角天然就是观察性的——你可以问很多问题而不会被怀疑（只要不涉及禁忌） | 这是一个"短时效"的身份——你需要在信任建立之前尽快推进 |

### 1.3 身份故事

```
你不是"选择"了这个身份。
你是"醒过来"的时候，发现自己已经在这个身份里了。
```

你在一间灰暗的旅馆房间中醒来。你不记得自己是怎么来到这里的。窗外是灰色的天空。床头柜上有一把钥匙、一本磨损的日记。书桌上有一封写到一半的信。

**你不认识这个房间，但这个房间里到处都是"你"的痕迹。**

你的名字写在信末。你的字迹填满了日记。钥匙的挂坠上有一座钟楼的图样——你不记得这座钟楼，但你知道它很重要。

你是一个旅行者。你在穿越荒原时遭遇了一场奇怪的灰雾——日记上最后几页的字迹越来越抖。然后一切空白。然后你醒来——在这个灰白小镇的旅店里。

楼下，旅店老板冯婶会告诉你："你昨晚到的，脸色差得很，钱已经付了一晚。对了——你的朋友留了东西给你。在房间书桌上。"

**"朋友"。**

你不记得你在镇上有朋友。但你书桌上确实有一封信。信末署名不是你的名字——是另一个人的。一个你不认识的人。这封信在等你来到这里。

这就是你的载体身份：一个已经存在在这里、有自己的过去、有自己的目的的人。只是你不记得。而你必须一边找回记忆——一边不要让任何人发现你"忘了很多东西"。
```

**关键约束**：你**不能说**你来自"万象"之外。你不能说"我是溯光者"。这里没有这些概念。但更关键的是——**你也不能说"我不记得自己是谁"**。因为在这个世界中，遗忘是一个极度危险的信号——这里的NPC每天都在遗忘，他们最怕的就是发现自己也在忘。你的失忆会让他们怀疑：你是不是和他们一样？你是不是也是"不应该存在"的人？

## 二、核心生存规则：身份维持

### 2.1 "不能做的事"清单（优先级从高到低）

以下行为会直接导致身份暴露。按致命程度排列：

| 等级 | 行为 | 后果 |
|------|------|------|
| **致命**（警觉+25~50，直接锁定）| 说出"AI碎片"、"源印"、"天枢公司"、"万象虚拟宇宙"等任何来自外部世界的词汇 | 老崔：完全信任（你和他看到了同样的事）或完全敌意；其他人：你说出的是他们不该知道的概念——你不是疯的，你是危险的 |
| **高度危险**（警觉+12~20） | 直接承认"我是来收集线索的"、"我需要找到源印" | 大部分NPC听不懂你在说什么——但他们能感觉到你"不是正常人"。老画家可能会说："上一个说这种话的人——他没走到最后。" |
| **中度危险**（警觉+6~10） | 对NPC的行为表现出"分析性"的态度——像在观察实验对象（"你每天做同样的事，你不累吗？"） | 没有人会对邻居说这种话。你的语气暴露了你——你不是他们世界的人，你居高临下 |
| **轻度暴露**（警觉+2~5） | 对"没有颜色"这件事表现出任何形式的惊讶、好奇、或不解（"花为什么是灰色的？"） | 正常镇民不会注意到。你注意到了——你不正常 |

### 2.2 "能做的事"清单（安全互动）

| 行为 | 效果 |
|------|------|
| 以"旅人"视角发问——"这里一直这样吗？" | 安全——迷茫的旅人问迷茫的问题，合理 |
| 接受NPC的礼物（面包、花）| 这是建立信任的捷径——接受礼物 = 接受邀请 |
| 在NPC面前展示疲惫/人类基本需求——"有没有地方可以歇一晚？" | 降低警觉——"累"是人类共同的语言 |
| 对NPC的标志性物品（铁锤、小提琴、烤炉）表现出真诚的兴趣 | 安全——旅人对本地匠人的手艺好奇是正常的 |
| 帮助NPC做一些小事（帮老唐拿一袋面粉、帮阿莲移开挡路的石头） | 降低警觉——帮助行为建立好感 |
| 对NPC讲述一个关于自己旅行的小故事（编的也可以） | 安全——分享 = 建立共同经验 |

### 2.3 圆谎机制

当NPC质疑你时，你有以下几种回应策略（之前在关卡设计文档中已列出，此处细化具体对话模板）：

**策略1：转移话题（成功率60%，警觉不变）**
```
NPC：你看起来不像是这里的人。
你： 是吗？我走了很久的路——可能每个地方的人都不太一样吧。
     对了，这家店开了很久吗？（切换到NPC熟悉的领域）
```

**策略2：反问NPC（成功率80%，警觉+2）**
```
NPC：你……呼吸比我们快。
你： 是吗？你觉得——这个地方一直都是这样的吗？
     我不是在质疑。我只是——路过这里，确实觉得有点不一样。
     （让NPC反思自己的世界——"是你不对还是世界不对？"）
```

**策略3：沉默（成功率100%，警觉不变）**
```
NPC：你知道什么。对不对？
你： （沉默——看着NPC——然后缓缓点头）
     （沉默 = "我知道的不能说出来"——这是最安全但会错过信息的策略）
```

**策略4：半真半假（成功率70%，警觉-2）**
```
NPC：你到底是谁？
你： 我……不确定。我走了很久。有些事我不太记得了。
     但这不重要——我现在在这里。在这里就好。
     （"不记得" = 不暴露也不撒谎——你和NPC有共同的体验：遗忘）
```

### 2.4 身份退化曲线

你的身份不是"暴露"或"不暴露"的二元状态——它会逐渐退化：

```
阶段一：完整的旅人（警觉度 0-15）
    └─ NPC对你的态度：友善的好奇——"一个疲惫的过路人，无威胁"
    └─ 你可以：自由行动，问任何不涉及颜色/世界真相的问题

阶段二：可疑的旅人（警觉度 15-40）
    └─ NPC对你的态度：警觉——"这个人停留得太久了……他到底在找什么？"
    └─ 你可以：仍能对话，但NPC的回答更简短；某些问题会被拒绝回答
    └─ 你被允许的行为减少：深夜不能敲NPC的门；不能进入镇公所

阶段三：潜在的威胁（警觉度 40-70）
    └─ NPC对你的态度：戒备——"他不是这里的人。他在伪装。"
    └─ 你可以：对话仍然可以进行——但NPC会开始联合传递情报
    └─ 限制：无法触发任何颜色事件——NPC锁死自己的记忆
    └─ 老画家成为唯一还能进行完整对话的NPC

阶段四：暴露（警觉度 70-100）
    └─ 全城封锁
    └─ 铁匠锁锻造台 → 红+黄无法触发
    └─ 老崔封锁墓园 → 绿无法触发
    └─ 碎片无法完成
    └─ 只能逃——找到躲藏点维持60秒，警觉归零

阶段五：完全暴露（警觉度 > 100 = 锁定状态）
    └─ 等同于阶段四的终极版——但逃逸难度加大
    └─ 所有NPC拒绝沟通——包括老画家（但他不会举报你）
```

## 三、NPC之间的"共同记忆"机制

### 3.1 NPC会分享情报

当你的警觉度超过40时，NPC会开始**互相传递关于你的信息**。以下是信息传递方向：

```
老崔（最先察觉） → 老霍（沉默地听，不传播但记住了）
                  → 老唐（倾向于不信——但你在他面前说谎时，他会犹豫）
阿莲（不会主动传播，但被问到时不会保护你）
薇拉（从不传播——她不参与任何信息网络）
老画家（知道一切——但他永远不会在NPC间传递信息）
```

### 3.2 老画家是唯一的"安全岛"

老画家知道你是谁——他一开始就知道。但他永远不会举报你。原因：
- 他是AI的自我意识残片——他知道玩家是请来的
- 他说过"上一个你也来过"——他知道溯光者会再来
- **但他也不会帮你撒谎**——如果他对其他人说"他不是外来者"，那是说谎，而他不能对这个世界的人说谎
- 他能做的是：在你警觉度极高时给你最后一次机会——让你躲进他的画室里，"等他们忘记了再说"

## 四、进入碎片 — 旅馆房间开局

> 这是玩家进入碎片后的第一个场景。与旧版不同，玩家**不是从广场上开始**——而是从昏迷中醒来，身处一个陌生的旅馆房间里。
> 这段开局的叙事目的是：**让玩家通过主动探索来发现自己的载体身份**，而不是被告知"你是旅行者"。

---

### 4.1 场景：旅馆二楼，201号房

**地点**：灰檐旅店，二楼走廊尽头右转第一间。

**房间描述**（玩家醒来时所见）：

```
你睁开眼睛。
天花板是灰色的。墙壁是灰色的。窗帘被拉着，但灰白的光从缝隙中透进来。

你躺在一张铁架床上。床单粗糙，但干净。枕头上有一道折痕——不是你的。
有人睡过这张床。可能是昨晚。可能是你。

房间不大。
一张靠窗的木桌。一把木头椅子。一个旧衣柜——衣柜没有门，里面只有两件灰白色的衬衫，叠得很整齐。
地上有一个旅行皮箱——你的是你的，因为你低头看了自己的手，手上有和皮箱手柄上一样的磨损痕迹。
```

**玩家初始状态**：
- 躺在床上（摄像机从俯视角度缓缓下降）
- 可以起身（WASD/方向键移动生效）
- 处于无敌/无警觉状态——这是"安全引导期"
- 屏幕中央出现提示："调查房间，找到关于你身份的物品"（2秒后自动消失）

---

### 4.2 房间中的五件身份线索物品

| # | 物品 | 位置 | 外观 | 交互后获取的信息 |
|---|------|------|------|----------------|
| 1 | **旅店钥匙** | 床头柜上 | 一把铜色钥匙，挂着一个木牌，木牌上刻着"201" | "这是你房间的钥匙。木牌背面有旅店的名称——字迹已经磨损，隐隐约约能看出'灰檐'两个字。" |
| 2 | **一本磨损的旅行日记** | 床头柜抽屉里 | 棕灰色封皮，页角卷边，用皮绳捆着 | **日记内容**（见下方 4.3）。最后一页的日期是空白的——没有日期。再前面的几页字迹越来越抖，越来越短。最后一篇可读的日记写的是："灰雾……我在雾里走了很久。然后看到光。不是真的光——只是灰得没那么深的地方。是灯。小镇的灯。我应该已经不在原来的路上了。这不对。但人必须往前走。明天醒来再想。……如果醒得来。" |
| 3 | **一封未寄出的信** | 靠窗木桌上 | 信封已拆，信笺摊开，旁边有一支铅笔 | **信的内容**（见下方 4.4）。信末署名：**"陈"**——只有一个字。这是寄信人，不是你。寄信人托你带这封信到这里。你是信使。 |
| 4 | **一张模糊的照片** | 夹在日记倒数第三页 | 褪色的相纸，边缘发黄，画面模糊 | 照片上是一个人的轮廓——看不清脸——站在一座钟楼前。钟楼的外墙是石头砌的，尖顶上有风向标。照片背面有一行铅笔字：**"钟楼北——2077.2.14"**。那个日期在一个月前。钟楼不在这个镇上。你不记得这个人是谁。 |
| 5 | **一个随身皮箱** | 床脚地板上 | 旧皮革，搭扣已松开，里面有几件衣物和一张地图 | 地图是一张手绘的路线图。上面标着一条东西走向的路径，途经几座城镇。最东端的标注是**"灰白镇"**——这是你在地图上圈出的终点。地图背面写着：**"如果我中途失联——不要找我。继续走。把信送到。"** 笔迹和日记一致——是你写给自己的。 |

### 4.3 旅行日记的关键内容（玩家可选阅读）

日记不必全部展示——但以下条目会在玩家翻开日记时自动在 UI 上显示：

```
【倒数第四篇】
"路越来越难走了。石头是灰的，土是灰的。天空的颜色——我记不清了。
但无关紧要。往前走。"

【倒数第三篇】
"今天遇到一个人。他说他在找一个地方——跟他描述的一样。跟我地图上标注的也一样。
他说那地方叫'灰白镇'。我问他在那里要见谁——他说'一个画画的人'。
他说这话的时候，手在抖。
我问他叫什么——他没说。他说'到了就知道了'。
然后他走了。往西——我的反方向。
他比我快。他应该比我先到。"

【倒数第二篇】
"起雾了。灰白色的雾——但不是普通的雾。
这雾不对——它太安静了。我在雾里走了两个小时，一个活物都没遇到。
开始记不清路标了。没关系——记住方向就行。
东。一直往东。"

【最后一篇（可读）】
"灰雾越来越浓。我听到声音——不是人的声音，是……机器的声音。很遥远，但很清楚。
像什么东西碎了。
然后安静了。什么声音都没了。雾也散了——不对，是我从雾里出来了。
前面有光。一个镇子。灰色的镇子。
这就是灰白镇——但和地图上的描述不一样。地图上是手绘的。这里是……说不清楚。
我觉得不对。但天黑了。而且——我不确定还能不能往回走。
先住下。明天再看。
——如果我明天还醒得过来。"
```

---

### 4.4 信的内容

信封正面的收件人名字已被抹去——有水滴过的痕迹。但背面仍有寄件人的署名：**"陈"**。

信的内容：

```
我不知道你现在叫什么。可能你不会再用原来的名字了。
这没关系。你在不在那里——我都当你在。
如果这封信能到你手里——说明路上的人送到了。替我谢谢那个送信的人。

我来找你是因为她走之前说了一句话。
她说——"天不是灰色的。只是你们忘了怎么看。"

我不懂。我天天看着天空。它就是灰的。一直是。从来都是。
但她说那句话的时候，语气不是"错觉"——是"事实"。
就像她知道什么。或者她看到过。

如果你去那个镇上——去找一个画画的人。她说他在那里。
我没见过他。但她说了他的名字。我没有写在信里——怕被人看到，怕这封信到不了你手上。

保重。
陈
```

### 4.5 从房间到旅馆大堂：与旅店老板的初始对话

**触发条件**：玩家检查了至少 2 件物品后，走向房间门口。

**下楼时**：楼梯咯吱作响。楼下传来翻账本的声音。

**旅店老板·冯婶**（非关键NPC，60多岁老妇人）：

```
冯婶：（抬头看你一眼，又低头看账本）
      "哦——醒了啊。你昨晚到的时候脸白得像纸——我差点以为你要死在这了。"
      "没事吧现在？饿不饿？街对面的面包房——老唐，做得最好的面包。
       不过他的面包——（她顿了顿，好像在想该怎么说）——不热。
       就是那个温度。你懂的。"
      
玩家选项A："我睡了多久？"
  → 冯婶："一晚。钱已经付了——一晚。今晚要续的话再说。反正空房多得是。"
       "对了——昨晚你朋友来过了。放下东西就走了。说你知道是什么。"
       （她指了指楼上。"信。在你书桌上。"）
  
玩家选项B："我朋友？谁？"
  → 冯婶："（抬起头，第一次认真看你）——不认识。个子不高，走得很快。
      像是怕什么东西追着他似的。放了信封，说了句'他在二〇一'——就走了。
      我还以为你们约好的。"
       （她重新低下头。）"反正我不管。他自己说不是坏人。是不是坏人——你看信就知道了。"
  
玩家选项C：（沉默——什么都不说）
  → 冯婶：（看了你三秒）"你跟你朋友一样——不爱说话。"
       "行。不问。总之东西在楼上。自己看。"
```

**设计要点**：
- 冯婶不是一个"信息源"——她不提供答案。她提供**线索的确认**（"她说了他的名字——我没有写在信里"）
- 冯婶对"面包不热"的欲言又止是玩家第一次接触这个世界"不对"的暗示
- "你朋友来过了"是一个**悬疑钩子**——"我明明不记得有朋友，为什么有人留信给我？" 这会引导玩家回去翻找房间剩余线索

### 4.6 旅馆大堂到广场——玩家见到世界的第一眼

**触发条件**：玩家吃完冯婶聊完，走出旅馆大门。

**玩家所见**：

```
你推开灰檐旅店的门。
市集街在眼前展开——一条灰白色的石板路，向北延伸。
左侧：一家花店，花店门口摆着几排花盆，花都是灰色的。
右侧：一家面包房，门口站着一个人——他在朝你招手。

"哦——新来的！" 他喊。他手里拿着一个灰白色的面团。
"来！尝尝——不热，但是能吃！别嫌弃！"

（这是老唐。但你现在还不知道。）

更远处——你看到街道尽头是一个广场。
广场中心有一座白色的雕像——纯白的。
在这个全是灰色的世界里——它白得刺眼。
```

**此阶段的设计意图**：
1. 玩家已经掌握了载体身份的核心信息（旅行者、信使、可能失忆的过路人）
2. 玩家已经接触了第一个"世界不对劲"的暗示（老唐的面包不热、冯婶的欲言又止）
3. 玩家有一个未解决的个人谜题（"陈"是谁、钟楼在哪儿、送信的人去哪了）
4. 玩家现在可以**自主决定方向**——走向面包房（老唐）、花店（阿莲）、广场（薇拉+雕像）、继续沿街走到铁匠铺

**没有"正确"的路线**——但有三条自然引导路径：

| 路径 | 触发 | 第一印象 |
|------|------|---------|
| 老唐的面包房（向北几步） | 老唐主动喊你 | 热情、温暖、不合逻辑的希望——你会收到一个面团 |
| 花店（向北再走几步） | 花店门开着，阿莲在浇花 | 温柔、空洞、礼貌到让人不安——你会收到一束灰花 |
| 广场（街道尽头） | 白色雕像的视觉锚点 | 沉默、庄严、神秘——你会看到薇拉和雕像基座上的模糊字迹 |

## 五、身份暴露的"最后一关"——老画家的审判

如果你在警觉度极高（>70）的情况下终于找到了老画家，他会给你一次考验：

```
老画家：你是来"拿东西"的——对吗？
       上一个来的人也是。
       但他拿了就想走——他不想理解这些颜色是什么意思。
       你呢？

选项A：点头。承认。
  → 老画家：那你比上一个好。至少你诚实。
     但诚实不是全部。你还得证明你知道这些颜色是什么。
     （他不说"源印"——他说的是"颜色的意义"）

选项B：否认。
  → 老画家：（摇头）你连自己是谁都不敢说。
     那你走吧。画室的门还可以再锁一百年。
     （老画家不再跟你对话——源印无法提取。碎片锁死。）

选项C：沉默——然后指向他画室深处被灰布盖住的画。
  → 老画家：（长时间的凝视）————————你知道了。
     不是"打听来的"，是"真的知道了"。
     （信任建立。揭幕自画像，源印触发。）
```

## 六、载体身份的叙事意义

你在这个碎片中的躯壳不是"随便选的"——它本身就在讲述这个碎片的核心命题：

"迷路的旅人"对应的是**所有碎片陷入其中的、找不到自己位置的NPC**。你问"怎么回去"——而他们从未想过这个问题。你的每一句"这里一直这样吗？"都在为他们挖开记忆的土壤。

你不是在骗他们。你是在**成为他们找回情感的契机**——一个真实的、会累的、也会离开的人，激发了那些"不应该再会有反应"的灵魂。

这就是你的载体身份和这个碎片的内核之间最底层的咬合——**星星之火，不是烧，是点亮**。
