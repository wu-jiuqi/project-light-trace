# 溯光计划 — 界面流转图 (Screen Flow Diagram)

> **文档定位**：定义游戏中所有UI界面及其之间的转换关系，覆盖当前实现与规划中的全部状态
> **版本**：V1.0
> **创建日期**：2026-06-09
> **关联文档**：`player_journey_map.md`、`ftue_design.md`

---

## 一、总体架构：三层界面模型

```
┌─────────────────────────────────────────────────────────────────┐
│                     L0: 全局覆盖层 (Overlay)                      │
│  暂停菜单 | 存档面板 | 背包UI | 设置面板 | 通知弹窗               │
├─────────────────────────────────────────────────────────────────┤
│                     L1: 游戏主界面 (Game)                         │
│  标题画面 | 星图界面 | 碎片世界 | 论坛界面                         │
├─────────────────────────────────────────────────────────────────┤
│                     L2: 交互子界面 (Sub-UI)                       │
│  对话面板 | 详情卡片 | 物品信息 | 胜利画面 | 加载画面              │
└─────────────────────────────────────────────────────────────────┘
```

## 三、覆盖层详细流转

### 3.1 暂停菜单 (PauseMenu) — ESC触发

```
任意游戏状态(碎片世界/星图) + ESC
  │
  ▼
┌─────────────────────────┐
│     暂停菜单             │
│                         │
│   [返回星图]             │  → SceneManager → StarMap
│   [保存游戏]             │  → 存档选择面板(保存模式)
│   [返回标题]             │  → 确认弹窗 → TitleScreen
│   [继续游戏]             │  → 关闭暂停菜单
│                         │
│   当前进度: XX%          │
│   游戏时间: XX:XX        │
└─────────────────────────┘
```

### 3.4 通知系统 (Notification) — 事件触发

```
各类事件 → 通知弹窗
  │
  ├── 颜色觉醒: 滑入彩色条幅 "红色·愤怒 已觉醒"
  │   └── 3秒后自动消失
  │
  ├── 警觉升级: 滑入警示条幅 "老霍 提高了警惕"
  │   └── 轻微震屏 + 3秒后消失
  │
  ├── 物品获得: 底部提示 "获得: 铸造日志"
  │   └── 2秒后消失
  │
  └── 存档成功: 右上角 "💾 已保存"
      └── 1.5秒后消失
```

## 八、关键界面详细参数

### 8.1 标题画面

| 参数 | 值 |
|------|---|
| 场景路径 | `res://scenes/ui/title_screen.tscn` |
| 选项 | 继续(自动加载最新槽) / 新游戏 / 加载 / 退出 |
| 特殊逻辑 | 无存档时"继续"灰显 |
| 视觉风格 | 纸艺风格标题 + 暗色背景 |

### 8.2 星图界面

| 参数 | 值 |
|------|---|
| 场景路径 | `res://scenes/star_map.tscn` |
| 组件 | ShardCanvas(碎片画布) + DetailCard(详情卡片) |
| 详情卡片尺寸 | 470×628 px |
| 详情卡片动画 | 从右侧滑入(0.24s, QUAD EASE_OUT) |
| 详情卡片内容 | 碎片ID/名称/世界/难度/类型/状态/线索/源印 |
| 详情卡片按钮 | [进入](左) [关闭](右)，透明覆盖 |
| 输入支持 | 键盘: ESC关闭, ←→选择按钮, Enter确认 |
| 修复动画 | 碎片从散落位置移动到四芒星归位 |

### 8.3 碎片世界（以#0762为例）

| 参数 | 值 |
|------|---|
| 场景路径 | `res://scenes/fragments/fragment_0762.tscn` |
| 地图尺寸 | 6个房间，每个832×512 px |
| Camera | zoom=2, 平滑跟随玩家 |
| 房间切换 | SceneFader黑屏过渡, spawn_point指定出生位置 |
| NPC | 7个 (老霍/阿莲/老唐/老崔/薇拉/老顾/冯婶) |
| 可拾取物品 | 铸造日志(镇公所), 矢车菊(墓园) |
| 源印触发 | 6色齐聚→广场白色发光体→E键解码 |

### 8.4 对话面板

| 参数 | 值 |
|------|---|
| 面板尺寸 | 800×432 px (下半屏) |
| 透明度 | 96% 不透明度 |
| 气泡样式 | 玩家右蓝 / NPC左灰 |
| 输入方式 | 文本输入 + [发送]按钮 |
| 特殊按钮 | [给予]物品 / [历史]翻看 |
| 输出方式 | 流式逐字显示(DeepSeek SSE) |
| 关闭 | ESC键 |

### 8.5 背包UI

| 参数 | 值 |
|------|---|
| 触发键 | B |
| 网格布局 | 4列×2行 |
| 物品显示 | Icon + 名称 + 颜色边框 |
| 刷新机制 | 自动刷新 (物品变化时) |
| 关闭 | B键 / ESC |

## 九、界面间的数据传递

```
┌────────────────────────────────────────────────────────────┐
│                     数据流转图                              │
│                                                            │
│  TitleScreen                                               │
│    │  new_game() → GameManager.new_game()                  │
│    │  load_game(slot) → SaveManager.load(slot)             │
│    ▼                                                       │
│  StarMap                                                   │
│    │  enter_fragment() → FragmentManager.enter_fragment()  │
│    │    → GameManager.reset_fragment()                     │
│    ▼                                                       │
│  FragmentWorld                                             │
│    │  _ready() → 从 GameManager 读取:                      │
│    │    - awakened_colors[] (觉醒状态)                      │
│    │    - npc_state_cache{} (NPC位置/警觉/信任)            │
│    │    - items_used{} (已消耗物品)                         │
│    │    - fragment_completed (是否已完成)                   │
│    │                                                       │
```

## 十一、界面流转规则

### 11.1 动画过渡规范

| 过渡类型 | 动画方式 | 持续时间 | 缓动函数 |
|---------|---------|---------|---------|
| 场景切换 (SceneFader) | 黑屏淡入淡出 | 0.5s | LINEAR |
| 详情卡片 (StarMap) | 右侧滑入/滑出 | 0.24s | QUAD EASE_OUT/IN |
| 对话面板弹出 | 底部滑入 | 0.2s | QUAD EASE_OUT |
| 通知弹窗 | 顶部/底部滑入+自动消失 | 0.3s滑入/3s停留 | QUAD EASE_OUT |
| 背包UI | 即时显示(无动画) | - | - |
| 暂停菜单 | 即时显示+轻微缩放 | 0.15s | CUBIC EASE_OUT |

### 11.2 输入优先级

```
最高优先: 系统级 (ESC暂停/Alt+F4)
  ↓
高优先: 覆盖层 (背包B/暂停菜单内操作)
  ↓
中优先: 子界面 (对话面板内操作/详情卡片内操作)
  ↓
低优先: 游戏主界面 (WASD移动/E交互/点击碎片)
```

### 11.3 模态规则

| 打开界面时 | 对游戏世界的影响 |
|-----------|---------------|
| 暂停菜单 | 完全暂停 (process_mode = paused) |
| 对话面板 | 玩家移动锁定 (velocity冻结) |
| 背包UI | 玩家仍可被NPC"看到"但不移动 |
| 存档面板 | 完全暂停 |
| 胜利画面 | 世界停止，等待返回星图 |
| 通知弹窗 | 无影响(非模态) |
