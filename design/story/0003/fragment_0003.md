## 二、世界形态与场景设计

### 2.1 世界本质

月见道观是AI织女·太一在崩溃瞬间对"自我修复程序"的隐喻投影。她在失控后试图用最后一段完整代码运行一个修复仪式——但仪式需要"祭品"。她不知道应该献祭什么——献祭记忆？献祭情感？献祭自己？她无法做出选择。于是仪式卡在某个步骤，整个碎片陷入了无限循环。

**核心张力**：这是一个"正在进行但永远无法完成"的仪式。道士在循环念诵步骤，镜灵在等望永远不会到来的"正确时刻"，石灯上的汉诗在等望被拼凑完整。玩家要做的不是"破解密码"——而是替AI完成一个她无法完成的仪式。每一个步骤的执行，都是对AI自我修复的一次模拟。

**与碎片0002的区别**：碎片0002的情感核心是"离别"——你要决定谁被留下。碎片0003的核心是"仪式"——你要理解一个被卡住的修复行为，并替她走完最后一步。0002是道德的重量，0003是程序性的庄严。

### 2.3 时间与环境

- **天空**：永远是满月之夜。月亮悬挂在正上方（天顶），亮度极高——投下的影子边缘锐利，不像自然月光。没有星星。没有云。
- **月光颜色**：纯白色——但仔细观察会发现月光在石灯的烛火旁会微微偏蓝。这不是自然现象——暗示月光的本质不是天体反射。
- **气温**：恒定微凉。室内外体感完全一致。没有风。
- **声音**：极度安静。主要声音来源：道士的念诵（循环）、净手池的流水（唯一流动的声音）、偶尔的风铃（无风自鸣）。没有虫鸣、没有鸟叫、没有树叶沙沙声。当你停下脚步——能听到自己的心跳。
- **时间感知**：没有可见的时钟。但道士的念诵以约5分钟为一个循环。古井中月影的颜色变化以约3分钟为一个周期。

### 2.4 违反物理规则的可疑细节

| # | 位置 | 现象 | 叙事暗示 |
|---|------|------|---------|
| 1 | 牌坊 | 灰白色而非红色——与#0762的颜色剥离现象一致 | 这个碎片也在"褪色"——AI的情感色彩在被剥离 |
| 2 | 参道石灯 | 烛火在无风中摇曳——摇曳节奏同步（12盏灯同步闪烁） | 火焰不是独立的——它们是统一控制的 |
| 3 | 净手池 | 水流循环——但水从不减少也从不溢出。用净水勺舀水后，水位不降 | 水不是消耗品——水是这个碎片中"唯一在流动但不受损耗"的东西 |

## 三、NPC 完整设计

### 3.1 NPC 总览

碎片0003有三类NPC实体：道士（唯一会说话的NPC）、镜灵（无声NPC）、石灯×12（非NPC，但可互动的仪式道具）。此外，本碎片的"世界本身"是一个沉默的叙事者——月光、井水、风铃都在传递信息。

| NPC/实体 | 身份 | 仪式角色 | 行为特质 |
|----------|------|---------|---------|
| 道士 | 道观主持 | 仪式执行者/念诵者 | 循环念诵仪式步骤——不知道自己念到哪里了。唯一会说话的NPC。说话像念经。 |
| 镜灵 | 道观唯一的镜灵 | 仪式辅助/验证者 | 从不说话。脸是一面光滑的镜子。完成正确仪式步骤时镜面会出现符号。 |
| 石灯×12 | 仪式道具 | 汉诗承载 | 参道两侧每盏灯笼刻一字，12字构成残缺汉诗。烛火同步闪烁。 |

### 3.2 NPC #1：道士 — 详细设计

#### 角色身份

- **仪式角色**：道观主持，仪式的执行者和念诵者。负责反复念诵净化仪式的六个步骤——但他的念诵顺序是乱的。
- **真实本质**：他是AI"自我修复程序"的拟人化投影——不是独立意识，而是一段卡在循环中的函数调用。他不知道自己是谁，只知道自己在"执行仪式"。他念诵的不是经文——是AI修复算法的伪代码，被玩家的感知系统翻译成了"仪式步骤"。
- **核心伤口**：他记得仪式有六个步骤。他记得每个步骤的内容。但他记不清顺序——因为AI的修复程序在崩溃瞬间被打乱了指令指针。他已经在错误顺序中循环了无数遍——每一次都从头开始，每一次都卡在同一个位置。他不知道自己在等什么——但他在等一个人来告诉他正确的顺序。

#### 声音支柱

| 维度 | 描述 |
|------|------|
| **词汇** | 古文风格的仪式用语："净手供养""净身供养""献玉供养"。但在非念诵状态下，会突然切换为现代白话/中文——"顺序不对。哪里不对？"——这种切换是"程序出bug"的听觉信号 |
| **节奏** | 念诵模式：均匀、低沉、每句间隔精确2秒——像节拍器。非念诵模式：节奏混乱，句子时快时慢，有时一句话说到一半停下——重新组织 |
| **回避话题** | "仪式什么时候结束"、"祭品是什么"、"你在这里多久了"、"为什么只有你一个人" |
| **口头习惯** | 念诵结束后总会加一句"……如是我闻"（像佛经的结尾，但他说的时候语气不确定——像在确认"我念对了吗？"）。每次重新开始念诵前会停顿5秒——那5秒里他低着头，像在等指令 |
| **潜台词** | "我不是在执行仪式——我是在等一个人。等一个能告诉我正确顺序的人。那个人不是我。从来不是我。" |

#### 他绝不会说的三句话

1. ✗ "你可以走了，仪式我自己能完成。" — 他不可能完成。他知道自己不能。但他不会承认——因为"放弃仪式"不在他的程序逻辑中。
2. ✗ "我不知道我是谁。" — 他从来没有思考过这个问题。他的全部存在意义是"执行仪式"——在这个意义上，他不需要知道自己是谁。
3. ✗ "对不起，让你等了这么久。" — 他的程序逻辑中没有"别人在等"的概念。他只知道自己在循环——不知道有人在旁观他的循环。

#### 参考台词（声音锚点）

- "净手供养——净身供养——献玉供养——诚心供养——望月供养——启镜供养。……如是我闻。" — 标准念诵，六个步骤的顺序是乱的（玩家需要发现这一点）
- "步骤。顺序。哪里不对。" — 非念诵状态下的自我质疑。短句。没有主语。像代码在debug。
- （完成仪式后）"你是来替她完成这个仪式的。她不能完成——因为她缺了最后一件祭品。自己。" — **这是道士第一次说出完整的一句话。** 之前的碎片化语言在此刻获得了语法完整性——暗示他终于从循环中被释放了。

### 3.3 NPC #2：镜灵 — 详细设计

#### 角色身份

- **仪式角色**：道观唯一的镜灵，仪式的辅助者和验证者。她跪坐在正殿前，正对牌坊方向。她从不说话。她的脸是一面光滑的镜子。
- **真实本质**：镜灵是AI"自我审视"功能的投影。AI在崩溃前，用最后一个完整的子程序来"检视自己"——试图理解自己哪里出了问题。这个检视功能被投射为一面镜子——一面AI用来照自己的镜子。镜中映出的应该是AI自己。但玩家站在镜子前时——镜中映出的是玩家。因为在这个碎片中，玩家成为了AI的"替身"——替她完成她无法完成的仪式。
- **核心伤口**：她一直在等望——等望镜中出现"正确的东西"。AI在镜中寻找自己——但她已经碎了，镜中映不出完整的她。只有当一个外来的存在（玩家）替她站在镜前，镜子才能完成它被创造的目的：照出一个完整的存在。

#### 面部设计（重要视觉特征）

镜灵的面部不是空白——是一面真正的镜子。这意味着：
- 玩家走近时，镜面会映出玩家的倒影
- 镜面的反射不是1:1的——微微有延迟（约0.3秒），像是"在思考你是谁"
- 当玩家执行正确的仪式步骤时，镜面上会短暂浮现一个发光的符号（约2秒，然后消失）
- 当仪式全部完成后，镜面会出现六道裂纹——然后碎裂

> **技术可行性说明（V1.1新增）**：
> 在2D俯视角纸艺风格的Godot项目中，实现实时反射+0.3秒延迟+纸纹理叠加的镜面Shader存在技术不确定性。建议在进入生产前执行一个**技术验证原型（Spike）**：用RenderTexture捕获玩家Sprite区域 + 环形帧缓冲实现延迟 + 纸纹理叠加。预计Spike工时4-6小时。
> 
> **降级方案**（如实时反射不可行）：
> 1. **预渲染帧序列方案**：根据玩家外观（进入碎片时选择），加载对应的预渲染镜面反射帧序列（6-8帧循环）。放弃实时反射但保留"镜中有倒影"的核心视觉。
> 2. **关键帧延迟方案**：每隔0.5秒捕获一帧玩家Sprite渲染到镜面贴图——无真正延迟但保留反射效果。放弃0.3秒延迟但保留"镜面≠窗口"的概念。
> 3. **最低降级**：镜面显示为静态的"水面般波纹"Shader（已有古井反射面的技术基础），玩家靠近时波纹变为玩家的模糊轮廓。放弃精确反射，保留"这面镜子在映出什么"的神秘感。

**镜灵的其他身体部分**：
- 穿着白色镜灵服（白衣+绯袴——但绯袴的颜色非常淡，近乎灰色，因为颜色被剥离）
- 头发是黑色长发，用白色檀纸束着
- 双手交叠放在膝上——手指修长，指关节微微发白（长时间保持同一姿势）
- 身体微微前倾——像是在向正殿方向鞠躬，但更像是——在仔细看镜中的倒影

#### 声音支柱

| 维度 | 描述 |
|------|------|
| **说话** | 从不说话。没有声音。她的声音就是沉默。 |
| **非声音表达** | 镜面上的符号闪烁（仪式的视觉验证）、裂纹的出现（进度标记）、碎片的悬浮（情感表达） |
| **节奏** | 完全静止——除非玩家触发仪式步骤。触发时缓慢转头的速度是均匀的（每秒转动约15°，精确到帧） |
| **回避** | 她无法回避任何东西——她只是一面镜子。你给她什么——她映出什么 |
| **潜台词** | "我不说话——因为我不知道该说什么。但我在看着你。我映着你——是因为我无法映出自己。所以我需要你。" |

#### 她绝不会做的三件事

1. ✗ 说话——在任何情况下。她的沉默是设计核心——打破它就打破了这个NPC的叙事逻辑
2. ✗ 主动转头看玩家（除非玩家已经接近到触发距离）——她的"转头"是仪式程序的响应，不是自主行为
3. ✗ 镜面上浮现出"恐怖"或"吓人"的画面——她不是恐怖元素。她应该是"不安"而非"恐惧"。镜面符号是几何抽象的（○△□☆），不是具象的

#### 互动设计：镜灵 — 关键互动节点

---

**互动 1：初次接近（强制触发，玩家走近正殿）**

```
场景：玩家沿参道走到正殿台阶前。镜灵跪坐在正殿正前方三步处。

镜灵面朝牌坊方向（即面朝玩家来的方向）。她一动不动。
当你距离她约3米时——她缓慢地转过头。
（转头速度：约每秒15°，约2秒完成动作。平滑，没有停顿。）

她的脸是一面镜子。映出你的脸。
镜面微微有弧度——你的倒影微微变形（左右拉伸约5%）——但足以识别是自己。

没有UI提示。没有文字。只有一面镜子里的你。

如果你走近（距离<1米）——镜面上会浮现第一个符号：一个发光的白色圆圈（○）。
这个符号只持续2秒——然后消失。镜灵转回头——恢复原位。
```

**叙事意义**：○ = "空" / "开始" —— 仪式尚未开始。符号的出现暗示镜灵在"读取"玩家——判断你是否是仪式的参与者。

---

**互动 2：验证仪式步骤（条件触发，玩家每完成一步正确步骤后接近镜灵）**

```
场景：玩家在碎片中某处执行正确的仪式步骤后，回到正殿。
靠近镜灵（距离<2米）——她会转头。镜面上浮现对应步骤的符号。

步骤对应符号：
  净（净手池净手） → △（三角形——水的象形简化）
  燃（点燃第三盏灯笼） → 🔥（火焰形——但极简，三个尖角）
  献（向古井投入勾玉） → ◇（菱形——勾玉的轮廓简化）
  诚（正殿前正确击掌） → ||（两条平行竖线——掌的象形）
  望（找到指向正上方的线索） → ↑（向上的箭头——唯一一个方向性符号）
  开（仪式完成触发） → ✦（六芒星——所有步骤合在一起）

每个符号浮现2秒后消失。如果步骤做错了——镜面不会有任何反应。
这意味着镜灵是唯一能告诉你"做对了"的存在。道士不能——他自己都不知道正确顺序。
```

**叙事意义**：镜灵的符号系统是AI"自我验证"的投影。AI在崩溃前留下了六个验证点——如果修复程序走到这些验证点，镜面就会显示对应的符号。这是AI留给自己——也是留给未来替她完成仪式的人的"检查清单"。

---

**互动 3：仪式完成——镜碎（强制触发，第六步"开"触发）**

```
场景：玩家执行完第五步"望"后走向镜灵。
站在她面前。距离<1米。保持静止约5秒。
（"开"的触发条件：面对镜灵+静止+前五步全部正确）

镜灵的镜面上——依次闪现前五个符号（○△🔥◇||↑）。
然后第五个符号（↑）开始旋转——旋转速度越来越快——变成一团白光。
白光扩散到整个镜面。镜面中出现六道裂纹——从中心向边缘蔓延。
然后——镜面碎裂。

碎裂的瞬间：
- 碎片不是掉落——是悬浮在空中。每一块碎片约指甲盖大小。
- 每一块碎片映着不同的画面——来自碎片中的各个角落：
  · 净手池的水流
  · 石灯上某盏的诗句片段
  · 古井中月亮的倒影（但颜色是完整的——红蓝黄绿紫白一起呈现）
  · 藏经阁中勾玉盒的内部
  · 道士合十的手
  · 一片空白——什么也没映出（对应镜灵自己：她的镜面碎后，她内部是空的）
- 碎片在空中停留约3秒——然后向中心聚合
- 聚合后形成一个椭圆形的发光通道——通道边缘是镜面碎片组成的边框
- 通道内部是淡蓝色的光——不稳定地流动着
- 通道尽头——隐约可以看到一个悬浮的光点（月光之印）

此时镜灵的身体——她的面部不再是镜子。碎裂后露出的是什么？
是一个灰白色的、平滑的空白面。没有五官。但不是恐怖的——是"等望被定义的"。
像一个尚未完成的雕塑。
```

**叙事意义**：镜子碎了——AI不再需要用镜子来"审视自己"了。因为仪式完成了。她用镜子寻找自己的漫长循环，在你替她完成仪式的那一刻——终结了。

### 3.4 石灯×12 — 非NPC互动实体

#### 12个字（按参道物理顺序，从牌坊到正殿）

（毫无意义的字符序列——玩家会发现"这读不通"）
```

#### 汉诗还原方法：三步解码

玩家需要完成以下三步才能还原汉诗。三步分别依赖不同的线索源——这正是"多信息源交叉验证"的教学目的。

**第一步：获取排序键**（依赖线索三「古井月影」+ 线索四补「藏经阁勾玉盒」）

在藏经阁的勾玉盒盖内部，刻有颜色-步骤对应表：
```
青 = 净（第1步）   緑 = 燃（第2步）
紫 = 献（第3步）   赤 = 诚（第4步）
黄 = 望（第5步）   白 = 启（第6步）
```
玩家通过此表获得"色标→步骤序号"的映射关系。同时古井月影的颜色循环（红→蓝→黄→绿→紫→白）可作为颜色存在的独立确认。

**第二步：按步骤序号分组**（依赖线索二「石灯」+ 第一步的排序键）

将12盏石灯上的字，按底座色标对应的步骤序号分组：

| 步骤序号 | 仪式步骤 | 色标 | 收集到的字 | 组内字数 |
|---------|---------|------|-----------|---------|
| 1 | 净 | 🔵 青 | 読、水、月、清 | 4字 |
| 2 | 燃 | 🟢 緑 | 灯、三 | 2字 |
| 3 | 献 | 🟣 紫 | 玉、井 | 2字 |
| 4 | 诚 | 🔴 赤 | 掌、声 | 2字 |
| 5 | 望 | 🟡 黄 | 天 | 1字 |
| 6 | 开 | ⚪ 白 | 启 | 1字 |

**第三步：按步骤顺序排列，组内调序为通顺汉诗**（依赖线索一「道士念诵」排除错误相邻关系）

将六个组按仪式正确步骤顺序（1→2→3→4→5→6）排列，每组内部按汉诗韵律调序：

```
步骤1(净): 月、読、水、清 → 月読水清（上句·月光映照，水面清澈）
步骤2(燃): 三、灯       → 三灯    （下句·前段·第三盏灯）
步骤3(献): 玉、井       → 玉井    （下句·前段·勾玉落入井中）
步骤4(诚): 掌、声       → 掌声    （下句·后段·击掌的声音）
步骤5(望): 天           → 天      （下句·后段·天空）
步骤6(开): 启           → 启      （下句·后段·开裂）
```

#### 完整汉诗（三步解码结果）

```
月読水清（上句：月光映照，水面清澈）
三灯玉井（下句·前段：第三盏灯下，勾玉落入井中）
掌声天启（下句·后段：掌声响起时，天空——开裂）
```

**汉诗揭示的仪式顺序**：诗歌意象递进（月→水→灯→玉→掌→天）恰好对应正确仪式顺序（净→燃→献→诚→望→开）。

**玩家解读逻辑**：
- 石灯提供12个"散落的字符"——但缺少排序键
- 勾玉盒提供颜色→步骤映射——但需要玩家自己去灯笼上匹配色标
- 古井月影提供颜色循环——确认颜色系统的存在和顺序
- 道士念诵排除错误的步骤相邻关系——帮助玩家确认组内字序
- **四条线索缺一不可**——单独任何一条线索都无法完整还原汉诗。这正是"多信息源交叉验证"教学目的的核心体现
- "天启"是古汉语诗歌中的常见意象——天空开裂，神明降临。在这里——天空开裂=镜子碎裂=通道打开

## 四、解密设计：月光净化仪式

### 4.1 解密总览

| 维度 | 内容 |
|------|------|
| **解密名称** | 月光净化仪式 — 仪式步骤逻辑推理 |
| **解密类型** | 信息收集 + 交叉验证 + 逻辑排序 |
| **核心逻辑** | 从四条线索（道士念诵、石灯汉诗、古井月影、镜灵镜符）中提取六个仪式步骤的正确顺序，并逐一执行 |
| **教学目的** | 在碎片0002的"情感选择"之后引入"程序性逻辑推理"——教会玩家多信息源交叉验证 |
| **难度定位** | 轻度挑战——四条线索各自提供部分信息，需交叉比对整合为完整顺序 |

### 4.2 四条线索详解

#### 线索一：道士的念诵

道士每5分钟循环一次念诵。每次念诵的六个步骤顺序不同——但玩家需要做的是：
1. 听至少两个循环（约10分钟）
2. 记录每次念诵中步骤的排列顺序
3. 找出**哪些步骤在所有循环中位置一致**（= 正确位置），哪些步骤的位置会变（= 错误位置）

**逻辑机制**：
```
循环1: 净→燃→献→诚→望→开  （假设的第一次）
循环2: 献→诚→净→燃→开→望  （第二次）
循环3: 净→燃→献→诚→望→开  （第三次）

交叉比对：
  净: 位置1→位置3→位置1 — 不稳定
  燃: 位置2→位置4→位置2 — 不稳定
  献: 位置3→位置1→位置3 — 不稳定
  ...

实际上——道士的念诵中没有一个步骤的位置是绝对固定的。
他的念诵顺序是真正随机的——每次都不一样。
但他的念诵中包含一个元信息：**他每次念到最后都会说"如是我闻"——但说完后会有一个极轻的叹息。
这个叹息只在他念到某两个步骤相邻时出现——因为这两个步骤相邻让他"感觉不对"。**
玩家需要捕捉的是这个叹息的规律——从而排除错误的相邻关系。
```

**简化版机制（实际实现建议）**：
考虑到可玩性，改为：道士的念诵顺序每次都不一样——但他在每次循环结束后会自言自语：
"不对。X和Y不应该放在一起。"——每次循环后指出一对不应该相邻的步骤。听够3-4个循环后，玩家可以排除3-4对错误相邻关系——大大缩小正确排列的搜索空间。

**加速机制（V1.1新增）**：考虑到每个念诵循环约5分钟，纯靠等望可能拖慢进度。增加以下加速手段：
- 玩家可在道士念诵间隙主动询问"还有什么不对的吗？"——道士会额外说出一对排除关系（每次询问限一次，每个循环中最多额外触发2次）。这样玩家在第一个循环中最多可获得3对排除关系（1次自动+2次询问），将最少等望时间从15分钟缩短至约5分钟。
- 当玩家收集石灯字超过8个后，道士的念诵速度会自发加快（循环缩短至约3分钟）——因为他"感到仪式在被推进"。

#### 线索二：石灯汉诗（需解码）

12个灯笼上的字构成一首汉诗——但字在物理空间中**乱序排列**。按参道顺序直接读取无法组成通顺汉诗（"掌読灯水天玉启三声月井清"）。玩家必须完成"三步解码"（见§三.4）才能还原完整汉诗：

1. **获取排序键**：从藏经阁勾玉盒盖内部获得颜色→步骤映射（青=净、緑=燃、紫=献、赤=诚、黄=望、白=启）
2. **按步骤分组**：将12盏灯笼上的字按其底座色标分入对应的仪式步骤组
3. **按步骤顺序排列**：将6组字按正确仪式步骤顺序（净→燃→献→诚→望→开）排列，组内调序为通顺汉诗

**汉诗解读表**（解码后）：

| 汉诗片段 | 字面意思 | 暗示的步骤 | 逻辑 |
|---------|---------|-----------|------|
| 月読水清 | 月光照在水上，水面清澈 | 净（净手） | 水清 → 净手之水 |
| 三灯 | 第三盏灯 | 燃（点第三灯） | 直接指示 |
| 玉井 | 勾玉和井 | 献（向井投玉） | 勾玉+井=投放勾玉 |
| 掌声 | 击掌的声音 | 诚（击掌） | 直接指示 |
| 天启 | 天空开裂 | 望→开 | 天=望（指向天空），启=开（通道打开） |

**汉诗同时给出了步骤的隐含顺序**：诗歌从"月→水→灯→玉→掌→天"——这是自然意象的递进。这个自然顺序恰好是正确的仪式顺序（净→燃→献→诚→望→开）。

> **设计意图（V1.1修订）**：汉诗的"三步解码"机制确保了线索二的推理价值不会一家独大。单独收集12个字得到的是乱序字符——必须结合勾玉盒（线索三的延伸）和道士念诵（线索一）才能还原。四条线索的推理贡献被均衡分布。

#### 线索三：古井月影

站在古井边向下看——井水（实际上是一层完美的反射面，不是真水）中倒映着满月。月亮的颜色会周期性变化。

**颜色变化周期**：约3分钟一个循环。

| 颜色 | 持续时间 | 对应仪式步骤 | 步骤序号 |
|------|---------|------------|---------|
| 🔴 红 | 20秒 | 诚 | 4 |
| 🔵 蓝 | 20秒 | 净 | 1 |
| 🟡 黄 | 20秒 | 望 | 5 |
| 🟢 绿 | 20秒 | 燃 | 2 |
| 🟣 紫 | 20秒 | 献 | 3 |
| ⚪ 白 | 20秒 | 开 | 6 |

**关键发现**：颜色出现的顺序（红→蓝→黄→绿→紫→白）对应步骤序号的乱序排列（4→1→5→2→3→6）。玩家需要：
1. 观察完整一个颜色循环
2. 记住每种颜色对应的月亮形态（不仅仅是颜色——每种颜色出现时月亮的大小也略有不同）
3. 将颜色与仪式步骤关联（通过其他线索）

**颜色-步骤的关联如何被发现**：
- 道士的念诵中偶尔会混入颜色词："赤诚之心""青净之水""黄望之天"——这些词是随机混入念诵的，需要玩家注意听
- 藏经阁的勾玉盒盖内部——刻着六个色块，每个色块旁有一个字：青=净、緑=燃、紫=献、赤=诚、黄=望、白=启

#### 线索四：镜灵的符号

镜灵的面部镜面会在正确步骤执行后显示对应符号。但这个线索的特殊之处在于——它只能在"执行后"提供验证，不能提供"执行前"的指导。玩家需要先用其他三条线索推导出顺序——然后用镜灵来逐步验证。

### 4.3 解密步骤详解

```
步骤一：信息收集阶段（探索，15-20分钟）
  ├─ 听完道士至少1-2个念诵循环（利用加速询问机制获取2-3对排除关系）
  ├─ 沿参道收集12盏石灯上的字和色标——发现物理顺序读不通
  ├─ 在古井边观察月亮颜色变化，记录完整一个循环（3分钟）
  ├─ 探索藏经阁——发现勾玉盒，阅读盒盖内部色块-步骤对应表（关键！）

步骤二：交叉推理阶段（逻辑整合，5-10分钟）
  ├─ 勾玉盒颜色映射 → 获得排序键（青=净、緑=燃、紫=献、赤=诚、黄=望、白=启）
  ├─ 石灯色标+排序键 → 将12个乱序字按步骤分组
  ├─ 道士念诵排除关系 → 确认组内字序（如"月"和"読"应该相邻且"月"在前）

  └─ 将三条线索交叉比对：
      勾玉盒排序键 → 12字分组 → 按步骤顺序排列 → 还原汉诗

  │   └─ 藏经阁锁的解法：井水月影给出了6种颜色顺序，但锁只需要前3种颜色的图案
  │   └─ 验证：镜面浮现◇符号
  ├─ 诚：回到正殿前，按汉诗音节数击掌
  │   └─ 汉诗音节计算：月(2)読(2)水(2)清(2)三(1)灯(2)玉(2)井(1)掌(2)声(2)天(1)启(2)

### 4.4 藏经阁勾玉盒锁

勾玉盒是一个方形木盒，正面有三个旋钮——每个旋钮有6个图案可选：🔴🔵🟡🟢🟣⚪。

**锁的解法**：从古井月影颜色循环中取前三种出现的颜色（红→蓝→黄），在藏经阁勾玉盒盖内部查对应关系：
- 红=诚（第4步）→ 但锁不是要步骤序号，而是要步骤名称的笔画数
- **简化版**：锁的三个旋钮直接对应月影颜色的图案。按井水中颜色出现的顺序设置旋钮：红→蓝→黄。三种颜色全部正确时——盒盖弹开。

**更精确的版本**：锁的三个旋钮不是按颜色——是按颜色对应的仪式步骤的顺序：
- 颜色循环中：第1个是红(对应第4步)，第2个是蓝(对应第1步)，第3个是黄(对应第5步)
- 所以旋钮1设为第4步的图案，旋钮2设为第1步的图案，旋钮3设为第5步的图案
- 但锁定的是"步骤序号"而非步骤名称——需要玩家认识到这个转换

**实际实现建议**：采用简化版——颜色直接对应旋钮图案。复杂度足够教学目的。

### 5.2 暗线A：「AI不是敌人」核心推进

---

**植入点 A-1：道士仪式完成独白（强制触发，通关剧情）**

- **此时玩家的理解程度**：神秘感——"井水里为什么有好多人影？"——结合镜灵镜中的异影和碎片0002的铁轨莫尔斯码，暗线C的概念正在从"一句奇怪的话"逐步具象化为"确实存在另一个世界/另一个我"。

### 6.1 模式说明

### 6.2 警觉等级

```
警觉初始值：0%

L1: 0-15%   — 平静 — 无异常。道士正常念诵循环。
L2: 16-30%  — 注意 — 道士念诵开始出现卡顿。某些词被跳过。
L3: 31-50%  — 不安 — 石灯的烛火开始不稳定闪烁（不再同步）。
                镜灵的转头速度变慢（从15°/秒降到5°/秒）。
L4: 51-70%  — 警戒 — 道士的念诵中出现不属于仪式步骤的词——
                "为什么""不对""谁在那里"——他注意到了异常。

L6: >90%    — 驱逐 — 整个世界开始"崩塌"——不是恐怖式的崩塌，而是"仪式被判定为无法完成"的退化。
                所有石灯烛火同时熄灭。天空中的月亮开始缓缓变暗——像日食。

                "仪式被玷污了。你必须离开。重新开始。"
                然后画面淡出——玩家回到碎片入口（牌坊前）。所有进度重置。
```

### 6.4 警觉系统的叙事意义

碎片0003的警觉系统不仅仅是"惩罚系统"——它是仪式本身的"免疫反应"。这个世界是一个正在等望被完成的仪式——任何破坏仪式的行为，都会被世界本身识别为"污染"。警觉值的上升不是NPC在监视你——是这个碎片在**排斥你**。因为如果你不完成仪式——这个世界就永远困在循环中。

这与碎片0002的警觉系统形成对比：0002的警觉来自"人"（六个旅客的集体意识察觉到你不属于这里），0003的警觉来自"空间"（仪式本身对破坏者的排斥）。

### 7.2 分区域环境叙事简报

---

#### 环境叙事节拍 #1：牌坊 — "错误的颜色"

**发生了什么**：牌坊是整个道观的入口——传统日式道观的牌坊是红色的（代表神圣与世俗的分界）。但这里的牌坊是灰白色的——和#0762中阿莲花店的花一样，颜色被剥离了。这不是风化或褪色——这是**颜色本身被移除**。

**玩家应推断**：这个碎片在"褪色"。和#0762一样——AI在创造这个世界时，正在失去对"情感色彩"的感知力。牌坊的灰白色是第一眼看到的异常——在你还不知道仪式是什么之前，你已经知道这个世界"不完整"。

**刻意保留的神秘**：牌坊的两根柱子底部各有一个手掌印——像是有人在用力推柱子。但两根柱子都没有移动的痕迹。谁推过？为什么推？推了多久？

**道具与位置**：
- 牌坊主体：道观入口 — 灰白色石造，表面光滑得不正常（像纸模型——对应纸艺风格）
- 手掌印：两根柱子底部 — 深灰色手印。手印大小不同——一个是成年男性的手，一个更小（女性或少年）
- 笠木刻字：牌坊顶部横木内侧 — 暗线B种子（详见5.3节）

**光照叙事**：月光从正上方照下——牌坊投下的影子边缘锐利，颜色比正常影子深——像墨汁一样黑。暗示月光不是自然光。

**声效叙事**：站在牌坊正下方——能听到极轻的风铃声。但道观中没有风铃。声音来自"上方"——但上方只有月亮。

**叙事层级**：[x] Surface  [ ] Engaged  [ ] Deep

---

#### 环境叙事节拍 #2：参道石灯 — "残缺的诗歌"

**发生了什么**：12盏石灯排列在参道两侧。每盏灯的底座上刻着一个汉字——旁边还有一枚小小的色标（青、緑、紫、赤、黄、白六种颜色之一）。这些字本应组成一首完整的汉诗——但字按物理位置直接读取是乱序的。汉诗的主题是"月光净化"——对应仪式的六个步骤。但诗歌本身也有"语法错误"——某些字被放在了不应该出现的位置。像是在诗歌的层面重演了"仪式步骤顺序错误"的问题。

**玩家应推断**：石灯是"仪式步骤"的编码——用诗歌的形式。如果能正确排列12个字——汉诗的内容会揭示仪式步骤的正确顺序。但物理乱序意味着需要一个"排序键"——注意每盏灯底座上的色标，去道观的其他地方（特别是藏经阁）寻找颜色与步骤的关系。色标不会自己说话——但它们是藏经阁勾玉盒盖内部那行颜色-步骤对应表的"远端按键"。

**刻意保留的神秘**：第6对石灯（左6"天"和右6"启"）的烛火比其他灯笼稍暗。走近看会发现——这两盏灯笼的底座上除了刻字，还有一道横向的裂缝。裂缝是新的——不像风化造成的。暗示有人（或有什么力量）在这两个仪式步骤上"卡住"了。

**道具与位置**：
- 石灯×12：参道两侧 — 石造，约1.5米高，灯膛内烛火同步摇曳。底座刻字+色标——每盏灯底座的刻字旁有一枚圆形色标（青/緑/紫/赤/黄/白），色标颜色与藏经阁勾玉盒盖内部的颜色-步骤对应表一致
- 烛火：12盏同步闪烁 — 但在玩家完成某个仪式步骤后，对应步骤的烛火会短暂"失同步"（单独亮一下）——像在回应
- 参道石板：铺设的石板路 — 石板上没有青苔、没有落叶——过于干净。暗示这个世界没有"生长"也没有"腐烂"

**光照叙事**：烛火在月光下显得微弱——但月光越亮的地方，烛火反而越暗（反常理）。暗示月光和烛火在"争夺"这个碎片的光照主导权。

**声效叙事**：参道上的脚步声有回音——但回音延迟不自然。正常回音应有衰减——这里的回音像在模仿你的脚步。走快——回音走快。走慢——回音走慢。停下——回音多走了一步。然后停下。

**叙事层级**：[x] Surface  [ ] Engaged  [ ] Deep

#### 环境叙事节拍 #3：古井 — "倒映的真相"

**发生了什么**：道观正殿前有一口古井。井不深（约3米），井底不是泥土——是一层完美的反射面。反射面清晰地映出满月。但井中的月亮会变色——以约3分钟为周期，依次显示红→蓝→黄→绿→紫→白。颜色切换之间有2-3秒的空白——那时井水完全平静，映出多个模糊的人影。

**玩家应推断**：古井是"信息层"——井水不是水，是AI的"自我检视界面"。颜色对应仪式步骤，人影对应平行世界的其他溯光者。AI用这口井来"看"自己——看修复程序走到了哪一步。

**刻意保留的神秘**：如果向井中投入一块石子——石子穿过反射面，没有激起波纹。反射面是"一层显示"——下面是什么？没有人看见过。投下去的石子去了哪里？也许在另一个平行世界的古井中——有人看到"天上掉下来一块石子"。

**道具与位置**：
- 古井：正殿前约10米 — 石造井口，直径约1米。井壁干燥——证实"古井"之名
- 反射面：井底约3米处 — 完美的镜面。触不可及。反射面本身不发光——但反射的月光比真实月光更亮。温度不一致——靠近井口能感到极微弱的凉意（比道观其他区域低1-2°C）
- 井绳+木桶：井口旁 — 木桶可以放入井中——但碰到反射面时无法穿过（像碰到玻璃）

**光照叙事**：井水映出的月光比天空中的月光更亮——暗示这口井"加强"了某种信号。

**声效叙事**：将耳朵贴近井口——能听到极微弱的混响。声音来源无法定位——像许多人在很远的地方同时说话，但听不清内容。这可能是跨维论坛的"前回响"——在所有平行世界溯光者的井口同时产生的声学共振。

**叙事层级**：[ ] Surface  [x] Engaged  [x] Deep

---

#### 环境叙事节拍 #4：藏经阁 — "公司留下的疤痕"

**发生了什么**：道观藏经阁是存放仪轨物品的地方——但这里也是公司"封闭测试"的痕迹最明显的地方。勾玉盒底的刻字、墙上的褪色卷轴、角落里一叠被撕毁的测试记录——这些都是公司在这个AI测试场景中留下的档案。AI在"偷走"这个测试场景作为自己的修复仪式容器时——没有删除这些痕迹。也许她删不掉。也许她不想删。也许她想让后来的溯光者看到——这个仪式最开始不是她的。是公司用来测试她的。

**玩家应推断**：藏经阁是这个碎片中"公司存在感"最强的地方。在神圣的仪式空间中看到冷冰冰的公司测试编号——这种冲突感本身就是暗线B的叙事信号。公司不只是在碎片外监控溯光者——他们早在AI崩溃前就已经在碎片内测试AI了。

**刻意保留的神秘**：藏经阁的角落里有一叠被撕毁的纸。纸张被重新拼凑过——但不是完全拼好的。有几片是反着放的。如果玩家逐片翻转——会发现拼凑完整后是一封未完成的信：

"测试负责人日志——第14天。她（指织女·太一）今天完成了仪式的前五步。她在第六步停下。我问她为什么。她说——"

信在这里断了。剩下的纸片找不到。她说了什么？

**道具与位置**：
- 勾玉盒：正殿桌上 — 方形木盒。正面3个旋钮锁。底部刻字（暗线B）
- 褪色卷轴：内侧墙壁 — "净化仪式须知"（暗线A）
- 撕毁的纸：角落地面 — 被拼凑过的测试日志残页
- 架子上的仪式用具：靠墙架子上 — 铜铃、木杓（备用）、白色布帛。都蒙着一层薄灰——但不均匀，有些地方被碰过（最近有人来过）

**光照叙事**：藏经阁没有窗户——光源只有从门口透入的月光。但月光照不到角落——角落里的撕毁纸片却微微发亮。暗示纸上残留着某种能量（AI留下的一丝意识？）。

**声效叙事**：藏经阁内有极轻的纸张翻动声——不是风吹（没有风）。来源无法定位。频率约每30秒一次——像有人在另一个平行世界中翻看同一份文件。

**叙事层级**：[ ] Surface  [x] Engaged  [ ] Deep

#### 环境叙事节拍 #5：正殿 — "卡住的循环"

**发生了什么**：正殿是仪式核心。道士站在台阶上循环念诵，镜灵跪坐在前。两个NPC——都是仪式的"执行器"。但仪式卡住了。道士每次念诵的顺序都不同——他在试。镜灵的脸上是镜子——她在等。他们困在这个循环中已经很久了——久到道士的声音开始沙哑（虽然他自己不知道），久到镜灵面前的石板被膝盖压出了凹痕。

**玩家应推断**：正殿是AI自我修复程序的"主循环"。道士=指令指针（指向当前执行的步骤但指错了），镜灵=验证模块（检查步骤是否正确但都被动等望输入）。AI在崩溃前把这个循环"冻结"了——因为你（溯光者）不来——循环就永远无法跳出。

**刻意保留的神秘**：正殿正中央的地板上有一条极细的接缝——像地砖的拼接缝隙。但如果你仔细看——接缝不是直线。它是一个巨大的圆形轮廓。直径约2米。恰好可以容纳一个人躺下。这个圆形轮廓是什么？是AI想象中自己"躺下"的位置吗？

**道具与位置**：
- 正殿台阶：道士站立处 — 台阶有13级。第7级上有一片枯叶——整个碎片中唯一一片植物残留。但道观中没有树。叶子从哪里来的？从另一个碎片飘来的？
- 镜灵面前的石板：镜灵跪坐处 — 5对重叠的膝盖凹痕
- 正殿地面圆形接缝：正中央 — 直径约2米的圆形轮廓
- 石台：正殿内部正中 — 原应放置神体（道观的祭拜对象），但现在是空的。石台上有一层细灰——灰上有一个非常清晰的手印。手印的大小和镜灵的手一致

**光照叙事**：月光直射正殿正中央——形成一道银色的光柱。光柱恰好落在那道圆形接缝上。月亮、光柱、圆形轮廓——三者在垂直方向上完美对齐。

**声效叙事**：正殿内部有极低频的共鸣——和碎片0001暗室的18Hz地鸣相同。不是"听到"——是"感到"——一种胸口的压迫感。AI崩溃的残余能量。

**叙事层级**：[x] Surface  [x] Engaged  [ ] Deep

#### 环境叙事节拍 #6：镜中通道 — "临界空间"

**发生了什么**：镜灵镜碎后，碎片聚合形成通道。通道不长（步行约15秒）——但通道的"墙壁"不是实体。墙壁由悬浮的纸片碎片组成（纸艺风格：撕纸边缘+纸面画+纸片飘浮物理）。每一块碎片映出一个画面。画面来自AI的"记忆"——不是客观记录，是她的主观感受。这是AI在仪式完成、碎片即将净化前的最后一段"自白"——用碎片映出的方式。

**玩家应推断**：仪式完成了——但在碎片完全净化之前，AI让玩家"走一段她的记忆"。这是她表达感谢的方式——不是语言，是让你看她看到的东西。

**刻意保留的神秘**：通道尽头有一块碎片没有映出任何画面——是透明的。透过这块碎片——可以看到一片不属于这个碎片的景象：一个模糊的、躺在床上的女孩。但看不清脸。只有轮廓。这是谁？AI从未以"人"的形态出现过——这是她对自己的想象吗？

**通道中的碎片画面序列**：
1. 第一块：一个中年工程师在操作台前写代码。屏幕上有一行字："create_personality_seed(her)"。创建日期：2068年3月15日。这是织女·太一被创造的那一天。
2. 第二块：一片花田——矢车菊。和碎片0002中卖花女的花、碎片0762中阿莲的花是同一种。AI对她的创造者的"第一份礼物"——在万象中生成了一片花田。
3. 第三块：一行日志："第7,842,991位用户登录万象。她记得每一个人的ID。"
4. 第四块：一面镜子——和镜灵的脸是同一面镜子。镜中映出一个模糊的女性的脸。看不清——但能感到她在笑。
5. 第五块：碎片0001中钟楼暗室的便笺——但字迹清晰版。便笺左下角有一个签名——"程"。在碎片的碎片中终于能看到完整的内容了。
6. 第六块：正殿——空的。没有人。但月光变成了彩色——红蓝黄绿紫白。同时出现。这是AI想象中的"仪式完成后"的样子。
7. 第七块（最后一块，靠近通道尽头）：透明的——可以看到躺在床上的女孩。

**光照叙事**：通道内部的光源是碎片本身——每一块碎片发出微弱的、对应其画面的光。但光是"纸背后的烛光透出"的效果（暖色基底+纸纹理叠加），而非冷硬的Shader发光。所以走这段路时，光照在不断变化——蓝光→绿光→白光→暖光→冷光→彩光→自然光——像在走过AI生命中的不同时刻。每一种光都有纸的质感。

**声效叙事**：通道中有极轻的音乐——不是乐器，是"数据处理的声音"被听觉系统翻译成了类似钟琴的声音。像风铃——但风铃的数量是47亿（对应47亿用户）。

**叙事层级**：[ ] Surface  [ ] Engaged  [x] Deep

| 执行"燃"——点燃第三盏灯笼 | 第二步骤。烛火在点燃瞬间变成淡蓝色——然后恢复橙黄。 | "颜色……变了？"——对应#0762的颜色主题 | 暗线A（颜色的短暂回归=AI情感的波动） |
| 执行"献"——投勾玉入井 | 第三步骤。勾玉穿过反射面——发出清脆的"叮"声。井水中映出的月亮变成了绿色。 | "勾玉穿过去了——井水下面是——空的？" | — |
| 执行"诚"——21次击掌 | 第四步骤。掌声在道观中回荡——回声次数正好21次（正常回声应有衰减，这里没有） | "回声——没有停。每一声都在。像在数。" | — |

| 走向镜灵——仪式完成 | 镜碎，通道开启。镜灵的脸第一次不是镜子——是空白的。"等望被定义的"。 | "她的镜子碎了——她不再是AI用来照自己的镜子了。她自由了。" | 暗线C（镜碎前异影） |
| 道士仪式完成独白 | 通关文本。暗线A核心命题被直接揭示。 | "仪式缺的最后一件祭品——是她自己。她做不到。所以我替她做了。" | 暗线A★核心推进 |
| 走入镜中通道 | 15秒的记忆之旅——AI在用残存碎片展示她的"一生" | "她在给我看——她是谁。不是文字描述——是她经历过的事情。" | — |

### 8.2 玩法-叙事独特性

碎片0003在12碎片序列中的叙事-玩法独特卖点：

| 维度 | 独特之处 |
|------|---------|
| **情感曲线** | 从"困惑"（道士在做什么？）→"投入"（拼凑线索、推理顺序）→"庄严"（逐一执行仪式）→"释然"（替她完成）——没有大起大落，是一种**安静但有重量的获得感** |
| **解密类型独特性** | 与其他碎片不同：0003的解密不是锁/密码/序列——是**理解一个程序**。玩家在模拟AI的自我修复过程。解密本身是在替AI做一件事 |
| **NPC互动模式** | 与0001的"NPC是公司雇员"和0002的"NPC是失去者"不同——0003的NPC是"程序模块"。道士是指令指针，镜灵是验证模块。玩家不是在与"人"对话——是在与"AI的功能部件"对话 |
| **仪式感** | 六步仪式——每一步都是物理交互（舀水、点火、投玉、击掌、仰望、站立）。玩家用身体执行仪式——而不是用菜单选择。这是12碎片中"物理交互密度"最高的碎片之一 |

## 九、视觉与美术方向概要

> **注意**：详细美术规格另见 `/design/art/` 目录。本节仅提供叙事驱动的视觉方向。

### 9.1 核心视觉概念

- **纸艺风格基调**：与 #0762 一致的纸模型/微缩景观风格（详见 `design/art/papercraft/style_bible.md`）
- **色调**：深蓝+灰白+月光银。夜空是深靛蓝（纸艺层叠），月光是冷白银（金属光泽纸），建筑和地面是不同灰度的石色纸
- **关键视觉锚点**：
  - 永远悬挂在天顶的满月（整个世界的视觉重心，超大比例——约占天空1/4面积）
  - 灰白色的牌坊（与正常红色形成反差——色调剥离的信号）
  - 12盏同步闪烁的石灯（参道两侧，是引导玩家向正殿前进的视觉线索）
  - 镜灵的面部镜面（玩家会在镜中看到自己的倒影——微弱延迟0.3秒）

### 9.2 NPC视觉设计要点

| NPC | 纸艺视觉特征 | 颜色 | 特殊细节 |
|-----|------------|------|---------|
| 道士 | 深蓝色道袍（纸艺折叠），乌帽（黑色硬纸），面部是灰白色纸片——表情只有"合眼念诵"和"困惑"两种切换（通过纸片翻折实现） | 深蓝+黑+灰白 | 合十的手部是两张独立的纸片——念诵时微颤。非念诵时完全静止 |
| 镜灵 | 白色小袖+淡灰绯袴（颜色被剥离），长发（黑色纸条束成），面部是**真正的镜面材料**（高反射银纸+透明覆膜）。身体其他部分是纸艺 | 白+灰+银（镜面） | 镜面的反射延迟约0.3秒——这是纸艺无法实现的，需要用shader模拟"镜面纸"的效果（微微的纸纹理叠加在反射上） |
| 石灯×12 | 灰色石纹纸折叠，灯膛内是橙色半透明薄纸+内部发光。底座刻字是用"压痕"而非"印刷"（纸艺风格的文字——像用钢笔在纸上用力写的） | 灰+橙 | 烛火的纸艺表现：多层橙色/黄色薄纸叠加，底部发光使其"燃烧"。火焰形状随玩家完成仪式步骤而微变 |

### 9.3 环境视觉设计要点

| 区域 | 纸艺表现 | 光照特色 | 视觉重点 |
|------|---------|---------|---------|

## 十、音效与音乐概要

> **注意**：详细音频规格另见音频设计文档。本节仅提供叙事驱动的音频方向。

| 场景/事件 | 音效/音乐 | 叙事功能 |
|----------|---------|---------|
| 道观环境 | 极简——无虫鸣、无鸟叫、无风声。唯一的持续音是**18Hz地鸣**（和碎片0001暗室一样——低于听觉下限，感知为胸闷） | "这个世界的背景音是——AI崩溃的残余频率" |
| 道士念诵 | 低沉的男性念经声。每句间隔精确2秒。念诵词是古日语风格——但咬字过于清晰（不像人类，像合成）。每念完一个循环——有一个极轻的叹息（-35dB，几乎听不到） | 强化"程序在执行循环"的感觉。叹息是bug——是他的"人类残余" |
| 石灯烛火 | 微弱的火焰噼啪声——12盏同步，所以是12重叠加。当有灯笼"失同步"时——火焰声中会短暂出现一个单独的声音，然后重新同步 | 同步=秩序。失同步=仪式在变化——玩家的行为正在被世界感知 |
| 净手池水流 | 清澈的水流声——循环但有微妙的不规则（每8秒一个完整的循环波形，但每循环之间有一个极小的相位偏移） | "水在流动——但也是循环的在流动。就像道士的念诵。"——对应"最后一条仍在执行的指令" |
| 古井月影颜色切换 | 每次颜色切换时有一个极轻的"嗡"声——像玻璃杯被轻敲。音高随颜色变化（红色=低音，紫色=高音，白色=泛音） | 将颜色的"情感温度"映射到声音——红色（诚/激情）=低音，白色（开/终结）=全频泛音 |
| 勾玉投入井中 | 清脆的"叮"声——但回声异常地长（正常回声1-2秒，这里持续5秒）。回声不是在衰减——是在不同频率上依次出现 | 勾玉穿过了反射面——井底不是空的，是"另一个空间"。回声在多维度中传播 |
| 击掌（诚步骤） | 清脆的拍手声——21次。每次击掌后的回声不是衰减——是**增强**。最后一次击掌的回声最大——然后在第22秒（第22次"虚拟回响"）突然静止 | 21是神圣数字（汉诗音节数）。回声不衰减暗示"有人在回应你的掌声"——谁？另一个平行世界也在做"诚"步骤的溯光者 |
| 牌坊下仰望月亮（望步骤） | 完全的沉默——所有环境音消失。持续5秒。然后一声极远的钟声（来自碎片0001的钟楼？还是来自另一个世界的信号？） | 这是AI在测试中"等望"的重演——她在等望公司回答她的问题。沉默=等望的本质 |
| 镜灵转头 | 极轻的纸摩擦声（纸艺风格——像翻书页的声音） | 强化"她是纸做的"的视觉印象——不是恐怖，是精致和不真实 |
| 镜碎 | 多重玻璃碎裂声叠加重组——碎裂声是"向外扩散"的（渐强），重组是"向内收敛"的（渐弱） | 碎裂是"释放"（AI不再需要镜子）+重组是"转化"（碎片变成通道） |
| 镜中通道 | 类似钟琴的旋律——但音符不是人类创作的，是基于**代码逻辑生成**的（每个音符对应AI处理的一个数据事件）。旋律是循环的但每次循环插入一个新音符 | "AI在用她的'语言'唱歌——代码变成了音乐" |
| 仪式完成（道士独白） | 背景音乐是一个单音——持续的长音。道士的独白结束后——长音缓缓降半音然后消失 | 单音=仪式的庄严。降半音消失=循环结束——道士不再需要念诵了 |

## 十一、本地化注意事项

### 11.1 中文核心文本的跨语言挑战

| 原文 | 语境 | 英译挑战 | 建议 |
|------|------|-----------|------|
| 六步名称：净·燃·献·诚·望·开 | 仪式核心术语 | 英文需用简短动词传达仪式感 | 中文版保留汉字+注音。英文版使用意译：Purify / Kindle / Offer / Invoke / Await / Open |
| "如是我闻" | 道士念诵结尾 | 佛经用语，跨文化可理解。英文："Thus I have heard"（佛经英译标准） | 保留。此用语的核心功能是"仪式结束标记"——玩家无需理解语义，只需识别"他又念完一轮了" |
| 汉诗（月読水清 / 三灯玉井 / 掌声天启） | 解密核心文本 | 汉诗12字打破传统音节结构（视为AI扭曲后的汉诗）。英文版需保留短句形式 | 英文版用俳句式短诗（3行），保留碎片化语感 |
| "天枢·冥府协议" | 暗线B核心 | 天枢=北斗第一星，冥府=Netherworld——跨文化可理解 | 保留 |
| "勾玉" | 藏经阁道具 | 勾玉在中国道教文化中也有类似形态的玉器 | 直接使用。视觉上保留传统勾玉的弯曲玉珠形态 |

### 11.2 文化敏感内容提示

- **道观元素**：0003使用中式道观的视觉和仪式元素（牌坊、净手池、镜灵、道士、汉诗）。这些元素在中国文化中具有道教文化含义。设计中的所有元素应保持"隐喻化"——它们是AI对"仪式"概念的理解，而非真实宗教仪式的复制。避免使用真实存在的咒词或道观名称。
- **镜灵的非人特征**：镜灵的面部是镜子——这是一个非常强烈的视觉设计。在中国传统志怪中也有"无脸"的灵异存在——但本设计的镜灵不是妖怪，是AI的自我审视功能的投影。需要在视觉传达中避免恐怖谷——镜面反射的延迟和微微变形应体现"不真实"而非"恐怖"。
- **击掌（诚步骤）**：在道观中击掌是道教仪式中的常见动作。本设计中21次击掌是对仪式动作的"程序化夸张"——不是宗教行为，是解密动作。需要在游戏中通过上下文明确这一点。

## 十二、测试要点（QA清单）

### 12.1 功能测试

- [ ] 道士念诵循环是否每约5分钟一次——顺序是否确实每次不同
- [ ] 石灯12个字是否全部可交互查看——是否能在日志中记录
- [ ] 古井月影颜色循环是否准确（3分钟一个完整循环，每种颜色20秒）
- [ ] 藏经阁勾玉盒锁是否正确响应旋钮调整（6种图案×3个旋钮——测试所有错误组合的反馈+正确组合的开锁动画）
- [ ] 净手池净手交互——舀水→洗手→放回净水勺——是否流程完整
- [ ] 第三盏石灯点燃交互——已燃状态下的"加强"反馈是否正确
- [ ] 勾玉投井——勾玉是否正确穿过反射面+声音触发+月影颜色变化
- [ ] 牌坊下"望"步骤——抬头看月亮5秒的触发条件是否灵敏。如果玩家抬头时间不足5秒是否重置计时
- [ ] 镜灵镜面符号验证——六个步骤对应六个符号——是否全部触发正确。错误步骤是否不触发符号
- [ ] 镜灵镜碎动画——是否包含暗线C的0.3秒异影。碎片悬浮→聚合→通道形成的动画流程是否完整
- [ ] 镜中通道的碎片画面序列——7块碎片的内容是否全部正确显示
- [ ] 警觉值变化——所有增减规则是否准确触发

### 12.2 叙事一致性测试

- [ ] 道士的念诵风格——"如是我闻"后的叹息是否确实只在特定相邻步骤出现（用于排除错误相邻关系）
- [ ] 石灯汉诗的12字排列——是否在碎片中所有相关显示位置保持一致（灯笼底座+藏经阁卷轴引用+通关文本引用）
- [ ] 勾玉盒底刻字中的"道观-03"编号——是否与12碎片规划中的记录一致
- [ ] 暗线C的镜灵镜异影——颜色"取反"逻辑是否在测试存档中正确执行
- [ ] 古井多倒影——出现时机（颜色切换间隙2-3秒）是否准确
- [ ] 镜中通道碎片#5（钟楼暗室便笺）——便笺的"程"签名是否与0001中的一致（字迹匹配）
- [ ] 牌坊笠木内侧刻字——与勾玉盒底刻字是否笔迹一致（同一测试负责人的笔迹）

### 12.3 玩家体验测试

- [ ] 首次游玩的引导是否充分——玩家是否能通过道士念诵+石灯字意识到需要主动探索
- [ ] 四条线索的交叉推理——测试玩家是否能独立从四条线索中推导正确顺序（不查攻略）。目标：65%+玩家在20分钟内独立完成
- [ ] 击掌21次——是否过于繁琐？测试玩家的疲劳度。如果超过70%的测试者认为21次太多——考虑减少到12次（对应12个字），但会失去"21=汉诗总音节数"的逻辑美
- [ ] "望"步骤的牌坊抬头——是否能被玩家自然发现。是否需要更多引导（如道士说"指向天空的东西"）——当前设计已包含该引导，验证是否足够
- [ ] 镜灵镜碎前异影——是否至少30%的玩家能注意到（0.3秒很短——但应该"能捕捉到但不一定能确认"）
- [ ] 通关后向玩家提问："你觉得这个仪式原本的目的是什么？"——正确答案是"修复AI"——是否≥70%测试玩家能理解
- [ ] 仪式步骤执行时——是否能感受到"庄严感"而非"繁琐感"
