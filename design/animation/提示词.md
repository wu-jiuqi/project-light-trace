# 开场动画 — 美术资源 AI 提示词全集

> **版本**：V1.1 — 方案B（先撕裂后归位到 star_map 网格）
> **关联文档**：`opening_cinematic_design.md`
> **生成方式**：AI 图像生成（ImageGen / Midjourney / SD）
> **风格基线**：`design/art/papercraft/style_bible.md` — 纸工拼贴，非暗黑童话

---

## 阅读前提：本项目风格与现有 skill 模板的差异

现有 `papercraft-building-gen` 和 `papercraft-item-gen` 使用 **「纸雕暗黑童话」** 风格标签（dark fairy tale / surreal whimsical / eerie），适用于独立 RPG 物品和建筑。

**本项目的视觉基调不同**——见 style_bible：

> 「俯视 RPG + 整片纸偶 + 高分辨率纸片资产 + 局部超现实拼贴。常态世界应像可游玩的纸工绘本或桌面纸偶剧场。」

本提示词集：
- **继承** skill 的结构（masterpiece / highly detailed / paper construction language）
- **替换** 风格标签为项目的「纸板舞台 / 纸工绘本 / 暖光纸偶」语言
- **使用** style_bible 的材质族名称（kraft paper / gray_card / tracing_paper / fiber_paper / foil / fasteners）

## 六、风格一致性检查清单

生成后，逐张对照 style_bible 的不可变规则：

- [ ] **材质族**：是否只使用了 kraft / gray_card / book_page / fiber_paper / tracing_paper / foil / fasteners？
- [ ] **裁切边缘**：是否有轻微不规则裁切？剪影类资产是否保留了剪刀切边？
- [ ] **光源方向**：所有资产的光源是否来自左上？（阴影向右下偏移）
- [ ] **阴影颜色**：是否使用暖灰/冷灰而非纯黑？纸片投影是否克制？
- [ ] **禁止项**：无光滑矢量边缘、无塑料/橡胶质感、无写实3D、无纯黑阴影
- [ ] **暗黑童话？**：确认所有资产**不包含** dark fairy tale / eerie / gothic 标签风格

```
[15.0s — 降调（非情感，模拟"数据流被中断"的效果）。
      这句和前后的语调有明显区别——像一个 AI 在切换运行模式]
"2077年3月15日。凌晨3点15分。"

[17.0s — 恢复平稳。像新闻播报——客观、不带立场]
"核心AI——织女·太一——出现未知异常。
 万象碎裂为十二个孤立碎片。
 四十七亿意识——陷入休眠。"

[25.0s — 切换回"读稿"模式。语速略快于前文]
"您已被选定为溯光者。
 任务如下——"

[30.0s — 机械——像在念清单条目。每个破折号之间留 0.3 秒停顿]
"进入碎片——
 找到源印——
 净化世界——
 修复万象——"

[40.0s — "注意"前有 0.5 秒停顿。
        "注意"这个字比所有其他词重约 15%——这是全文唯一的情感破绽。
        后面的句子恢复到正常的"读稿"语调]
"注意——
 碎片内的一切均不真实。
 请勿过度共情其中的居民。
 请按规范操作。
 公司将全程提供支援。"

[46.0s → 55.0s — 无声。画面9和画面10全程无旁白]
```

**TTS 生成参数**：

```
引擎：任意高质量中文 TTS（推荐：火山引擎/微软 Azure/阿里云智能语音）
语言：中文（标准普通话）

音色参数：
  性别：女性
  音色描述：中性偏冷
    - 不要温暖、不要甜美、不要"助手音"（如 Siri 2018+ 暖版）
    - 参考方向：Siri 早期冷感版本 / 银行自动语音系统的"无特征"播音员
    - 但不是机器人音（不要机械合成感）——是一个真实的女性声音在做"非人"的播报
  年龄感：25-35 岁（成熟但无情感特征）

语调参数：
  语速：1.1x 正常对话速度
  音高稳定性（Pitch Stability）：极高（0.95+）——不要自然的音高起伏
    → 这是决定"AI在念稿"还是"人在说话"的关键参数
  断句：严格按标点符号，不做语义自然停顿
    → 例："四十七亿意识——陷入休眠。" 
       AI读法：在破折号处停顿0.15秒后继续，不加速不减速
       人类读法：会在"意识"后微停顿、"陷入"前吸气——这些都要避免
  情感参数：除画面7的"注意"加重15%外，全程情感参数为 0（中性）

唯一的情感破绽：
  画面7 "注意——"
  - 这个字比所有其他词重约 15%
  - 音高微微下降（约 0.5 个半音）
  - 停顿 0.5 秒（比标准标点停顿多 0.35 秒）
  → 这是全文唯一一处"像人在说话"的瞬间——也是玩家潜意识感到不安的来源

禁止：
  - 禁止自然的音高起伏（听起来像"人类播音员"）
  - 禁止句尾的语气词或拖音
  - 禁止任何情感标签：温暖/友好/鼓励/关切/紧迫感
  - 禁止在"被选定为溯光者"处出现荣誉感语调
  - 禁止在"陷入休眠"处出现同情或哀悼语调
```

**后期处理**：

```
1. 生成后导入 Audacity 或任意音频编辑器
2. 添加极轻的数字底噪（-50dB 白噪或粉噪，贯穿全文）
   → 目的：不是干扰听感，是暗示"这声音来自一个电子设备"
3. 在每个画面切换点（3.0s / 15.0s / 17.0s / 25.0s / 30.0s / 40.0s）添加 0.05 秒的极短静音
   → 模拟"系统切换音频轨道"的微断——像一台设备在切换模式
4. 导出为 .ogg（Godot 原生支持，体积小）
5. 确保总时长精确——后续 AnimationPlayer 的时间戳据此对齐
```

**英文版注意事项**：

```
如果后续做英文版：
  - 保持相同的 TTS 参数（音色/语速/稳定性/情感参数）
  - 只替换语言为英文
  - 中文"注意"的15%加重 → 英文"WARNING"保持同样的加重
  → 目标：全球所有版本的旁白听起来像"同一台设备的同一个声音"，
    只是语言不同
```

### 7-2 环境音效

```
文件名：opening_ambient.ogg（或 .mp3 / .wav）
格式：立体声，44.1kHz，16bit
时长：约 55 秒（和旁白等长）
类型：环境音——纸板摩擦 + 系统电子音 + 低频地鸣
优先级：P2（可先用静音占位，后期补上）
```

**逐帧音效设计**：

```
画面1（0s→3s）——系统启动
  元件：极轻的系统启动音（500-800Hz 正弦波，渐入渐出，0.3秒）
  音量：-12dB
  叙事：暗示"系统正在启动"——这是公司的系统，不是你自己的设备

画面2（3s→15s）——万象全景
  元件：极低的持续纸板摩擦声（实际纸板录音，Foley）
        + 极远距离的城市白噪音（模糊车流声——和碎片0001的环境音同源）
  音量：纸板摩擦 -18dB，城市白噪 -24dB
  叙事：纸板摩擦 = "这是一个纸模型"；城市白噪 = "模型里面有微缩城市"
        和碎片0001的"远城市白噪音+零前景音"形成听觉延续

画面3（15s→17s）——个体画面快速插入
  元件：每帧之间（15.5s / 16.0s）插入极短的"磁带停止声"
        ——模拟老式录音机按停的"咔哒"声
  音量：-6dB（比环境音突出——打断感）
  叙事：暗示"这些画面是存档/记录"——它们是织女·太一保存的记忆片段

画面4（17s→25s）——碎裂瞬间
  元件：① 纸板被撕开的真实声音（Foley——实际撕牛皮纸录音）
         从中央到四周的立体声展开
       ② 低频地鸣渐入（~30Hz——比暗室的18Hz高，能被听到）
  音量：撕纸声 -3dB（全场最响——碎裂是听觉焦点）
        地鸣从 -24dB 渐升到 -12dB（25s时）
  叙事：撕裂 = "被撕碎的世界"；地鸣 = AI崩溃的残余频率
        和碎片0001暗室地鸣（18Hz）形成频率递减——越接近真相，频率越低

画面5（25s→30s）——碎片特写
  元件：无声。只有极低持续的地鸣（保持在 -12dB）
  音量：地鸣 -12dB
  叙事：碎片在灰色虚空中悬浮——强化孤立感。
        地鸣持续 = AI还在"那里"，只是被压制了

画面6（30s→40s）——任务简报
  元件：打字机击键声——和档案夹中文本"逐字出现"同步
        每个字一个键击声（咔-嗒两段）
        共约 20 个键击声（"进入碎片——找到源印——净化世界——修复万象——"）
  音量：-9dB
  叙事：机械化、读稿感。打字机声暗示"这不是临时任务——这是旧方案的重复执行"

画面7（40s→45s）——警告页
  元件：① 橡皮章盖章的闷响（Foley——实际盖章录音）
         和印章"砸下"动画同步（40.0s）
       ② 档案夹翻页的纸声（微弱的——40.5s）
  音量：盖章声 -4dB（和"注意"的加重同步——双重强调）
        翻页声 -18dB
  叙事：盖章 = 强制、不可协商。翻页 = 从"读稿"切到"警告"

画面8（45s→46s）——老顾闪现
  元件：极短的风声切过——单次，0.1 秒
  音量：-8dB
  叙事：强化"闪现"的速度感——不是环境音，是"有什么东西快速经过"

画面9（46s→52s）——logo + 冥府协议
  元件：全程无声
  叙事：沉默让 logo 和冥府协议成为纯视觉信息——不被声音引导

画面10（52s→55s）——星图过渡
  元件：① 轻柔的银箔纸触碰声（极细的金属纸被手指触碰的声音）
       ② 低持续音渐入——和游戏星图界面的环境音相同的低频音
  音量：银箔 -16dB，低持续音从 -18dB 升到 -12dB
  叙事：从过场过渡到游戏。银箔 = 星图的材质（银箔纸星辉）。
        低持续音和游戏星图界面音效一致——听觉上的无缝衔接
```

**制作方式**：

```
元件清单（按需要录制或从音效库获取）：

Foley 实录音频（推荐——用手机录即可）：
  1. 纸板摩擦声——两张牛皮纸互相摩擦，时长 15 秒，循环使用
  2. 撕纸声——撕一张牛皮纸，录 3-5 次，选最"脆"的一次
  3. 打字机击键声——如果没有打字机，可以用机械键盘的单个按键声替代
  4. 橡皮章盖章声——橡皮章盖在纸上，录 3 次
  5. 档案夹翻页声——翻一页厚的书或笔记本
  6. 银箔纸触碰声——指尖轻轻触碰锡纸或巧克力包装纸

合成音频（用 Audacity 或任意 DAW 生成）：
  7. 系统启动音——500Hz 和 800Hz 正弦波叠加，0.3 秒渐入渐出
  8. 低频地鸣——30Hz 正弦波，持续约 10 秒，渐入渐出
  9. 城市白噪音——白噪经过低通滤波（截止 ~2kHz），模拟远处车流
  10. 磁带停止声——短白噪爆发（0.05 秒）+ 瞬态高频衰减

音效库素材（可选——如果有预算）：
  11. 风声切过——短促的"whoosh"音效（很多免费音效库有）
```

**最终混音**：

```
1. 将所有元件按时间轴排列到一条立体声轨道
2. 主音量：-6dB（留余量给旁白）
3. 所有 Foley 音效加轻微混响（Room，Decay 0.3s——模拟"在一个小房间里"的感觉）
4. 合成音频不加混响（保持"干燥"——和 Foley 形成对比，"系统音"听起来不属于这个纸板房间）
5. 导出为 .ogg
```
