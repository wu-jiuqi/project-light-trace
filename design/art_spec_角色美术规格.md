# 碎片 #0762「颜色的葬礼」—— 角色美术资源规格

> **目标风格**：日式RPG像素风，俯视视角 32×32 px 角色规格
> **整体色调**：全部灰阶。仅极少数"颜色残留"处允许微弱色彩
> **动画规格**：每角色需准备 4方向（上下左右）× 3帧（idle动画）的精灵表
> **美术优先级**：P0 静态立绘 + 4方向1帧（行走用）；P1 4方向3帧idle；P2 觉醒后彩色版

---

## 玩家载体：「迷路的旅人」

### 基本信息
- **身份**：穿越灰雾抵达灰白镇的旅行者，受托送信到此
- **性别**：不限定（可通过服装保持中性）
- **年龄感**：约25-30岁，长途跋涉后的疲惫感
- **体型**：中等身材，微微驼背——走了太久的痕迹

### 灰阶版外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 中短发，有些凌乱——刚起床没梳。发色中灰。 | `#8A8A8A` |
| **面部** | 略显疲惫但眼神清醒。眉间有一道很浅的竖纹——长期皱眉的痕迹。 | 肤色 `#A8A8A8` |
| **上衣** | 灰白色旅行衬衫，领口微敞。袖口卷到手肘。衣领上有一圈褪色到几乎看不见的暗线——本来可能是另一种颜色的领饰。 | `#B0B0B0` 主体 / `#989898` 褶皱阴影 |
| **外套（可选）** | 搭在肩上的深灰旅行斗篷——不是穿着的，是披着的。斗篷下摆有磨损和泥渍。 | `#7A7A7A` / 磨损边缘 `#6A6A6A` |
| **裤子** | 深灰旅行长裤，膝盖处有补丁——方形补丁，针脚粗犷但不乱。 | `#808080` / 补丁 `#8E8E8E` |
| **鞋子** | 磨损的短靴，鞋底被磨得一边高一边低。鞋面上有干涸的泥渍。 | `#6E6E6E` |
| **包包** | 左肩斜挎一个灰棕色皮制信使包——里面装着那封信的位置。包带在衣服上压出了一道白色的印子。 | `#8A7A6A`（暖调灰棕） |

### 关键视觉标记
- **信使包**：左肩斜挎。包的一角露出一个信封的灰色角——这是玩家的唯一"视觉身份锚"。玩家在任何场景中都应该能通过这个包认出自己
- **手部磨损**：手背和指节有淡淡的茧痕——不是铁匠那种厚茧，是"长期握登山杖/地图"留下的薄茧
- **衣领暗线**：衣领上的褪色线是唯一的"颜色残余暗示"——在灰阶版中呈现为比周围深一阶的灰（`#9A9A9A`）。觉醒后恢复为淡青色。

### 姿态
- **idle**：微微低头，呼吸缓慢。偶有动作——抬起左手捏一下右肩（像在缓解肌肉酸痛），或者调整一下信使包的位置
- **行走**：步伐稳健但不快——没有急事。"反正已经走了这么久了，不差这几步"
- **站立（对话中）**：面对NPC时稍微侧身——不是不礼貌，是旅途中的习惯：随时准备继续走

### 彩色版（世界恢复色彩后）
| 部位 | 灰阶→彩色 |
|------|----------|
| 头发 | 灰 → 深棕色 `#5A4030` |
| 上衣 | 灰白 → 米白色 `#F0EDE4` |
| 外套/斗篷 | 深灰 → 深墨绿 `#3A4A3A` |
| 裤子 | 灰 → 灰棕色 `#7A6A5A` |
| 衣领暗线 | 深灰 → 淡青色 `#8AB8C8`（原来是青色的领饰） |
| 信使包 | 灰棕 → 深棕皮革 `#6A4030` |

## NPC #1：铁匠·老霍（红色·愤怒）

### 基本信息
- **年龄**：约52岁
- **体型**：粗壮，肩宽腰圆，典型的长期体力劳动者体格。身高在中上——不矮，但因为常低头俯身锻造，整体高度感被压低
- **职业**：铁匠

### 灰阶版外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 灰色短发，两鬓剃得很短，头顶稍长。发色不是自然灰——更像是煤灰染的。 | `#929292` |
| **面部** | 方脸，下颌宽。眉毛粗重，鼻梁上有一道横向的旧疤——不是刀伤，是铁渣溅的。嘴型永远是向下的——不是不高兴，是习惯。 | 肤色 `#A0A0A0` / 疤痕 `#8A8A8A` |
| **上衣** | 无袖的灰色厚布背心，领口被汗渍浸出一圈深灰。露出两条粗壮的手臂——前臂上满是老茧。 | `#8E8E8E` / 汗渍圈 `#7A7A7A` |
| **围裙** | **核心视觉锚点**。帆布制铁匠围裙，覆盖从胸口到膝盖的范围。围裙正面散布着灰色铁屑粉末和随机烫痕。**右下角有一块深酒红色的污渍**——在黑白世界中呈现出异样的深色——这是唯一能在初始状态看到的颜色残留。 | `#949494` 主体 / 红色污渍 `#6A4040` |
| **裤子** | 深灰色粗布长裤，膝盖处有厚布补丁。裤脚塞进靴筒里。 | `#7E7E7E` |
| **靴子** | 厚底铁匠靴，靴头有铁片加固——已经磨得发亮。 | `#686868` / 铁片 `#AAAAAA` |

### 颜色残留
- **围裙上的铁水烫痕**：位于围裙右下角，约 4×6 px 的深酒红色斑块。在灰阶世界中，这块污渍的颜色比其他灰色区域"更深一层"——并不是明显的红色，而是"让人觉得不对劲的深灰色"。当玩家第一次注意到它时，如果盯着看超过 3 秒，屏幕会微微出现一条提示："这块污渍……似乎不是灰色的。"

### 姿态
- **idle**：右手举锤在空中半停——锤子永远不上不下。左手扶在铁砧上。每隔 5-8 秒，锤子落下一次（敲在铁胚上），然后回到原位
- **行走**：极少行走。如果出门，步伐沉重、每一步落地有声
- **觉醒后**：锤击频率明显加快。不再停留在半空中——一锤接一锤

### 彩色版（红色觉醒后）
| 部位 | 灰阶→彩色 |
|------|----------|
| 头发 | 灰 → 深灰夹白 `#686868` / `#AEAEAE` |
| 上衣背心 | 灰 → 深麻灰褐 `#7A6A5A` |
| 围裙 | 灰 → 焦茶色 `#8A7A5A` / 铁水烫痕→鲜红 `#C04040` |
| 锻造台火焰 | 无色灰 → 橙红烈焰 `#E06030` + 橙黄高光 `#F0A040` |

## NPC #2：花店老板娘·阿莲（蓝色·悲伤）

### 基本信息
- **年龄**：约35岁
- **体型**：中等偏纤细。站姿永远端正——但端正得像个假人
- **职业**：花店店主

### 灰阶版外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 浅灰色长发，松松地挽成一个低髻，用一根灰白色的发簪固定。额前有两缕碎发——刚好遮住眼角。 | `#B4B4B4` / 发簪 `#D0D0D0` |
| **面部** | 五官精致但缺乏活力。嘴角永远有微笑——但微笑只在嘴唇上，到不了眼睛。眼睛下方有极淡的阴影——不是黑眼圈，是"什么都没有"的空洞。 | 肤色 `#AEAEAE` / 阴影 `#9A9A9A` |
| **长裙** | 及踝的灰白色连衣裙，领口有简单的交叉设计。裙摆上有一排非常浅的刺绣——矢车菊的轮廓，但线已经磨到几乎看不见了。 | `#B8B8B8` / 褶皱 `#A4A4A4` / 绣线 `#ACACAC` |
| **围裙** | 腰间系着一条深灰色短围裙——花店工作用。围裙口袋里插着一把灰白的小剪刀。 | `#8E8E8E` |
| **手** | 修长的手指，指甲剪得很整齐。右手掌心有一道很浅的绿灰色污渍——长期握花茎擦出来的。 | 肤色同面部 |

### 姿态
- **idle**：站在花架前，手里拿着一把灰色矢车菊，正在用剪刀修剪花茎。动作缓慢而精确——每一根花茎都剪同样的长度
- **行走**：步伐轻盈无声。走到下一个花架，拿起下一盆——继续修剪
- **送花时**：如果有人站在她面前超过 5 秒，她会从手边拿起一束包好的矢车菊，双手递出——头微微低着
- **觉醒后**：不再修剪花茎。她站在花店门口，看着街道——流泪

### 彩色版（蓝色觉醒后）
| 部位 | 灰阶→彩色 |
|------|----------|
| 头发 | 浅灰 → 浅栗色 `#A08070` |
| 长裙 | 灰白 → 淡蓝灰色 `#B8C4D4` |
| 围裙 | 深灰 → 亚麻原色 `#B8A890` |
| 矢车菊 | 灰色 → 蓝色 `#6090C0` |
| 眼泪 | 无色 → 淡蓝色光点 `#80B0E0` |

## NPC #3：面包师·老唐（黄色·希望）

### 基本信息
- **年龄**：约47岁
- **体型**：圆脸，壮实但并不胖——是"揉面揉出来的"厚度。身高偏矮
- **职业**：面包师

### 灰阶版外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 灰色短发，微卷，发际线后移（额头很宽）。头顶有一撮总是竖起来的头发——被面粉粘住了。 | `#9A9A9A` / 面粉灰白 `#C8C8C8` |
| **面部** | 圆脸，红光满面——虽然是灰阶，但他是所有NPC中脸部最"亮"的（`#B4B4B4`）。眼睛比正常人大一像素——不是画错，是他的眼睛真的有光。眉毛稀疏但永远向上挑着——永远在笑。 | 肤色 `#B4B4B4` / 眉毛 `#8A8A8A` |
| **上衣** | 灰白色的厨师上衣，领口有一圈浅灰的滚边。袖子卷到肘上——前臂上有面粉的灰白痕迹。 | `#BCBCBC` / 滚边 `#A8A8A8` |
| **围裙** | 灰白色的帆布围裙，长度到膝盖。围裙正面永远有新面粉的灰白掌印——他擦手从来只擦在围裙上。围裙左下角绣着一个小小的麦穗图案——已经洗到褪色。 | `#B0B0B0` / 面粉印 `#D0D0D0` |
| **裤子** | 浅灰布裤，膝盖处被烤炉烤得微微变深。 | `#A0A0A0` |
| **鞋** | 灰色布面平底鞋——在面包房里穿的。鞋面有零星的灰色面团渍。 | `#909090` |

### 关键视觉标记
- **眼中的光**：他的眼睛是 2 px 而非其他 NPC 的 1 px——在像素美术中，这 1 px 的差距就是"有希望"和"没有希望"的区别。这不是颜色——这是亮度。他的眼睛 `#D0D0D0`，比其他人的 `#A0A0A0` 亮两个色阶
- **围裙上的面粉手印**：这是他最具标志性的视觉元素。新印子覆盖旧印子——密密麻麻
- **手中永远有面团**：idle 动画中，他右手一直在揉一个灰色面团——揉、翻、压、再揉

### 姿态
- **idle**：半侧身站在烤炉前，右手反复揉捏一团面团。每揉三次——转身、打开烤炉门看看（摇头叹气）、关上、继续揉
- **行走**：快步——但不是急，是"总有东西要烤"的那种快。走到哪里手里都端着一个烤盘
- **送面包时**：弯腰（他比大部分人都矮），双手递上面团——"来！不热——但是能吃！"
- **觉醒后**：站在烤炉前，看着金黄色的面包，不再揉面。手里端着面包——不送人，只是看着。

### 彩色版（黄色觉醒后）
| 部位 | 灰阶→彩色 |
|------|----------|
| 头发 + 面粉 | 灰+灰白 → 深棕 `#5A4030` + 乳白面粉 `#F8F0E0` |
| 上衣 | 灰白 → 米白色 `#F0EDE0` |
| 围裙 + 麦穗刺绣 | 灰白 → 亚麻色 `#C8B890` / 麦穗→金黄色 `#D8B040` |
| 面包 | 灰色 → 金黄色 `#E0B040` / 面包皮→深棕金 `#B08030` |
| 烤炉火焰 | 无色灰 → 暖黄光 `#F0D060` 从炉门透出 |

## NPC #4：守墓人·老崔（绿色·恐惧）

### 基本信息
- **年龄**：约55岁
- **体型**：干瘦——瘦得几乎能看到每一根手指骨的轮廓。驼背极为明显——不是因为老了，是因为"缩"。他把自己缩成了一个球。
- **职业**：守墓人

### 灰阶版外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 灰白乱发，大片斑秃。胡茬也是灰白色的，稀稀拉拉，显然很久没打理。 | `#A8A8A8` / 斑秃 `#B4B4B4`（头皮） |
| **面部** | 凹陷的双颊，颧骨突出。眼睛下方有很深的灰黑色阴影——不是熬夜，是恐惧。**最重要的是他的眼睛**——瞳孔深处有一抹极淡的绿色反光（`#4A5A4A`），在灰阶世界中几乎看不出来，但如果玩家近距离站在他面前（对话状态），会注意到他的眼睛"比灰色深一点点" | 肤色 `#9A9A9A` / 阴影 `#7A7A7A` / 眼瞳绿 `#4A5A4A` |
| **上衣** | 深灰色破旧大衣，已经看不出原来的样子。衣领竖起来裹住脖子——每时每刻都竖着。大衣上有多处缝补——针脚很乱，是他自己缝的。 | `#6E6E6E` / 补丁 `#7A7A7A` / `#828282` |
| **裤子** | 深灰布裤，膝盖处被磨得发亮——他长时间蹲在屋角 | `#707070` |
| **手** | 手指细长干瘦，指节突出。左手无名指上戴着一个灰白色的戒指——已经变形了。右手始终握着一把生锈的铁钥匙——那是墓园门的钥匙。从来没有松开过。 | 肤色同面部 / 戒指 `#909090` / 钥匙 `#5A5A5A` |

### 颜色残留
- **眼瞳深处的绿光**：极其微弱。在正常 gameplay 视距下看起来只是"比灰色深一阶的点"。只在对话特写中可辨认。当 4 种颜色被找回后（绿色觉醒前），这个绿光会开始闪烁——频率约 2 秒一次

### 姿态
- **idle**：蹲在屋角，双腿蜷起，双臂抱住膝盖。眼睛从两腿之间看出去。这是他唯一的安全姿势。
- **门缝说话时**：蹲在门后面，从门缝向外看——只有一只眼睛和半边嘴露在缝隙中
- **行走**：几乎不走路。如果出门（仅在觉醒后），走路像在逃——快、轻、贴着墙
- **手部动作**：他的右手永远在摩挲那把铁钥匙——不是在玩，是确认它还在
- **觉醒后**：站起来了。不再蹲着。但身体的习惯性抽搐仍然存在——肩膀每隔几秒就微微一缩

### 彩色版（绿色觉醒后）
| 部位 | 灰阶→彩色 |
|------|----------|
| 头发 | 灰白 → 花白（本来的颜色） `#8A8A8A` / 白色 `#D0D0D0` |
| 瞳孔绿光 | 极淡灰绿 `#4A5A4A` → 明显绿色荧光 `#40A040`（带微弱发光效果） |
| 大衣 | 深灰 → 墨绿色破旧大衣 `#3A4A3A`（原来就是绿色的——但在灰阶里看不出来） |
| 皮肤 | 灰 → 苍白（长期不见光） `#D8D0C8` |

## NPC #5：小提琴手·薇拉（紫色·思念）

### 基本信息
- **年龄**：约28岁
- **体型**：纤细，肩窄。站姿极度端正——但是"被框住"的那种端正。脖子微微伸长——像在听什么很遥远的声音
- **职业**：无

### 灰阶版外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 深灰色长发，一丝不苟地盘成一个高髻。没有任何碎发。发簪是一根细长的灰白色骨簪。 | `#7A7A7A` / 骨簪 `#C0C0C0` |
| **面部** | 瓜子脸，下巴很尖。眉毛细长，永远微微蹙着——不是生气，是"听"。眼神不看向任何人——永远望向远方（东偏北，画室的方向）。嘴唇自然闭合，没有任何弧度——不笑也不悲，是"等"的表情。 | 肤色 `#AAAAAA` |
| **连衣裙** | **核心视觉锚点**。及膝的深灰色长裙，袖子到手腕。裙摆整齐不皱——她极度在意自己的仪容。**裙摆右下角有一条几乎看不见的暗纹**：那是曾经紫色的刺绣——在灰阶世界中呈现出和裙子布料之间极细微的色阶差（`#7E7E8A` vs `#7A7A7A`）。只有在她侧身或光线角度特殊时才能注意到。 | 裙子 `#7A7A7A` / 暗纹 `#7E7E8A` |
| **小提琴** | 她每天架在左肩上的灰棕色小提琴。琴身有光泽——她每天擦。琴弦是四根极细的灰白色线——不仔细看会以为没有弦。琴弓搭在弦上方 2px 的位置——永远不落下。 | 琴身 `#8A7A6A` / 琴弦 `#C8C8C8` / 琴弓 `#9A8A7A` |
| **手指** | 左手手指上有一道很浅的压痕——长期按琴弦留下的。右手的小指微微向外翘——那是持弓的惯性姿势 | 肤色同面部 |
| **鞋** | 深灰色平底鞋，鞋头圆圆的。鞋面上有细小的磨损——她在广场上站立太久磨出来的 | `#787878` |

### 颜色残留
- **裙摆暗纹**：极淡的紫灰色刺绣，纹样是螺旋状——像音符在空中旋转的轨迹。在灰阶中，这条纹的颜色 `#7E7E8A` 和裙子 `#7A7A7A` 之间只有 4 个数字的色差。是"盯久了才会觉得那里和别处不一样"的区别

### 姿态
- **idle**：站立，左肩架琴，下巴微微压着腮托。右手持弓，弓停在弦上方 2px——静止。整个身体只有一个地方在动：左手的手指——在琴弦上无声地、极其缓慢地变换按弦位置。她在"无声地演奏"
- **行走**：只在来广场和离开广场时移动。步态像在走一条直线——不是真的有人画了线，是她在自己的轨迹里不愿意偏差一步
- **觉醒后**：第一个完整拉完一曲的姿势——头微侧，眼睛终于看向琴——而不是远方。右手拉弓的动作流畅而缓慢

### 彩色版（紫色觉醒后）
| 部位 | 灰阶→彩色 |
|------|----------|
| 头发 | 深灰 → 深棕色 `#4A3028` |
| 裙子 | 深灰 → 深紫灰色 `#6A5878` |
| 裙摆暗纹 | 紫灰 `#7E7E8A` → 亮紫色刺绣 `#9060C0`（螺旋音符纹） |
| 小提琴 | 灰棕 → 红棕色 `#8A5040` |
| 琴弦光点 | 无色 → 淡紫色光点从琴弦飘出 `#B080E0` |
| 眼睛 | 灰色 → 深紫灰 `#584868` |

## 特殊NPC：老画家（源印载体）

### 基本信息
- **年龄**：不可判断——面容苍老但眼神是年轻人的。可能70岁，也可能不存在"年龄"
- **体型**：极瘦。瘦到风能把他吹走——所以老唐给他送面包。
- **身份**：AI「织女·太一」在崩溃前留下的最后一段完整自我意识投射。他不是NPC——他是这个碎片唯一"清醒"的存在。

### 灰阶版外观

| 部位 | 描述 | 色值 |
|------|------|------|
| **头发** | 花白长发，散乱地披在肩上。刘海遮住了半边脸——不是造型，是懒得剪。发梢上沾着灰色颜料。 | `#B8B8B8` 白 / `#7A7A7A` 灰 / 颜料 `#989898` |
| **面部** | 颧骨极高，两颊深陷。但他有整个碎片中**最亮的眼睛**——不是有光，是"知道"。他的眼神让其他NPC害怕——因为被看穿的感觉。眼角有深深的笑纹——但他已经不笑了。嘴唇上总有灰色颜料。 | 肤色 `#A4A4A4` / 眼睛 `#D0D0D0`（和老唐一样亮，但性质不同——老唐是希望的亮，他是"看穿了一切"的亮） |
| **上衣** | 一件曾经可能是白色的长罩衫——现在全是灰白颜料的溅痕和手指擦过的痕迹。袖子长的盖住了手背。领口歪在一边——露出锁骨。罩衫下摆有一行淡淡的字迹——他用颜料写的，但已经被洗掉了大半，只能看到:"……不是……灰色……" | `#B0B0B0` / 颜料溅痕 `#C0C0C0` / `#9A9A9A` / `#8A8A8A` |
| **裤子** | 深灰布裤，松松垮垮。右边裤腿的膝盖处有一个破洞——跪在地上作画磨的 | `#7E7E7E` |
| **手** | 十根手指上全是干涸的灰白颜料。指甲缝里也是。右手食指和中指的侧面有厚厚的茧——调色盘握得太久了。**零时之前，这只手握着的是彩色颜料——现在全白了** | 手背面 `#A8A8A8` / 颜料 `#C8C8C8` |

### 关键视觉标记
- **手上的颜料**：他的手指上覆盖着多层干涸颜料——不同深浅的灰色层层叠叠。这是他在零时之后反复尝试"调出颜色"的痕迹。所有尝试都失败了——但颜料越叠越厚

### 姿态
- **idle**：坐在画架前，右手举着调色盘，左手悬在画布上方——永远差1px就落笔。他在犹豫什么？还是在等什么？这个姿势是他对玩家说的第一句无声的话
- **转身时**：动作极慢——像是每个动作都在确认自己还活着
- **揭幕自画像**（六色觉醒后）：双手缓缓拉开灰布——然后退后一步。再退一步。然后站定。这是他第一次站直。
- **消失**（源印提取后）：从脚开始逐渐透明化——不是变白，是"变淡"。最后一瞬间，他会看着玩家——眼角有蓝色的泪光。然后什么都没有了

### 彩色版（六色恢复后——仅在终局可见）
| 部位 | 灰阶→彩色 |
|------|----------|
| 头发 | 花白 → 纯白 `#E8E8E8` + 深灰 `#585858` |
| 罩衫 | 灰白 → 纯白 `#F0F0F0`，上面的颜料溅痕恢复为彩色——红蓝黄绿紫橙的斑点，像是被打翻的调色盘 |
| 手指颜料 | 灰 → 六种颜色层层叠叠——每根手指不同颜色 |
| 眼睛 | 灰亮 → 六色微光在瞳孔中旋转 `#C04040` `#6090C0` `#E0B040` `#40A040` `#9060C0` `#E08030` |
| 自画像（揭幕后） | 全彩——老画家自己的脸，瞳孔中六色旋涡 = 源印符号 |

### 角色色板统一灰度

```
皮肤浅色  #B4B4B4   皮肤中色  #A8A8A8   皮肤深色  #989898
衣物浅色  #C0C0C0   衣物中色  #909090   衣物深色  #707070
阴影最暗  #484848   高光最亮  #D0D0D0   纯白用途  #F0F0F0（仅雕像）
```

### 颜色残留的可见性设计

所有"颜色残留"遵循一个原则：**在正常 gameplay 视距下，颜色残留是不可见的。玩家需要靠近或触发特定条件才能注意到。**

| 残留点 | 可见触发条件 | 灰阶色值 vs 周围色值的差异 |
|--------|------------|--------------------------|
| 铁匠围裙铁水烫痕 | 距离老霍 < 2 tiles | `#6A4040` vs 周围 `#949494`（差异仅色调，明度相近） |
| 花店后巷干枯矢车菊 | 距离矢车菊 < 1 tile | `#5A6A8A` vs 周围泥色 `#6A6A6A`（轻微的蓝色偏） |
| 老崔瞳孔绿光 | 对话特写状态 | `#4A5A4A` vs 瞳孔其它部分 `#484848`（几乎一样，仅1-2像素差） |
| 薇拉裙摆紫色暗纹 | 她侧身或光线特殊角度 | `#7E7E8A` vs 裙子 `#7A7A7A`（极细微的蓝紫偏） |
