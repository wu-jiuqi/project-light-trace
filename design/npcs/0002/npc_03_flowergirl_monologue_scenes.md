### S05 · 花不逃

> **景别**：大特写 —— 拇指 & 花瓣
> **情感基调**：花的哲学——不是对抗命运，而是把全部力量注入命运之中

> **独白对应**："我一直在走——不是因为下一站花好卖。是因为——我怕这一站没有风。你注意到了吗——这个车站没有风。花靠风授粉——没有风，花怎么开？我不知道。但它们开了。不靠风——不靠时间——靠自己的意志。花能做到的事——我不敢做。我不敢停在没风的地方——等自己开。"
> **景别**：中景·环境强调
> **情感基调**：恐惧与羡慕并存——承认自己的不敢，同时见证花的"敢"

**画面描述**：中景。卖花女站在黄昏车站的中央——不是蹲着，她站了起来，但还没有离开篮子。画面极力表现"无风"——纸灯笼完全静止，每一条纸穗都笔直垂下；纸雕站台立柱的层叠纸边没有任何掀起的痕迹；她衣着的纸折边角纹丝不动。静止是绝对的。但她手中的矢车菊——在完全无风的车站中——蓝得正盛。花瓣舒展，蓝色饱满。她低头看着手中的花——表情不是恐惧，是困惑的敬畏。"花不靠风也开了"——而她不敢停在无风的地方。她身后的铁轨伸向暮色深处——那是她一直"走"的方向，也是她不敢"停"的地方。

## 视觉一致性约束

为确保12个场景的视觉连贯性，所有提示词遵循以下统一约束：

| 约束项 | 规范 |
|--------|------|
| **风格** | 纸雕暗黑童话（Papercraft Dark Fairy Tale）—— 平面分层、手工质感、故事书插图感 |
| **禁止项** | 透视深度、真实阴影、体积光、3D感、雕塑感、玩具感、照片感、微缩模型感 |
| **色彩基调** | 矢车菊蓝（多深度层次） + 暖琥珀金（暮光） + 深暮紫蓝（暗影） + 米白泛黄（纸张/皮肤） + 铜色（水壶/道具） |
| **角色视觉** | 纤细木偶比例、小手小脚、风格化剪影、面孔由纸雕组件拼合、靛蓝灰旅行布衣、纸折补丁 |
| **核心道具** | 柳编花篮（纸藤交叉编织纹理）、铜质喷水壶（凹陷+暗斑）、矢车菊花朵（多层蓝色描图纸叠加） |
| **环境视觉** | 建筑由层叠风化木纸板构成、纸灯笼（撕纸灯罩）、可见缝线与纸折装饰 |
| **纸张质感** | 所有表面必须呈现纸张肌理——纤维纹理、毛边、折痕、缝线痕迹、描图纸半透明层叠 |
| **灯光处理** | 暮光以琥珀色纸层叠加表现、暗影以深紫色纸层叠加表现——禁止体积光与投影 |
| **色彩限制** | 每场景严格控制在5-6色内——禁止全色域 |
| **面部表现** | 不使用写实五官——眼睛由纸条切割构成、嘴唇由纸折表现、表情通过纸片的微小倾斜和纸折角度传达 |

## 使用建议

1. **生成顺序**：建议从 S01（核心环境 + 角色基准）和 S04/S07（核心道具特写：花）开始，先确定视觉基准。
2. **角色一致性**：卖花女的视觉形象在所有含角色的场景（S01/S02/S06/S08/S09/S10/S11/S12）中必须保持一致——S01 作为角色基准图，后续含角色的场景以 S01 的角色形象为参考。
3. **色彩校准**：S01 建立了"黄昏站台环境 + 篮子 + 水壶"的环境色彩基准；S04 建立了"矢车菊·最深蓝"的道具色彩基准。生成其他场景前应先确认这两个场景的色彩方向。
4. **道具追踪**：铜质水壶在 S01/S02 中出现（完整），S10/S11 中再次出现（花被取走后水壶仍在原地）——需保持一致。篮子从 S01 的"满"到 S10 的"空了大半"到 S11 的"空篮"——注意花量的递减。
5. **画面尺寸**：建议全部使用 `1280x896`（横版）以适配游戏内 16:9 显示比例；S07/S08（双构图及特写）如需竖向可改为 `1024x1536`。
6. **情感曲线**：S01-S02 铺垫（日常·推销） → S03-S05 深化（执念·花的哲学） → S06-S09 转折（漂泊·残缺·恐惧·自我认知） → S10-S12 收束（交付·释然·守护）。视觉节奏（景别从小到大再拉远）应与情感曲线同步。
