## 二、阶段一：发现期（DISCOVER）— 0～5分钟

### 2.1 玩家状态
- **知识水平**：对游戏一无所知，可能通过Steam/社交媒体/朋友推荐而来
- **情感状态**：好奇、轻度期待、怀疑（"又一个独立游戏？"）
- **核心问题**："这是什么游戏？值得我花时间吗？"

### 2.2 接触路径

| 路径 | 触发方式 | 首次印象构建 |
|------|---------|------------|
| Steam商店页 | 浏览/搜索 | 纸雕暗黑童话视觉 + "潜入碎裂AI宇宙"一句话简介 |
| 朋友分享的截图 | 社交传播 | 星图界面的玻璃碎片视觉冲击 |
| 社区讨论 | Reddit/B站/贴吧 | "半解密提示系统"的可讨论性引发好奇 |
| 视频/直播 | 内容创作者 | 某个碎片的完整体验展示 |

### 2.3 关键决策点 #1：是否下载/打开游戏

**促进因素**：
- 独特的纸艺视觉风格（与其他像素游戏完全不同）
- "AI碎裂宇宙"的世界观新颖性
- 叙事驱动的谜题（非数值成长、非战斗）

**阻碍因素**：
- Godot引擎可能导致的Web加载等待
- "又一个人独立游戏"的刻板印象
- 缺乏知名IP背书

### 2.4 情绪曲线
```
兴趣 ┤        ╭─ 看到星图界面
    ┤       ╱
    ┤      ╱
    ┤  ═══╪═══ 浏览商店页
    ┤    ╱
    ┤   ╱
    ┤──╯
    └────────────────────→ 时间
   0min              5min
```

## 三、阶段二：入门期（ONBOARD）— 5～25分钟 ★核心空白

### 3.1 玩家状态
- **知识水平**：刚下载/打开游戏，不熟悉操作和世界观
- **情感状态**：期待+紧张+轻微困惑
- **核心问题**："我要做什么？怎么操作？这个世界是怎么回事？"

### 3.2 当前缺陷（需要FTUE设计的根本原因）

| 缺口 | 现状 | 玩家感受 | 严重度 |
|------|------|---------|--------|
| 无开场叙事 | 标题画面直接进入，无世界观导入 | "这是谁？我在哪？为什么？" | ⚠️⚠️⚠️⚠️⚠️ |
| 无操作引导 | WASD移动无提示，E键交互无说明 | "我该怎么走？这个E是什么？" | ⚠️⚠️⚠️⚠️ |
| 无目标引导 | 星图界面后不知该做什么 | "这些碎片是什么？该点哪个？" | ⚠️⚠️⚠️⚠️ |
| 无UI介绍 | 背包/存档/暂停等系统无引导 | "B键为什么弹出一个框？" | ⚠️⚠️⚠️ |
| 无情感锚点 | 缺少让玩家"在意"的初始叙事 | "我不认识这些NPC，为什么要帮他们？" | ⚠️⚠️⚠️⚠️⚠️ |

### 3.3 理想的FTUE流程（详见 `ftue_design.md`）

```
标题画面 → 开场叙事动画(30s) → 星图引导(碎片0001强制首发)
  → 晨曦镇教学关卡 → 首次对话 → 首次解密 → 首个源印
  → 返回星图(完成感+解锁0002/0003) → 自由探索
```

### 3.4 情绪曲线
```
        ╭─ 看到开场动画
期待 ┤  │  ╭─ 学会移动
    ┤  │  │  ╭─ 第一次对话
    ┤──╯  │  │  ╭─ 解开第一个谜
    ┤     │  │  │  ╭─ 发现便笺(暗线A)
    ┤     │  │  │  │
    ┤     ╰──╯  │  │
    ┤           ╰──╯── 通关教学关卡
    └──────────────────────────→ 时间
   5min    10min   15min   20min
```

## 四、阶段三：沉浸期（IMMERSE）— 2～8小时

### 4.1 玩家状态
- **知识水平**：掌握了核心操作，理解了"碎片→探索→对话→源印"循环
- **情感状态**：沉浸、好奇、逐渐建立世界观认知
- **核心问题**："下一个碎片是什么？这个世界还有什么秘密？"

### 4.2 核心循环（当前已实现）

```
┌──────────────────────────────────────────┐
│              核心游戏循环                   │
│                                          │
│  星图 ──→ 选择碎片 ──→ 查看提示           │
│   ↑                        ↓             │
│   │                  进入碎片世界          │
│   │                        ↓             │
│   │              WASD探索 + 收集物品       │
│   │                        ↓             │
│   │              E键对话 NPC                │
│   │                        ↓             │
│   │              触发颜色觉醒/解密         │
│   │                        ↓             │
│   │              发现源印 → E键解码        │
│   │                        ↓             │
│   └──── 返回星图 ←── 碎片修复归位          │
│           ↑                              │
│    修复进度+8.3%（1/12）                   │
└──────────────────────────────────────────┘
```

### 4.3 碎片推进建议顺序

| 顺序 | 碎片ID | 名称 | 设计目的 | 情绪峰值 |
|------|--------|------|---------|---------|
| 第1 | 0001 | 启程之镇 | 教学：学会观察推理 | "这个镇子是假的" |
| 第2-3 | 0002/0003 | 黄昏驿站/月下神社 | 巩固：不同风格谜题 | 离别感伤 / 仪式美感 |
| 第4 | 0004 | 工坊物语 | 拓展：NPC关系网 | 暗线A种子确认 |
| 第5 | 0762 | 颜色的葬礼 | 情感高潮：六色觉醒 | "AI不是失控"的震撼 |
| 第6-7 | 0915/1138 | 遗忘庭院/时钟车站 | 暗线推进：暗线B浮现 | 对公司产生怀疑 |

### 4.5 玩家可能遇到的困惑点

| 困惑 | 触发条件 | 建议的引导方式 |
|------|---------|-------------|
| "碎片太难了，卡关了" | 解密无法推进 | 半解密提示系统自动推送L2/L3提示 |
| "NPC不理我了" | 警觉值过高导致SHUTDOWN | NPC在封闭时说"明天再来吧"暗示时间重置 |
| "暗线是什么？" | 收集了暗线种子但不理解 | 在星图的"源印日志"中提供信息关联视图 |
| "我应该先玩哪个碎片？" | 面对多个可选碎片 | 星图上高亮推荐碎片（难度+叙事重要性） |

## 五、阶段四：转折期（PIVOT）— 8～12小时

### 5.1 玩家状态
- **知识水平**：完成了6-8个碎片，暗线A/B部分揭示
- **情感状态**：认知被颠覆、开始怀疑公司叙事
- **核心问题**："公司在骗我吗？AI到底是好是坏？"

### 5.2 关键事件序列

```
碎片#0047「倒悬图书馆」
  └→ 发现陈技术的便利贴 → 确认"提示不完整"是故意的
  └→ 发现冥府协议第一阶段文档

碎片#0915「遗忘庭院」
  └→ 冥府协议第二阶段证据
  └→ 公司信任度下降触发: 数据终端收到泄露报告
```

## 六、阶段五：社群期（COMMUNITY）— 12～18小时

### 6.1 玩家状态
- **知识水平**：暗线B明确，暗线C初现（"上一个你"的回响）
- **情感状态**：坚定、孤独→被支持（论坛解锁后）
- **核心问题**："其他溯光者经历了什么？我们能一起做什么？"

### 6.2 溯光者论坛解锁

```
碎片#2049「镜中人」通关
  └→ 触发条件: 修复进度 ≥ 66%(8/12)
  └→ 论坛解锁动画: 星图界面出现新的"论坛"入口
  └→ 首次进入论坛: 看到3-5条历史帖子

论坛核心机制:
  ┌─────────────────────────────────┐
  │  溯光者论坛 (In-Game)            │
  │                                 │
  │  📌 置顶: "溯光计划公告"         │
  │  🔥 热门: "冥府协议到底指什么?"   │
  │  💬 讨论: "碎片1138的车站密码"    │
  │  🆘 求助: "碎片3015卡关"         │
  │  🔒 加密: "给能看到的人——"      │
  │                                 │
  │  [发帖] [搜索] [私信]            │
  └─────────────────────────────────┘
```

### 6.3 论坛的叙事功能

| 帖子类型 | 叙事作用 | 实例 |
|---------|---------|------|
| 线索拼接 | 同半解密提示在不同世界不同显示，对比补全 | "我这边显示'白_'，有人显示别的吗?" |
| 暗线拼图 | 不同平行世界保留了不同的真相碎片 | 暗线B拼图：各世界找到的证据不同 |
| 失踪帖 | 某些用户突然不发了——暗示现实中被"处理" | "用户#2049-03 上次活跃: 3天前" |
| 日期异常 | 某些帖子的日期自己改变 | "发布于2077年3月14日" → "发布于2077年3月15日" |
| 神秘发帖者 | 内容不符合任何已知平行世界 | "不要在碎片3015里打开第三扇门" |

### 7.3 三种结局的情绪设计

| 结局 | 情感基调 | 视觉表达 | 最终画面 |
|------|---------|---------|---------|

| B: 永驻 | 忧郁·接受 | 万象静止在最后一刻，像是被琥珀封存 | 47亿人悬浮在永恒的中间态，无声 |
| C: 新生 | 希望·共创 | 新的万象从碎片中重新生长，前所未见的色彩 | AI与人类在新的宇宙中和平共存 |

## 八、阶段七：回响期（ECHO）— 22小时+

### 8.1 玩家状态
- **知识水平**：已知至少一种结局的完整故事
- **情感状态**：满足、感动、好奇（"另一个结局是什么？"）
- **核心行为**：回想、讨论、二刷

### 8.2 二刷驱动力

| 驱动力 | 机制 | 目标 |
|--------|------|------|
| 不同结局 | 终局选择影响结局 | 体验另外2种结局 |
| 碎片顺序不同 | 非线性碎片选择改变叙事体验 | 不同路线触发不同对话和线索组合 |
| 暗线完整度 | 第一次可能遗漏许多暗线线索 | 收集全部暗线内容 |
| NPC警觉路线 | 不同对话策略导致不同NPC反应 | 看"信任路线"vs"警觉路线"的差异 |
| 论坛新内容 | 新玩家加入论坛后看到不同帖子 | 不同平行世界的视角 |

### 8.3 社区留存

| 留存锚点 | 描述 |
|---------|------|
| 理论讨论 | "AI到底是对的？还是错的？"——没有标准答案的开放讨论 |
| 解密分享 | 半解密提示的不同解法分享 |
| 隐藏内容 | 0.2秒的光墙外景、林指导幕间对话等极低概率事件触发条件分享 |
| MOD支持 | (未来)允许玩家设计自定义碎片 |
