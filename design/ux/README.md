## 🎯 核心发现

### 当前UX三大缺口
1. **叙事空洞**：无世界观导入，无角色身份建立 → 解决方案：开场叙事动画
2. **操作黑洞**：无WASD/交互/背包/存档引导 → 解决方案：碎片#0001教学关卡
3. **目标迷失**：碎片选择无引导，难度无区分 → 解决方案：星图FTUE引导模式

### 最优先实现（P0）
1. 开场叙事动画（~45秒）
2. 碎片#0001完整教学关卡
3. 星图FTUE引导模式

### 预期效果
- 从"标题→首次通关"时间：<20分钟
- 首次通关后留存率目标：>60%
- "我懂了"时刻：日晷解密成功
- "哇"时刻：暗门开启、便笺发现

## 📐 设计原则

1. **叙事驱动引导**：不要弹出教程弹窗，让NPC在对话中自然教学
2. **发现优于告知**：背包、存档等系统让玩家自己发现
3. **失败是教学**：触碰光墙扣合规度——教规则，不是惩罚
4. **尊重自由**：不强制玩家选#0001，但强烈引导
5. **情感锚点优先**：先让玩家"在意"，再教"怎么玩"
