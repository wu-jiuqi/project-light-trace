# 纸工资产 QA 清单

每张候选图进入 `ready` 前必须完成本清单。

## 通用

- [ ] 透明背景无残留、灰边、光晕和无关碎片。
- [ ] 外轮廓有手工裁切感，但不损失主体识别度。
- [ ] 光源来自左上方，投影向右下方偏移。
- [ ] 使用 Style Bible 中已批准的材质族。
- [ ] 没有水印、签名、伪文字和额外角色。
- [ ] 缩小到运行时尺寸后仍然可读。
- [ ] 文件使用小写英文、数字和下划线命名。

## 角色

- [ ] 画布为统一尺寸，脚底锚点位于底边中央。
- [ ] 角色身份锚点清晰可见。
- [ ] 左右方向分别生成，没有错误镜像。
- [ ] 姿态替换没有改变脸、服装、颜色和纸张材质。
- [ ] 阴影与纸偶分离，便于运行时轻量摆动。
- [ ] 如有残留色，`_sp_mask` 与基础资产同尺寸。

## 环境模块

- [ ] 模块可与相邻模块拼接。
- [ ] 没有不可控文字。
- [ ] 资产不是完整场景截图。
- [ ] 建筑、地面、前景遮挡物分层导出。
- [ ] 在 `16px` 逻辑网格和 `4x` 主文件尺度下对齐。

## UI

- [ ] 中央留白足够，不影响中文文字渲染。
- [ ] `normal`、`hover`、`pressed`、`disabled` 状态差异明显。
- [ ] 小字号正文不会被纸纹覆盖。
- [ ] 装饰层不会遮挡焦点、按钮和进度信息。

## 异常演出

- [ ] 常态和异常之间存在明确视觉反差。
- [ ] 焦点清晰。
- [ ] 字幕安全区没有高对比干扰。
- [ ] 拼贴素材不包含必须阅读的 AI 生成文字。
